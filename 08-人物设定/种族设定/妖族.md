# 妖族

> 妖族是以妖兽为主体修炼成人的种族，拥有强横的肉身和天赋神通。本文详细介绍妖族的起源、修炼体系与社会结构。

---

## 妖族概述

妖族是由妖兽修炼化形而成的种族，天生拥有强横肉身和天赋神通。妖族数量仅次于人族，是修仙界第二大种族。妖族以血脉为尊，血脉越高，天赋越强。

### 妖族特点

| 特点 | 说明 |
|------|------|
| 肉身强横 | 天生体质远强于人族 |
| 天赋神通 | 拥有与生俱来的神通能力 |
| 血脉至上 | 以血脉等级决定地位 |
| 寿元悠长 | 同境界寿元长于人族 |
| 悟性偏弱 | 悟性不如人族 |
