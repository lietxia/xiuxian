## 种族概述

### 人族

人族是修仙界的主要种族，具有强大的学习能力和适应能力。

| 特点 | 说明 |
|------|------|
| 数量最多 | 人族数量在九界中最多 |
| 学习能力强 | 能够快速学习各种功法 |
| 适应能力强 | 能够适应各种环境 |
| 修炼速度快 | 修炼速度在各族中较快 |
| 寿命较短 | 寿命在各族中较短 |

### 妖族

妖族是由野兽修炼而成的种族，具有强大的天赋能力。

| 特点 | 说明 |
|------|------|
| 天赋强大 | 具有各种天赋能力 |
| 寿命悠长 | 寿命比人族长 |
| 修炼较慢 | 修炼速度比人族慢 |
| 化形困难 | 需要达到一定境界才能化形 |
| 智慧较低 | 初期智慧比人族低 |

### 魔族

魔族是修炼魔气的种族，具有强大的战斗力。

| 特点 | 说明 |
|------|------|
| 战斗力强 | 战斗力在各族中最强 |
| 修炼速度快 | 修炼速度很快 |
| 寿命较短 | 寿命比人族短 |
| 易入魔道 | 容易走火入魔 |
| 社会地位低 | 在修仙界中地位较低 |

### 鬼族

鬼族是灵魂状态的种族，不受肉身限制。

| 特点 | 说明 |
|------|------|
| 不死不灭 | 灵魂不灭，几乎不死 |
| 不受资质限制 | 任何灵魂都可以修炼 |
| 初期实力弱 | 初期实力很弱 |
| 对阳气敏感 | 对阳气、雷电有弱点 |
| 社会地位低 | 在修仙界中地位最低 |

### 神族

神族是天道创造的种族，具有强大的神力。

| 特点 | 说明 |
|------|------|
| 神力强大 | 具有强大的神力 |
| 寿命无限 | 寿命几乎无限 |
| 数量极少 | 数量在各族中最少 |
| 天道眷顾 | 受到天道眷顾 |
| 社会地位高 | 在修仙界中地位最高 |
