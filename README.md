# 修仙设定Wiki

> 完整的修仙世界设定知识库，为修仙小说创作提供全面的参考资料。

## 简介

本项目是一个系统化的修仙（仙侠）世界设定知识库，涵盖修炼体系、世界观、功法法术、势力组织、资源物品、地理设定、神兽妖兽、人物设定、写作参考等九大板块，共计 **920+ 篇** Markdown 文档。

## 在线阅读

基于 [VitePress](https://vitepress.dev) 构建，支持本地预览：

```bash
# 安装依赖
npm install

# 开发模式预览
npm run dev

# 构建静态站点
npm run build

# 预览构建结果
npm run preview
```

## 内容结构

```
docs/
├── worldbuilding/    # 🌍 世界观 - 天道规则、界面划分、历史历法、社会文化
├── cultivation/      # ⚡ 修炼体系 - 境界划分、资质天赋、天劫体系、职业体系
├── techniques/       # 🔮 功法法术 - 功法分类、法术体系、阵法符箓、战斗体系
├── resources/        # 💎 资源物品 - 灵石货币、丹药体系、法宝体系、天材地宝
├── factions/         # 🏯 势力组织 - 宗门制度、正魔佛妖势力、商会家族
├── geography/        # 🗺️ 世界地理 - 九界地理、秘境绝地、灵脉
├── creatures/        # 🐉 神兽妖兽 - 四灵神兽、妖兽体系、灵宠
├── characters/       # 👤 人物设定 - 人物模板、各族人物、特殊体质
└── writing-guide/    # 📝 写作参考 - 情节模板、爽点设计、文笔技巧
```

## 核心设定亮点

- **九阶境界体系**：炼气 → 筑基 → 金丹 → 元婴 → 化神 → 炼虚 → 合体 → 大乘 → 渡劫（飞升）
- **九界架构**：凡界、修真界、仙界、魔界、妖界、佛界、冥界、神界、外天
- **天劫体系**：三级分类（小劫/大劫/灭世劫）× 九种劫型，威力递增
- **功德业力气运量化**：数值化规则，互相影响，可直接用于小说设定
- **功法九阶分类**：炼丹、炼器、符箓、阵法四大职业体系完整覆盖
- **势力生态**：宗门内部结构、坊市制度、家族等级制、商会体系

## 技术栈

- [VitePress](https://vitepress.dev) 1.x - 静态站点生成
- 纯 Markdown 内容，无需数据库
- 支持本地搜索、中文导航
