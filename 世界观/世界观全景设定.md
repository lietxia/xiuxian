# 修仙世界观全景设定

> 本文档从宏观角度详细介绍修仙世界的宇宙观、位面设定、时间体系、空间结构等深层设定，为构建宏大的修仙世界提供理论基础。

---

## 一、宇宙起源

### 混沌初开

传说在天地未开之前，整个宇宙是一片混沌。混沌之中，有一至高存在——盘古大神。

**混沌时期（0-∞年前）**
```
混沌 ──→ 孕育盘古 ──→ 开天辟地 ──→ 天地分离
         ↓              ↓              ↓
      混沌精气      清气上升      浊气下降
      混沌法则      三十三重天    九幽冥界
```

### 开天辟地

据《创世典籍》记载：

**盘古开天**
- 盘古于混沌中苏醒
- 以巨斧劈开混沌
- 清气上升为天
- 浊气下沉为地
- 盘古身化万物

**盘古身化**
- 双眼化为日月
- 身躯化为山川大地
- 血液化为江河湖海
- 发丝化为草木森林
- 汗珠化为雨露甘霖
- 十二滴精血化为十二祖巫

### 天地初成

天地初成之后，灵气充盈，法则稳固。天地间开始孕育第一批生灵。

**第一批生灵**
- 先天神祇：盘古意志所化
- 混沌生灵：混沌中遗留
- 天地精灵：灵气凝聚而成
- 太古凶兽：混沌怨气所化

## 二、位面体系

### 位面层级

```
┌─────────────────────────────────────────┐
│              混沌海（虚无之海）               │
├─────────────────────────────────────────┤
│  ┌─────────────────────────────────┐    │
│  │      天外天（至高空间）              │    │
│  ├─────────────────────────────────┤    │
│  │  ┌─────────┐ ┌─────────┐        │    │
│  │  │  神界   │ │  仙界   │        │    │
│  │  ├─────────┤ ├─────────┤        │    │
│  │  │  魔界   │ │  佛界   │        │    │
│  │  ├─────────┤ ├─────────┤        │    │
│  │  │  妖界   │ │  冥界   │        │    │
│  │  └─────────┘ └─────────┘        │    │
│  │            ↓                    │    │
│  │  ┌─────────────────────────┐   │    │
│  │  │      修真界（人界）        │   │    │
│  │  │  ┌─────┐ ┌─────┐      │   │    │
│  │  │  │凡间界│ │灵界 │      │   │    │
│  │  │  └─────┘ └─────┘      │   │    │
│  │  └─────────────────────────┘   │    │
│  └─────────────────────────────────┘    │
│                  ↓                       │
│  ┌─────────────────────────────────┐    │
│  │      九幽冥界（下层空间）           │    │
│  └─────────────────────────────────┘    │
└─────────────────────────────────────────┘
```

### 各界详解

**混沌海**
- 宇宙最外围
- 虚无混沌
- 偶尔有混沌生物出没
- 大能可借此穿梭诸天

**天外天**
- 超越三界之外
- 传说是道祖居所
- 普通人无法进入
- 时间空间在这里失去意义

**神界**
- 神灵居所
- 掌控天道法则
- 凡人不可至
- 神族起源之地

**仙界**
- 仙人居所
- 天庭所在
- 分三十三重天
- 修仙者飞升目标

**魔界**
- 魔族居所
- 魔祖罗睺领地
- 魔气充盈
- 与仙界对立

**佛界**
- 佛教道场
- 极乐世界
- 西天灵山
- 菩萨罗汉居所

**妖界**
- 妖族起源
- 万妖之国
- 妖族圣地
- 各种妖兽聚集

**冥界**
- 亡魂归宿
- 六道轮回
- 十殿阎罗
- 阴气极重

**修真界**
- 人族主要活动区域
- 修仙文明中心
- 凡间界与灵界组成
- 灵气相对均衡

```
修真界一天 = 凡间界一年
冥界时间流速不定
```

## 四、空间结构

### 空间层级

**空间裂缝**
- 最不稳定的传送方式
- 随机出现
- 危险极大
- 可能传送到任何地方

**临时传送阵**
- 可移动的阵法
- 需要消耗灵石
- 目的地固定
- 一次性使用

**固定传送阵**
- 永久性阵法
- 连接两个固定地点
- 需要持续维护
- 消耗灵石运转

**空间通道**
- 大能开辟
- 稳定安全
- 可长期使用
- 需要特殊材料维护

### 空间距离

| 距离单位 | 换算 | 说明 |
|----------|------|------|
| 丈 | 3.3米 | 最短距离 |
| 里 | 500米 | 小距离 |
| 百里 | 50公里 | 中距离 |
| 千里 | 500公里 | 大距离 |
| 万里 | 5000公里 | 超大距离 |
| 亿里 | 宇宙级 | 星际距离 |

## 五、世界规则

### 天道法则

**天道特性**
- 无情而公正
- 维护世界运转
- 不可违背
- 超越一切存在

**天道职责**
- 维持世界平衡
- 执行因果报应
- 管理轮回转世
- 降下天劫

### 因果律

**因果法则**
- 种因得果
- 善有善报，恶有恶报
- 修士行动受因果约束
- 业力影响修炼

**因果报应**
- 小因果：影响不大
- 大因果：影响气运
- 滔天因果：可能遭天谴
- 逆天因果：必遭天罚

### 气运法则

**气运特点**
- 不可见但存在
- 可以转移
- 影响机缘
- 大气运者得天眷顾

**气运获取**
- 积累功德
- 庇佑苍生
- 传承延续
- 天材地宝认主

**气运流失**
- 杀戮过多
- 造下因果
- 背叛师门
- 气运耗尽则亡

## 六、世界运转

### 轮回体系

```
生灵的轮回流程：
┌─────────────────────────────────────────┐
│                                           │
│    出生 → 成长 → 修炼 → 死亡              │
│         ↓                        ↓       │
│       修炼                   返回轮回      │
│         ↓                        ↓       │
│    飞升仙界 ←───────────── 六道轮回        │
│                                           │
└─────────────────────────────────────────┘
```

**轮回六道**
- 天道：积德行善，飞升仙界
- 人道：正常轮回，转世为人
- 修罗道：战死或好战者
- 畜生道：作恶或愚痴者
- 饿鬼道：贪婪吝啬者
- 地狱道：大奸大恶者

### 大劫体系

**小劫（万年）**
- 灵气潮汐
- 影响较小
- 修炼速度变化

**中劫（十万年）**
- 妖魔乱世
- 灾难降临
- 影响一个大陆

**大劫（三十六元会）**
- 天地大劫
- 改朝换代
- 涉及整个修真界

**无量量劫（不可计算）**
- 世界毁灭
- 宇宙重开
- 只有大能可逃脱

### 灵气潮汐

**涨潮期**
- 灵气浓郁
- 适合修炼
- 天材地宝涌现
- 修士突破容易

**退潮期**
- 灵气稀薄
- 修炼困难
- 资源匮乏
- 修士突破困难

## 七、世界历史

### 纪元划分

**混沌纪元**
- 时间不可考
- 盘古开天之前
- 只有混沌生灵
- 已无法考证

**太古纪元**
- 盘古开天之后
- 先天神祇活跃
- 巫妖大战
- 神魔混战

**上古纪元**
- 人族崛起
- 修仙文明开始
- 宗门林立
- 仙魔大战

**中古纪元**
- 修仙文明繁荣
- 天庭建立
- 仙界统一
- 规则完善

**近古纪元**
- 修仙文明鼎盛
- 各大势力成型
- 经典功法定型
- 延续至今

## 八、世界边界

### 天地的尽头

**天界尽头**
- 三十三重天之上
- 天道所在
- 无法到达
- 传说有天柱支撑

**大地尽头**
- 虚空混沌海
- 大陆边缘
- 危险区域
- 偶尔有珍贵材料

**世界之壁**
- 包围整个世界的屏障
- 隔绝混沌
- 只有特殊地点可以穿越
- 极其坚固

### 世界之外

**混沌海**
- 包围世界的混沌
- 充满危险
- 偶尔有混沌生物
- 大能可借此往来诸天

**其他世界**
- 传说有无数世界
- 可能有其他文明
- 穿越极其困难
- 可能性未知

## 九、世界能量体系

### 能量种类

| 能量类型 | 所在界域 | 主要使用者 |
|----------|----------|------------|
| 灵气 | 修真界 | 人族、妖族 |
| 仙气 | 仙界 | 仙人 |
| 神力 | 神界 | 神祇 |
| 魔气 | 魔界 | 魔族 |
| 佛力 | 佛界 | 佛教徒 |
| 妖力 | 妖界 | 妖族 |
| 冥气 | 冥界 | 鬼修 |
| 混沌之气 | 混沌海 | 混沌生灵 |

### 能量转换

```
混沌之气 → 灵气/仙气/魔气/神力/佛力/妖力/冥气
    ↓
各能量可相互转化（需要特殊条件）
    ↓
最终回归混沌
```

### 灵气分布

**灵气浓郁区**
- 大型灵脉附近
- 洞天福地
- 上古遗迹
- 秘境内部

**灵气普通区**
- 普通山脉
- 一般平原
- 中小型城市

**灵气匮乏区**
- 沙漠戈壁
- 死地绝域
- 凡间界大部分地区
