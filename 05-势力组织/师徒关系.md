# 师徒关系

> 师徒关系是修仙世界中最重要的关系之一，是功法传承和修炼指导的主要方式。

---

## 师徒关系概述

### 什么是师徒关系？

师徒关系是修仙世界中师傅与徒弟之间的关系，是功法传承和修炼指导的主要方式。师傅传授功法、指导修炼，徒弟学习功法、尊师重道。

### 师徒关系的特点

| 特点 | 说明 |
|------|------|
| 传承性 | 功法传承的主要方式 |
| 指导性 | 修炼指导的主要方式 |
| 情感性 | 建立深厚情感 |
| 义务性 | 双方都有义务 |
| 长久性 | 关系长久稳定 |
