# 剑宗

> 剑宗是以剑道修炼为核心的修仙势力，以"一剑破万法"为信念。本文详细介绍剑修门派的组织结构与特点。

---

## 剑宗概述

剑宗是正道势力中以剑道修炼为核心的门派群体。剑修追求剑道极致，以剑入道，实力在同阶修士中往往最强。

### 剑宗核心理念

| 理念 | 说明 |
|------|------|
| 一剑破万法 | 以剑道破一切法术 |
| 剑心通明 | 剑心如明镜，不染尘埃 |
| 人剑合一 | 剑即是人，人即是剑 |
| 以剑证道 | 以剑道证悟天地大道 |

---

## 主要剑宗

### 蜀山剑派

| 项目 | 内容 |
|------|------|
| 地位 | 剑修之首，正道第二大势力 |
| 掌门 | 剑无极（合体期） |
| 弟子数量 | 约两万人 |
| 势力范围 | 修真界西部 |
| 主修功法 | 蜀山剑诀（八品） |
| 特色 | 剑阵无双、飞剑千柄 |
| 镇宗法宝 | 蜀山万剑归宗阵 |

### 凌霄剑宗

| 项目 | 内容 |
|------|------|
| 地位 | 剑修第二，擅长单体剑术 |
| 掌门 | 凌霄剑尊（合体期初期） |
| 弟子数量 | 约一万人 |
| 主修功法 | 凌霄剑经（七品） |
| 特色 | 单体剑术登峰造极 |

### 天剑阁

| 项目 | 内容 |
|------|------|
| 地位 | 剑修第三，培养剑修天才 |
| 阁主 | 天剑老人（炼虚期巅峰） |
| 弟子数量 | 约五千人 |
| 主修功法 | 天剑诀（七品） |
| 特色 | 不拘一格，各流派剑修汇聚 |

## 剑修特点

| 特点 | 说明 |
|------|------|
| 攻击力最强 | 同阶修士中剑修攻击力最高 |
| 防御较弱 | 剑修重攻轻守 |
| 修炼艰苦 | 剑道修炼极为艰苦 |
| 数量稀少 | 剑修对资质要求极高 |
| 性格刚直 | 剑修多性格刚直 |
