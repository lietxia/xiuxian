# 书院

> 书院是正道势力中以儒道修炼为核心的门派，以"文道"入修。本文详细介绍书院的组织结构与特点。

---

## 书院概述

书院是以儒家思想为基础的修仙势力，以读书入道、文道修炼为核心。书院修士以笔为剑、以文入道，是正道势力中独特的存在。

### 书院核心理念

| 理念 | 说明 |
|------|------|
| 修身齐家 | 先修己身，再济天下 |
| 读书入道 | 以读书参悟天地大道 |
| 浩然正气 | 修炼浩然正气，万邪不侵 |
| 文道天下 | 以文道教化天下 |
