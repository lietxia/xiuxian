# 仙盟

仙盟，全称"修仙界同盟"，是修真界最大、最具权威的跨门派联盟组织，由各大正道门派联合创立，旨在维护修仙界的秩序与稳定，抵御外敌入侵，调解门派纷争。仙盟虽非权力机构，却在修仙界拥有举足轻重的话语权，其裁决往往被视为公正之判。

## 仙盟的起源

仙盟的创立可追溯至上古封神大战之后。封神大战旷日持久，阐教、截教、人教三教纷争，最终以截教落败、通天教主被囚而告终。战后，幸存的正道修士意识到门派纷争只会让魔族趁虚而入，于是在阐教副教主燃灯道人的倡导下，十二大正道门派联合创立了仙盟，誓约互不侵犯、共同御敌。

自创立至今，仙盟已延续数万年，历经多次改组与壮大，从最初的十二门派扩展到如今的数百成员门派，成为修仙界最具影响力的联盟组织。

## 仙盟的宗旨

### 核心理念

- **维护正道**：弘扬正气，抵制邪道，维护修仙界的正义秩序
- **止戈息争**：调解门派之间的矛盾与纷争，避免无谓的杀伐
- **共御外敌**：当魔族、妖修等外敌入侵时，仙盟各派须同仇敌忾、一致对外
- **传承道统**：保护上古功法、丹方、炼器秘术等珍贵文化遗产
- **庇佑凡人**：保护凡人免受妖魔侵害，维护人间秩序

### 仙盟盟约

所有加盟门派须遵守仙盟盟约，主要条款包括：
1. 不得无故侵犯其他加盟门派
2. 不得滥杀凡人、沾染过多因果
3. 不得修习公认邪功、祸害苍生
4. 遭遇外敌入侵时，须响应仙盟征召
5. 门派重要传承须向仙盟报备，不得私自外流

违反盟约者，将被仙盟除名，并遭到仙盟联合制裁。

## 仙盟组织架构

### 盟主

仙盟最高领导人，由仙盟大会选举产生，任期百年，可连任一次。盟主通常由九大执事门派中推举，需具备大乘期以上修为，且德高望重、公正无私。

**历任盟主（近现代）**：
- **太虚真人**（青云门）：第三十二任盟主，掌盟三百年，以公正著称
- **清虚仙子**（昆仑派）：第三十三任盟主，唯一的女性盟主
- **元一真人**（蜀山剑派）：第三十四任盟主，剑道通神

### 九大执事门派

仙盟的核心决策层，由九大实力最强的正道门派组成：

| 执事门派 | 总部所在 | 擅长领域 |
|---------|---------|---------|
| 昆仑派 | 昆仑仙山 | 剑法、丹药 |
| 蜀山剑派 | 蜀山 | 剑道 |
| 蓬莱阁 | 蓬莱仙岛 | 炼器、阵法 |
| 武当派 | 武当山 | 太极拳法 |
| 少林寺 | 嵩山 | 炼体、禅法 |
| 青云门 | 青云山 | 综合功法 |
| 峨眉派 | 峨眉山 | 剑道、医术 |
| 茅山派 | 茅山 | 符箓、术法 |
| 紫霄宫 | 紫霄峰 | 天道推演 |

九大执事门派各派一名长老常驻仙盟总部，参与日常决策。

### 仙盟长老会

由九大执事门派各推举三名长老，加上盟主，共二十八人组成。长老会负责仙盟重大事务的审议与裁决，如接纳新成员、开除违规门派、发动联盟战争等。

### 仙盟执法堂

负责执行仙盟决议、调查违规行为、处置违反盟约的门派。执法堂由盟主直接管辖，堂主由历任盟主任命。

**执法堂堂规**：
- 铁面无私，不论门派大小一律平等
- 证据为先，未经查实不得行动
- 宽严相济，初犯者可警告处置，再犯者严惩不贷

### 仙盟评审堂

负责审核新门派加入仙盟的申请，以及评估各成员门派的实力等级。评审堂由德高望重的元老组成，评定结果具有权威性。

## 仙盟成员门派等级

仙盟成员门派按实力分为九个等级，与修士修为境界对应：

### 一品门派

只有九大执事门派可达一品，统领整个正道江湖。一品门派拥有渡劫期长老坐镇，底蕴深厚，传承万年以上。

### 二品门派

有合体期以上修为长老坐镇的大型门派，在修仙界享有极高声望。数量不超过三十个。

### 三品门派

有大乘期长老坐镇的门派，数量约五十个，是正道江湖的中坚力量。

### 四品至六品门派

分别对应炼虚期、化神期、元婴期长老坐镇的中型门派，数量众多，是仙盟的基层力量。

### 七品至九品门派

分别对应金丹期、筑基期、练气期长老坐镇的小型门派，多为地方性小门派，影响力有限。

## 仙盟主要活动

### 联盟大会

每百年召开一次，由盟主主持，九大执事门派及所有成员门派参加。大会主要议程包括：
- 汇报过去百年的工作
- 表彰有功门派与个人
- 审议新加入或开除门派
- 选举或确认盟主
- 讨论修仙界重大事务

### 仙道大比

每五十年举办一次的青年修士比试，由各成员门派选送优秀弟子参赛。仙道大比不仅是实力的较量，更是各门派展示实力的舞台。前十名优胜者将获得仙盟赐予的丰厚奖励，包括珍稀丹药、上古功法残篇、仙器仿品等。

### 联盟征战

当魔族、妖修等外敌大规模入侵时，仙盟可发动联盟征战。九大执事门派须各自出兵，其他成员门派按等级出兵，共同御敌。历史上最大规模的联盟征战是"万魔之战"，修仙界险些覆灭，最终在仙盟的统筹下击退魔族。

### 遗迹探索

仙盟定期组织成员门派探索上古遗迹、秘境，所得资源按贡献分配。遗迹探索不仅能获取珍贵资源，还能增进各门派之间的合作与信任。

## 仙盟的影响力

### 正面影响

- **维护秩序**：仙盟的存在有效遏制了门派之间的无序争斗
- **抵御外敌**：在魔族入侵等危机时刻，仙盟发挥了中流砥柱的作用
- **传承保护**：仙盟保护了大量上古功法与文化遗产
- **培养人才**：仙道大比激励了青年修士的修炼热情

### 负面影响

- **门派傲慢**：部分大派弟子仗着门派势力骄横跋扈
- **裙带关系**：九大执事门派有时利用职权打压中小门派
- **效率低下**：重大决策须经过多方协调，效率较低
- **门户之见**：各派敝帚自珍，真正的顶级功法仍不愿共享

## 仙盟与各方势力的关系

### 与正道门派

仙盟是正道门派的联合组织，天然维护正道利益。但正道门派之间也存在竞争，有时仙盟的调解也难以服众。

### 与邪道势力

仙盟与邪道势力势不两立。历史上多次正邪大战，背后都有仙盟的统筹指挥。但仙盟也深知邪道难以彻底消灭，通常采取压制而非赶尽杀绝的策略。

### 与中立势力

中立门派（如逍遥派）可作为仙盟观察员，参与部分活动，但不享有表决权。仙盟也尊重中立门派的立场，不强迫其参与正邪之争。

### 与冥界

仙盟与冥界保持着若即若离的合作关系。当涉及鬼修入侵等事务时，仙盟会与冥界酆都大帝交涉。

### 与天庭（神界）

仙盟名义上接受天庭的统辖，但修仙界相对独立，天庭很少直接干预。天庭若有事务需借助修仙界力量，通常通过仙盟传达。

## 仙盟的争议与危机

### 历史争议

- **正邪界限模糊**：某些功法的正邪属性存在争议（如双修功法），导致门派被误伤
- **执事门派特权**：九大执事门派权力过大，中小门派颇有微词
- **盟主选举黑幕**：历任盟主选举中，多次传出舞弊传闻

### 近年危机

- **新锐门派崛起**：一些未加入仙盟的新兴门派实力已超越部分老牌执事门派，要求改革的呼声越来越高
- **魔道渗透**：有迹象表明，魔族势力正在渗透部分中小门派，试图从内部瓦解仙盟
- **人才流失**：许多天才修士选择独立修炼而非加入门派，导致门派人才青黄不接

## 如何加入仙盟

### 申请条件

- 门派成立须满三百年
- 门派中须有至少一名元婴期以上修为的长老
- 门派功法须为正道传承，不得修炼公认邪功
- 须有两名仙盟成员门派推荐
- 须接受仙盟评审堂的资质审查

### 加入流程

1. **提交申请**：向仙盟评审堂提交申请书及门派资料
2. **推荐审核**：两名推荐门派提交推荐函
3. **资质审查**：评审堂派员实地考察门派实力
4. **长老会表决**：三分之二以上长老同意即可通过
5. **宣誓盟约**：掌门率全派弟子宣誓遵守盟约
6. **授予盟牌**：正式成为仙盟成员门派
