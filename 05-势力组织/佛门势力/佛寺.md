# 佛寺

> 佛寺是修仙界佛门势力的核心组织，以慈悲为怀、普度众生为宗旨。本文详细介绍佛门四寺的组织结构与特点。

---

## 佛寺概述

佛门是修仙界与道门并列的两大正统修炼体系，以慈悲为怀、普度众生为宗旨。佛门四寺是佛门最核心的势力。

### 佛门核心理念

| 理念 | 说明 |
|------|------|
| 慈悲为怀 | 以慈悲心对待一切众生 |
| 普度众生 | 救助一切苦难众生 |
| 因果报应 | 善有善报、恶有恶报 |
| 超脱轮回 | 追求超脱六道轮回 |

## 佛门四寺

### 大雷音寺

| 项目 | 内容 |
|------|------|
| 地位 | 佛门祖庭 |
| 住持 | 如海方丈（合体期） |
| 僧众 | 约两万人 |
| 特色 | 佛门至高佛法传承 |
| 镇寺法宝 | 如来金身 |

### 菩提禅院

| 项目 | 内容 |
|------|------|
| 地位 | 佛门第二，禅修为主 |
| 院长 | 菩提禅师（炼虚期） |
| 僧众 | 约一万人 |
| 特色 | 禅定功夫天下第一 |

| 项目 | 内容 |
|------|------|

| 僧众 | 约八千人 |
| 特色 | 金刚不坏身、降魔武力 |

### 普陀山

| 项目 | 内容 |
|------|------|

| 特色 | 治疗之术、度化之能 |
