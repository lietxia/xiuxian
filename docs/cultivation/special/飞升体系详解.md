## 一、飞升概述

### 1.1 什么是飞升

飞升，是指修仙者修为达到极致，渡过天劫后，肉身与灵魂进入更高层次世界的过程。飞升后，修仙者将成为真正的仙人，获得永恒的寿元和更强大的力量。

### 1.2 飞升的历史

飞升的历史可追溯至上古时代。传说第一位飞升者是上古大能盘古，他以无上神通开天辟地，最终飞升至天外天。此后，飞升成为修仙者的终极目标，无数修仙者为之奋斗终生。

### 1.3 飞升的意义

| 方面 | 说明 |
|:---:|:---|
| 寿元 | 飞升后获得永恒寿元 |
| 力量 | 飞升后获得更强大的力量 |
| 世界 | 飞升后进入更高层次的世界 |
| 地位 | 飞升后成为真正的仙人 |
| 传承 | 飞升后可传承更高级的功法 |

## 二、飞升条件

### 2.1 境界要求

飞升需要修为达到渡劫期巅峰，且渡过飞升天劫方可飞升。

| 境界 | 要求 | 说明 |
|:---:|:---|:---|
| 渡劫期 | 境界圆满 | 修为需达到渡劫期巅峰 |
| 道心 | 道心坚定 | 道心需坚定不移 |
| 神识 | 神识强大 | 神识需达到一定程度 |
| 灵力 | 灵力充沛 | 灵力需达到极致 |
| 肉身 | 肉身完美 | 肉身需达到极致 |

### 2.2 功德要求

飞升需要一定的功德值，功德值可通过行善积德获得。

| 功德等级 | 功德值 | 说明 |
|:---:|:---:|:---|
| 无功德 | 0 | 无法飞升 |
| 小功德 | 1000 | 飞升概率较低 |
| 中功德 | 10000 | 飞升概率中等 |
| 大功德 | 100000 | 飞升概率较高 |
| 无量功德 | 1000000 | 飞升概率极高 |

### 2.3 天道认可

飞升需要天道认可，天道会根据修仙者的修为、功德、心性等综合因素决定是否允许飞升。

| 认可条件 | 权重 | 说明 |
|:---:|:---:|:---|
| 境界 | 30% | 修为境界 |
| 功德 | 25% | 功德值 |
| 心性 | 20% | 道心坚定度 |
| 神识 | 15% | 神识强度 |
| 肉身 | 10% | 肉身强度 |
