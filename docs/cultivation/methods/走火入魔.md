# 走火入魔

> 走火入魔是修仙者修炼过程中的重大危机，轻则修为倒退，重则身死道消。本文详细介绍走火入魔的原因、症状、治疗与预防方法。

---

## 走火入魔概述

走火入魔，简称入魔，是指修仙者在修炼过程中因各种原因导致经脉逆行、气血紊乱、神魂失守的状态。轻者修为倒退、精神受损，重者当场暴毙、化为灰烬。

### 走火入魔的本质

走火入魔的本质是修炼者体内灵气运行失控，与经脉、神魂产生冲突。灵气本是天地精华，一旦失控便如脱缰野马，在体内横冲直撞，造成不可逆的伤害。

| 阶段 | 症状 | 伤害程度 | 治疗难度 |

| 萌芽期 | 烦躁不安、灵气紊乱 | 轻微 | 易 |

| 深度入魔 | 元神受损、性情大变 | 极重 | 极难 |
| 完全入魔 | 神志全失、沦为魔物 | 不可逆 | 几乎不可能 |

## 入魔后的修仙者

入魔后即使救治成功，修仙者也可能面临以下后果：

| 后果 | 说明 | 恢复可能 |
|------|------|---------|
| 修为倒退 | 修为可能倒退一小境界至大境界 | 可重新修炼 |
| 经脉受损 | 经脉留下永久损伤 | 可修复但耗费巨大 |
| 神魂创伤 | 神魂留下阴影 | 极难完全恢复 |
| 心魔潜伏 | 心魔潜伏体内 | 可能再次发作 |
| 性格改变 | 性情可能发生变化 | 部分可恢复 |
| 寿元折损 | 入魔折损部分寿元 | 不可恢复 |
