# 特殊法则

特殊法则是十二条大道法则中较为罕见、难以领悟的法则。这些法则通常涉及万物运转的深层规则，领悟难度极高，但威力也极为强大。特殊法则包括杀戮法则、命运法则、因果法则、毁灭法则、造化法则等。

## 杀戮法则

### 法则本质

杀戮法则的本质是"征伐"。万物之间存在竞争与征伐，杀戮法则便是掌控了"征伐"的规则。杀戮法则的领悟者，可以操控杀戮之力，增强攻击威力，削弱敌人战意。

### 核心能力

| 能力名称 | 效果 | 领悟门槛 |
|---------|------|---------|
| 杀戮之意 | 增强攻击威力30% | 元婴期 |
| 杀戮领域 | 领域内敌人战意降低50% | 化神期 |
| 杀戮化身 | 化身杀戮，攻击威力翻倍 | 合体期 |
| 杀戮法则 | 掌控杀戮法则，攻击威力三倍 | 大乘期 |
| 杀戮道果 | 凝聚杀戮道果，攻击威力五倍 | 真仙 |

### 领悟条件

1. **境界要求**：最低元婴期
2. **杀戮要求**：需经历至少1000次生死搏杀
3. **悟性要求**：悟性需达到"天才"级别（千中无一）
4. **机缘要求**：需经历至少一次"杀戮顿悟"

### 代表人物模板

**杀戮道祖**
- 境界：真仙巅峰
- 法则掌控度：90%（一品道果）
- 核心能力：攻击威力五倍，领域内敌人战意降低80%
- 战绩：曾以杀戮法则灭杀三位同阶修士

## 特殊法则之间的关系

特殊法则之间存在相互关联、相互影响的关系：

**相生关系**：
- 毁灭法则 + 造化法则 = 鸿蒙法则
- 阴阳法则 = 太极法则
- 五行法则 = 混沌法则

**相克关系**：
- 杀戮法则克造化法则
- 造化法则克毁灭法则
- 毁灭法则克杀戮法则

**融合关系**：
- 毁灭法则 + 造化法则 = 鸿蒙法则
- 阴阳法则 = 太极法则
- 五行法则 = 混沌法则
