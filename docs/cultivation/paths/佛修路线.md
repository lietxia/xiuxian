# 佛修路线

佛修，修仙界最神秘的修炼路线。他们不修灵气，修功德；不争资源，修因果。佛修讲究"渡人渡己"，以慈悲为怀，以因果为则。

佛修看似与世无争，实则拥有恐怖的战力。金身罗汉一拳可碎山河，佛陀一念可度化万灵。

## 佛修劣势

佛修并非完美路线，也有明显的劣势：

- **修炼速度慢**：功德积累需要时间，前期进度远不如其他路线
- **限制极多**：不杀生、不妄语、不淫邪，诸多戒律约束
- **因果敏感**：对因果感知太强，容易被因果纠缠
- **度化有限**：面对心性坚定者，度化神通无效
- **资源依赖**：需要香火愿力，离开寺庙佛刹修炼困难

佛修之路，是一条漫长而艰难的道路。但一旦证得佛陀果位，便可超脱轮回，不生不灭。
