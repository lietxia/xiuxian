# 修仙世界历史年表

## 凡间纪年说明

凡间以王朝更替纪年。修仙界则另有体系：

### 修真纪年

修真界以"劫"纪年：
- **一元**：129600年 = 12会
- **一会**：10800年
- **一运**：1080年
- **一纪**：360年
- **一年**：360日

当前处于第五十三元第三会第三运第三十七纪第三年

## 太古时期

- **鸿钧合道**：鸿钧合身天道，成为大道化身
- **天道运转**：天道开始运转，法则成型

- **太上老君成圣**：道德天尊成圣，建立兜率宫

- **通天教主成圣**：通天教主成圣，建立碧游宫

- **仙界秩序建立**：三十三重天成型
- **仙界规则确立**：飞升规则、仙人职责确立

### 冥界建立

- **酆都大帝诞生**：酆都大帝诞生
- **六道轮回建立**：轮回法则确立
- **冥界秩序建立**：十殿阎罗任命

### 仙魔大战

- **魔祖罗睺诞生**：罗睺在混沌中诞生
