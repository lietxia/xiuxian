# 气运法则

> 气运是天道对生灵的眷顾程度，无形却真实存在。气运高者逢凶化吉、机缘不断；气运低者诸事不顺、灾祸连连。气运之子得天道垂青，往往能成就一番伟业。

---

## 气运的本质

### 什么是气运？

气运是天道对生灵眷顾程度的量化体现。每个生灵自诞生起便拥有一定的气运，气运的高低决定了此人的命运走向。气运不是迷信，而是天道平衡世界的一种机制。

### 气运的两大载体

| 载体 | 说明 | 特点 |
|------|------|------|
| 天道眷顾 | 天道对生灵的直接眷顾 | 可增可减，受行为影响 |
| 命格承载 | 先天命格所承载的气运 | 相对固定，难以改变 |

### 气运的特点

| 特点 | 说明 |
|------|------|
| 先天性 | 每个生灵天生拥有一定气运 |
| 可变性 | 气运会随行为和事件变化 |
| 流转性 | 气运可以在生灵之间流转 |
| 聚集性 | 气运高者会吸引更多气运 |
| 消耗性 | 气运可以被消耗 |
