# 修仙界节日庆典

修仙界岁月漫长，动辄以百年千年计。修士虽追求长生，却非无情之物。每逢大事，必有庆典；四时流转，亦有节气。庆典既是礼仪传承，也是宗门势力的展现。

---

## 一、宗门庆典

### 1.1 入门大典 / 入灵大会

新弟子拜入宗门，是宗门延续的根本。入门大典规模视宗门等级而定，大宗门往往三至七日不绝。

| 环节 | 内容 | 备注 |
|------|------|------|
| 祭祖入门 | 新弟子于祖师殿前焚香叩拜，诵读宗门戒律 | 由长老主持，掌门亲临 |
| 收礼会客 | 同盟宗门、世家派遣使者观礼并送上贺礼 | 贺礼多为灵石、法器、丹药 |
| 坐而论道 | 新弟子聆听长老讲道，感悟宗门功法根基 | 部分宗门在此环节筛选天赋 |
| 赐下信物 | 身份玉牌、弟子服、基础功法 | 标志正式成为宗门弟子 |

大宗门的入门大典往往持续数日，期间开放宗门部分秘境供新弟子体验。小宗门则简化为一日之内完成。

### 1.2 拜师大典

拜师不同于入门。入门是加入宗门，拜师是确立师承关系。修仙界极重师徒名分，拜师大典庄重肃穆。

- **宣读祖训**：由辈分最高的长老诵读本脉祖师遗留的训诫
- **行拜师礼**：三跪九叩，奉茶献礼，师父饮茶则表示接纳
- **赐予宝物**：师父赐下法器、功法或护身法宝，作为师徒信物
- **刻录名册**：师徒姓名录入宗门传承谱系，不可更改

拜师之后，师徒关系终身不渝。背叛师门者，为修仙界所不齿。

### 1.3 宗门祭

宗门祭是祭祀先祖、已故前辈的大型活动，通常每年一次或数年一次。

| 类型 | 频率 | 规模 |
|------|------|------|
| 年祭 | 每年一次 | 宗门内部，全体弟子参加 |
| 大祭 | 十年或百年一次 | 邀请同盟势力观礼 |
| 国祭 | 千年一次 | 整个修真界瞩目 |

宗门祭不仅有缅怀先人的意义，更有政治意味。大祭期间，宗门会展示实力、宣布重大决策、与盟友巩固关系。部分宗门会在大祭期间举办比武、论道等活动。

### 1.4 合宗大典

两个或多个宗门合并为一，需举行合宗大典。

合宗大典的流程：
1. 双方掌门签署合宗契约，以天道誓言约束
2. 共同祭拜双方祖师，将牌位合入同一祖师殿
3. 重新划分宗门职务、资源分配
4. 全体弟子重新登记造册
5. 昭告修真界，接受各方贺礼

合宗往往意味着势力格局的重大变化，周边宗门会密切关注。

### 1.5 开宗大典

新建宗门的成立仪式。

- 创派祖师需达到**金丹期**以上修为，方可开宗立派
- 选定山门地址，布置护山大阵
- 广发请帖，邀请周边势力观礼
- 立下宗门戒律、传承功法
- 挂匾揭牌，正式昭告天下

小宗门开宗可能只有数人参加，大宗门开宗则震动一方。

### 1.6 仙籍册封

仙门册封弟子仙籍，是给予正式身份认可的仪式。

仙籍等级：

| 等级 | 对应修为 | 权限 |
|------|----------|------|
| 外门仙籍 | 炼气期 | 基础资源供给 |
| 内门仙籍 | 筑基期 | 可进入内门秘境 |
| 核心仙籍 | 金丹期 | 可参与宗门决策 |
| 太上仙籍 | 元婴及以上 | 宗门供奉，不受俗务约束 |

## 二、进阶庆典

修为突破是修士一生中最重要之事。每一次大境界突破，都值得隆重庆祝。

### 2.1 大乘进阶庆典

大乘期修士已是半步飞升，进阶庆典震动整个修真界。

| 要素 | 说明 |
|------|------|
| 昭告时间 | 提前三月乃至一年昭告天下，让各方势力有时间准备 |
| 参加门槛 | 金丹期以下修士通常无资格参加，需有元婴期以上修为或大宗门弟子身份 |
| 贺礼规格 | 各方势力献上顶级灵材、仙器、珍稀丹药 |
| 附属活动 | 论道大会、拍卖会、交易会往往同期举行 |
| 持续时间 | 短则七日，长则数月 |

大乘庆典期间，常有修士借此机缘突破瓶颈。因此各方势力争相参加，既是贺喜，也是寻找自身机缘。

### 2.2 元婴庆典

元婴期修士可开辟洞天，寿元超过千年。元婴庆典是宗门的大事。

- 宗门会大摆筵席，以灵果灵酒款待宾客
- 各方势力送上贺礼，巩固关系
- 元婴修士需展示自身实力，接受各方检验
- 部分宗门会借此机会宣布新的长老任命

元婴庆典的规模仅次于大乘庆典，通常持续三至七日。

### 2.3 筑基庆典

筑基是修仙的第一道门槛。突破筑基，意味着正式踏入修仙之路。

筑基庆典多为家族庆典：
- 家族设宴庆祝，邀请亲朋好友
- 赠送灵石、法器等贺礼
- 部分家族会将筑基者列入族谱核心
- 若是宗门弟子，宗门会赐下筑基丹等资源

筑基庆典虽规模不大，但对修士本人意义重大。从此脱离凡人之列，寿元大增。

## 三、飞升庆典

### 3.1 飞升异象

修士渡过天劫、肉身飞升仙界，会产生天地异象。

| 异象 | 描述 |
|------|------|
| 祥云漫天 | 五彩祥云铺满天际，方圆千里可见 |
| 仙乐飘渺 | 天降仙音，闻者心神宁静，瓶颈松动 |
| 金光万道 | 金色光柱冲天而起，持续数日不散 |
| 灵雨降世 | 天降灵雨，草木疯长，凡人百病皆消 |
| 天花乱坠 | 虚空中涌现天花，触之有悟道之效 |

### 3.2 飞升影响

飞升是整个修真界的大事。

- **宗门庆祝**：所属宗门大摆筵席，持续数月乃至数年
- **天下同庆**：修真界各方势力送来贺礼
- **传承激励**：后辈弟子以此为榜样，修炼热情高涨
- **势力变化**：飞升者的宗门声望大涨，附属势力增多
- **秘境开启**：飞升之地往往成为秘境，蕴含飞升者遗留的机缘

近万年来飞升者寥寥无几，每一次飞升都会被载入修仙界史册。

## 四、节气修炼

修仙界与凡间共用四时历法，但修士更注重节气对修炼的影响。天地灵气随节气流转，顺应节气修炼事半功倍。

### 4.1 四时八节

| 节气 | 时间 | 修炼要点 | 适宜活动 |
|------|------|----------|----------|
| 立春 | 正月 | 万物复苏，木灵气充盈 | 种植灵草，修炼木系功法 |
| 春分 | 二月 | 阴阳平衡，灵气平稳 | 突破瓶颈，炼制平衡类丹药 |
| 立夏 | 四月 | 火灵气渐盛 | 修炼火系功法，炼器 |
| 夏至 | 五月 | 阳气最盛，火灵气巅峰 | 渡劫（天劫威力相对稳定） |
| 立秋 | 七月 | 金灵气渐生 | 修炼金系功法，炼制飞剑 |
| 秋分 | 八月 | 阴阳再平衡 | 闭关突破，悟道参禅 |
| 立冬 | 十月 | 水灵气渐盛 | 修炼水系功法，祭炼水属性法宝 |
| 冬至 | 十一月 | 阴气最盛，水灵气巅峰 | 修炼阴属性功法，祭炼阴器 |

### 4.2 二十四节气与七十二候

二十四节气细分七十二候，每候五日。高阶修士会根据每候灵气变化微调修炼。

部分重要节气的特殊修炼价值：

| 节气 | 特殊之处 |
|------|----------|
| 惊蛰 | 雷灵气爆发，修炼雷系功法的最佳时机 |
| 清明 | 生死之气交替，修炼阴阳术法的良机 |
| 芒种 | 灵草成熟，采集灵药的黄金时期 |
| 白露 | 露水含灵，可炼制特殊丹药 |
| 霜降 | 肃杀之气弥漫，剑修磨砺剑意的好时机 |
| 大寒 | 极寒之中藏生机，冰系修士突破的契机 |

### 4.3 朔望月修炼

月相变化影响潮汐灵气，修士据此安排修炼。

| 月相 | 日期 | 灵气特点 | 修炼建议 |
|------|------|----------|----------|
| 朔日 | 初一 | 阴气最重，灵气内敛 | 修炼内功，收敛气息 |
| 上弦 | 初七初八 | 阴阳渐趋平衡 | 修炼攻防术法 |
| 望日 | 十五 | 阳气最重，灵气外放 | 修炼大威力术法，炼器炼丹 |
| 下弦 | 二十二二十三 | 阳气渐消 | 修炼恢复类功法，疗伤 |

部分修炼月系功法的修士，会在望日吸收月华精华，提升修为。

## 五、其他重要节日

### 5.1 宗门创立纪念日

每个宗门都有自己的创立之日。每逢此日，宗门上下同庆。

- 弟子放假一日（或数日）
- 宗门发放灵石、丹药等福利
- 举办比武大会，优胜者有额外奖励
- 掌门或长老讲道，传授修炼心得
- 百年、千年整数纪念尤为隆重

### 5.2 掌门继位大典

新掌门接任，是宗门权力交接的关键时刻。

继位大典流程：
1. 老掌门宣布传位决定
2. 新掌门展示实力，证明有资格胜任
3. 接受宗门信物（掌门印、镇宗法宝等）
4. 全体弟子行礼参拜新掌门
5. 昭告天下，各方势力送来贺礼

继位大典期间，常有不轨之徒趁机生事，因此宗门戒备森严。

### 5.3 送别宴

修士远行、闭死关、或赴险地之前，亲友设宴送别。

- 送别宴气氛或悲或壮，视情况而定
- 赠送护身法宝、保命丹药
- 共饮灵酒，互道珍重
- 若是赴死之行，则有"壮士一去不复返"的悲壮之意

### 5.4 迎宾大典

迎接贵客来访的正式仪式。

| 来客等级 | 迎接规格 |
|----------|----------|
| 普通修士 | 外门弟子迎接，安排客舍 |
| 金丹修士 | 长老出迎，以灵茶灵果款待 |
| 元婴修士 | 掌门亲迎，大开山门 |
| 大乘修士 | 全宗迎接，最高规格礼遇 |

迎宾大典体现宗门的待客之道，也展示宗门实力。

### 5.5 神丹大典

炼丹宗师炼出绝世神丹，会举办神丹大典。

- 展示神丹品质、功效
- 邀请各方势力观礼
- 拍卖或赐予有缘人
- 炼丹宗师借此扬名

神丹大典不定期举办，一旦举办必定轰动一方。高品阶神丹甚至能引来大乘修士亲自到场。

## 六、庆典习俗

### 6.1 贺礼规矩

修仙界送礼讲究门道。

| 送礼场合 | 常见贺礼 | 忌讳 |
|----------|----------|------|
| 进阶庆典 | 灵石、法器、丹药、灵材 | 不可送与修炼功法相克之物 |
| 宗门大典 | 灵脉地图、珍稀灵草、护山阵法 | 不可送带有诅咒或禁制之物 |
| 飞升庆典 | 仙器、仙丹、天材地宝 | 飞升后无法携带凡间之物，需送可入仙界之品 |

### 6.2 庆典禁忌

- 庆典期间不可动武，违者视为挑衅
- 不可送与受礼者功法相克的物品
- 丧事期间不参加庆典，视为不吉
- 低阶修士不可在高阶修士庆典上失礼

### 6.3 灵酒灵果

庆典必备之物。

| 灵酒 | 灵果 | 用途 |
|------|------|------|
| 百年灵泉酿 | 朱果 | 金丹以下修士饮用 |
| 千年桃花醉 | 火灵果 | 元婴修士饮用 |
| 万年仙露 | 悟道茶 | 大乘修士专用 |
