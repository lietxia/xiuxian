# 功德体系

> 功德是天道对善行的奖赏，可抵消业障、提升气运、帮助渡劫。

## 功德的本质

### 什么是功德？

功德是天道对善行的奖赏，是一种特殊的力量。功德可以抵消业障、提升气运、帮助渡劫，是修仙者追求的重要目标。

### 功德的特点

| 特点 | 说明 |
|------|------|
| 天道奖赏 | 天道对善行的奖赏 |
| 积累性 | 可以不断积累 |
| 消耗性 | 可以被消耗 |
| 转化性 | 可以转化为其他力量 |
| 保护性 | 可以保护修仙者 |

5. **因果关系**：因果越深，功德越多
