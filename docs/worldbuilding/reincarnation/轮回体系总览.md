# 轮回体系总览

## 一、六道轮回

轮回乃天地大道运转之根基，万物生灵死后灵魂归于轮回，根据生前因果业力分配至六道之一。

| 道名 | 属性 | 入道条件 | 代表 |
|------|------|----------|------|
| 天道 | 至善 | 功德深厚、修行有成 | 仙神转世 |
| 人道 | 平衡 | 无大善大恶 | 凡人轮回 |
| 阿修罗道 | 嗜战 | 好勇斗狠、战功赫赫 | 战魂转生 |
| 畜生道 | 愚痴 | 愚昧无知、造下口业 | 妖兽化形 |
| 饿鬼道 | 贪婪 | 贪得无厌、掠夺资源 | 饿鬼受苦 |
| 地狱道 | 极恶 | 罪大恶极、十恶不赦 | 受刑赎罪 |

六道之中，天道与地狱道最为难得。天道需累世修行功德圆满，地狱道则需犯下滔天罪孽。多数灵魂在人道与畜生道间流转。

## 二、轮回运作机制

灵魂进入轮回需经历完整流程：

**第一步：魂归地府**
修士死后，灵魂脱离肉身，被地府阴差接引。若灵魂强大（元婴期以上），可自行前往地府，或被天道法则强制牵引。

**第二步：判官殿审判**
判官查阅生死簿，核算生前功过。功德与业力相抵，得出净业值：
- 净业值 > 1000：入天道
- 净业值 100~1000：入人道
- 净业值 -100~100：入阿修罗道
- 净业值 -1000~-100：入畜生道
- 净业值 < -1000：入饿鬼道或地狱道

**第三步：忘川河洗魂**
灵魂饮下忘川河水，前世记忆逐渐消散。记忆消散速度与灵魂强度成反比：
- 凡人灵魂：彻底遗忘
- 金丹修士：残留模糊印象
- 元婴修士：保留部分记忆碎片
- 化神以上：可抵抗大部分遗忘

**第四步：奈何桥过境**
踏上奈何桥，前世因果化为锁链缠绕灵魂。因果越重，锁链越多，过桥越艰难。

**第五步：投胎转世**
根据审判结果，灵魂投入对应道中，开启新生。

## 三、跳出轮回的途径

### 修炼成仙
最正统的方式。修士境界突破至渡劫期，渡过天劫后飞升仙界，灵魂烙印于天道，不再受轮回束缚。需满足：
- 境界达到渡劫期
- 渡过九重天劫
- 功德值 >= 5000

### 悟透轮回法则
极少数天才可参悟轮回大道本身，以法则之力超脱轮回。如仙逆中王林悟透轮回，一念可定生死。需满足：
- 境界达到大乘期
- 参悟轮回法则至大成
- 灵魂强度 >= 10000

### 斩却三尸
道教正统之法，斩去善念、恶念、执念，灵魂纯净无暇，自然脱离轮回。需满足：
- 境界达到合体期
- 心境圆满
- 斩去至少一尸

## 四、轮回与因果

轮回与因果互为表里。因果乃轮回之因，轮回乃因果之果。

**因果积累规则：**
- 杀生：每杀一人，因果+10（修士+100）
- 救人：每救一人，因果-10（修士-100）
- 誓言未践：因果+500
- 恩将仇报：因果+1000

**因果对轮回的影响：**
- 因果值 < 0：轮回时获得更好出身
- 因果值 0~1000：正常轮回
- 因果值 > 1000：轮回时遭遇磨难
- 因果值 > 10000：可能堕入畜生道或地狱道

**因果线：**
大能修士可看见因果线，每条线连接着前世今生的恩怨。因果线越多，灵魂负担越重，突破境界时心魔越强。

## 五、特殊轮回现象

**天道轮回：** 大能陨落后，天道会为其保留一缕真灵，待时机成熟转世重修。如遮天中帝尊的布局。

**强制轮回：** 犯下大罪的修士，会被天道强制打入轮回，且每一世都饱受苦难，直到业力消除。

**轮回印记：** 跳出轮回的修士，灵魂上会留下轮回印记，可凭此印记进入轮回救人或惩罚恶人。

**轮回空间：** 传说中存在一处轮回空间，乃轮回法则的具现化，进入其中可直面前世今生，但十死无生。
