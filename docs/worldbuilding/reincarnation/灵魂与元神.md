# 灵魂与元神

## 一、三魂七魄

灵魂由三魂七魄构成，乃生灵之根本。修士修炼，本质是壮大三魂七魄，最终凝聚元神。

### 三魂

| 魂名 | 属性 | 功能 | 壮大方式 |
|------|------|------|----------|
| 天魂 | 命运 | 连接天道，感应因果 | 渡劫、悟道 |
| 地魂 | 根基 | 承载修为，稳固境界 | 打坐、炼丹 |
| 人魂 | 意识 | 主导思维，决定性格 | 阅历、感悟 |

三魂缺一不可。天魂受损则命运多舛，地魂受损则修为倒退，人魂受损则意识混乱。

### 七魄

| 魄名 | 功能 | 对应情绪 |
|------|------|----------|
| 天冲魄 | 直觉感应 | 恐惧 |
| 灵慧魄 | 智慧悟性 | 惊讶 |
| 气魄 | 灵力运转 | 愤怒 |
| 力魄 | 肉身力量 | 喜悦 |
| 中枢魄 | 协调平衡 | 悲伤 |
| 精魄 | 生命力 | 厌恶 |
| 英魄 | 战斗本能 | 欲望 |

七魄齐全则神魂稳固。每损失一魄，灵魂强度下降15%，且会失去对应情绪的控制。

## 二、灵魂强度与境界对应

灵魂强度是衡量灵魂质量的核心指标，与境界密切相关：

| 境界 | 灵魂强度 | 灵魂形态 | 特殊能力 |
|------|----------|----------|----------|
| 练气期 | 1~10 | 虚幻魂影 | 无 |
| 筑基期 | 10~50 | 凝实魂影 | 灵魂感知 |
| 金丹期 | 50~200 | 魂光初现 | 灵魂攻击 |
| 元婴期 | 200~1000 | 元神雏形 | 元神出窍 |
| 化神期 | 1000~5000 | 元神成型 | 元神分身 |
| 合体期 | 5000~20000 | 元神不灭 | 元神寄体 |
| 大乘期 | 20000~100000 | 元神化道 | 一念成界 |
| 渡劫期 | 100000+ | 元神永恒 | 超脱轮回 |

灵魂强度不仅影响战斗，还决定修炼速度、悟性上限、渡劫成功率。

## 三、元神凝聚过程

元神是灵魂的高级形态，凝聚过程贯穿多个境界：

### 金丹期：初步凝聚
金丹内孕育一缕元神之力，灵魂开始由虚转实。此时元神脆弱，无法脱离肉身。

**标志：** 灵魂可内视金丹，感知范围扩大至百里。

### 元婴期：凝聚成型
元神与金丹融合，形成元婴。元婴乃修士第二生命，肉身毁灭后可凭元婴存活。

**标志：** 元神可短时间出窍，持续时间与灵魂强度成正比。

### 化神期：完善稳固
元婴蜕变为真正的元神，具备独立意识与战斗能力。可分化元神分身。

**标志：** 元神可长时间离体，分身拥有本体30%战力。

### 合体期：不灭元神
元神与肉身完美融合，肉身不灭则元神不灭。即便肉身毁灭，元神也可夺舍重生。

**标志：** 元神强度突破10000，具备不灭特性。

## 四、灵魂攻击与防御

### 灵魂攻击手段

| 攻击方式 | 威力系数 | 消耗 | 特点 |
|----------|----------|------|------|
| 神识冲击 | 1.0x | 低 | 基础攻击 |
| 灵魂风暴 | 2.0x | 中 | 范围攻击 |
| 元神出窍 | 3.0x | 高 | 直接攻击 |
| 搜魂术 | - | 中 | 获取记忆 |
| 夺舍 | - | 极高 | 占据肉身 |

### 灵魂防御手段

**神识护盾：** 以神识构建防御屏障，可抵挡灵魂攻击。强度与灵魂强度成正比。

**心如止水：** 心境圆满者，灵魂攻击效果减半。

**灵魂法宝：** 某些法宝专门用于防御灵魂攻击，如养魂玉、镇魂钟。

**元神铠甲：** 化神期以上可凝聚元神铠甲，大幅提升灵魂防御。

## 五、元神出窍的限制与风险

元神出窍是强大修士的标志性能力，但存在严格限制：

**时间限制：**
- 元婴期：最多出窍1时辰
- 化神期：最多出窍1日
- 合体期：最多出窍1月
- 大乘期：无限制

**距离限制：**
- 元婴期：最远100里
- 化神期：最远1000里
- 合体期：最远10000里
- 大乘期：无限制

**风险：**
- 元神出窍期间，肉身极度脆弱
- 元神离体过久会逐渐虚弱
- 遭遇灵魂攻击可能元神受损
- 元神被灭则彻底陨落

## 六、夺舍规则

夺舍是占据他人肉身的禁术，规则严苛：

### 夺舍条件
- 夺舍者灵魂强度必须高于被夺舍者
- 被夺舍者灵魂必须虚弱或已被压制
- 每具肉身只能被夺舍一次
- 夺舍间隔不得少于百年

### 夺舍限制
- 夺舍者灵魂强度下降50%
- 新肉身与灵魂契合度最多70%
- 夺舍后修为下降一个大境界
- 存在被反噬的风险

### 夺舍副作用
- 性格逐渐被原肉身影响
- 修炼速度下降30%
- 突破瓶颈难度+100%
- 天道降下惩罚，渡劫威力+50%

夺舍乃邪道之术，正道修士不屑为之。但在生死存亡之际，夺舍仍是最后的保命手段。
