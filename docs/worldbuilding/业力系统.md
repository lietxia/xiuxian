# 业力系统

> 业力是天道对众生行为的记录与反馈，善恶皆有报，因果循环不爽。业力深重者，天劫加重、气运衰减、走火入魔；业力清轻者，天道眷顾、机缘不断。

---

## 业力的本质

### 什么是业力？

业力是天道对一切生灵行为的记录。每一次行为，无论善恶，都会在天道中留下印记，形成业力。业力不是简单的惩罚或奖赏，而是天道维持平衡的机制。

### 业力的三大类型

| 类型 | 来源 | 特点 | 影响 |
|------|------|------|------|
| 因果业力 | 一切因果行为 | 最为基础，无处不在 | 影响命运走向 |
| 杀戮业力 | 杀生行为 | 最为暴烈，难以消除 | 影响心性和天劫 |
| 情劫业力 | 情感纠葛 | 最为纠缠，牵连深远 | 影响修为和转世 |

### 业力的特点

| 特点 | 说明 |
|------|------|
| 记录性 | 天道记录一切行为，无法逃避 |
| 累积性 | 业力会不断累积，不会自动消失 |
| 流转性 | 业力可以流转，影响他人 |
| 平衡性 | 善恶业力相互制衡 |
| 觉醒性 | 业力会在特定时刻觉醒 |

| 规则 | 说明 | 例外 |
