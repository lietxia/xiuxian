# 修仙界刑罚与追杀

修仙界的刑罚体系简单粗暴，核心原则只有一个：实力决定一切。

## 宗门内部惩罚

宗门对违规弟子的惩罚，根据严重程度分级。

| 惩罚类型 | 适用对象 | 具体措施 | 恢复可能 |
|----------|----------|----------|----------|
| 罚贡献点 | 轻微违规 | 扣除100-1000贡献点 | 重新积累即可 |
| 禁闭思过 | 较重违规 | 关入禁闭室三个月至三年 | 期满释放 |
| 废除修为 | 严重违规 | 打散丹田，废除全部修为 | 几乎不可能恢复 |
| 逐出宗门 | 极重违规 | 剥夺弟子身份，逐出山门 | 永久 |
| 处死 | 极其严重 | 当众处死，以儆效尤 | 无 |

### 废除修为的细节

废除修为是仅次于死刑的重刑。执行时，由长老亲自出手，用特殊手法打散弟子丹田中的灵力，摧毁经脉中的灵力循环。

| 被废修为后的状态 | 说明 |
|------------------|------|
| 寿命 | 恢复凡人寿命，可能迅速衰老 |
| 体质 | 灵力反噬，体质大幅下降 |
| 再修炼 | 几乎不可能，丹田已毁 |
| 社会地位 | 沦为废人，无人愿意接触 |

## 修仙界通缉

对于叛逃、投敌、残害同门等重罪，宗门会发布通缉令。

| 通缉类型 | 发布者 | 赏金范围 | 有效期 |
|----------|--------|----------|--------|
| 悬赏令 | 宗门、世家 | 1000-100000下品灵石 | 直到目标死亡 |
| 追杀令 | 大宗门、联盟 | 100000-1000000下品灵石 | 直到目标死亡 |
| 必杀令 | 顶级势力 | 无上限 | 永久有效 |

### 悬赏等级

| 等级 | 目标境界 | 赏金(下品灵石) | 追杀者要求 |
|------|----------|----------------|------------|
| 铜级 | 练气期 | 1000-5000 | 任何修士 |
| 银级 | 筑基期 | 10000-50000 | 筑基期以上 |
| 金级 | 金丹期 | 100000-500000 | 金丹期以上 |
| 玄级 | 元婴期 | 1000000+ | 元婴期以上 |
| 天级 | 化神期以上 | 无价 | 不限 |

## 赏金猎人

赏金猎人是专门追杀通缉犯的修士群体。

| 特征 | 说明 |
|------|------|
| 组织形式 | 多为松散联盟，也有少数严密组织 |
| 收入来源 | 悬赏赏金、通缉犯随身财物 |
| 风险 | 极高，通缉犯往往走投无路、拼死反抗 |
| 社会地位 | 两极分化，强者受人尊敬，弱者被人轻视 |

赏金猎人公会在各大坊市都有据点，提供通缉信息、任务接取、赏金结算等服务。

## 逃亡与隐匿手段

被通缉的修士会想尽办法逃亡和隐匿。

| 手段 | 效果 | 风险 | 成本 |
|------|------|------|------|
| 改头换面术 | 改变容貌、气息 | 高阶修士可识破 | 低 |
| 夺舍 | 占据他人身体 | 可能失败，魂飞魄散 | 极高 |
| 隐居深山 | 远离人烟 | 资源匮乏，修炼停滞 | 无 |
| 投靠敌对势力 | 换取庇护 | 可能被出卖 | 高 |
| 伪造身份 | 获取新身份证明 | 可能被查出 | 中 |

### 改头换面术详解

| 术法等级 | 效果 | 持续时间 | 识破难度 |
|----------|------|----------|----------|
| 低阶易容术 | 改变容貌 | 数天 | 容易 |
| 中阶易容术 | 改变容貌和气息 | 数月 | 较难 |
| 高阶易容术 | 彻底改变外貌和气息 | 数年 | 很难 |
| 夺舍重生 | 占据他人身体 | 永久 | 几乎无法识破 |

## 身份玉牌与追踪

身份玉牌是修仙界证明身份的重要工具。

| 玉牌类型 | 功能 | 制作者 | 追踪可能 |
|----------|------|--------|----------|
| 宗门弟子牌 | 证明宗门身份、记录贡献点 | 宗门炼器师 | 可远程定位 |
| 坊市通行牌 | 进出坊市的凭证 | 坊市管理方 | 记录出入时间 |
| 散修身份牌 | 证明散修身份 | 散修联盟 | 无追踪功能 |
| 世家血脉牌 | 证明世家血脉 | 世家炼器师 | 可感应血脉 |

### 身份玉牌的追踪机制

大宗门的弟子牌内置追踪阵法，只要弟子还活着，宗门就能大致定位其位置。

| 追踪距离 | 精度 | 说明 |
|----------|------|------|
| 千里内 | 精确位置 | 宗门山门范围内 |
| 万里内 | 大致区域 | 宗门势力范围内 |
| 十万里内 | 模糊方向 | 需要特殊阵法辅助 |

被废除修为的弟子，其弟子牌会被当场销毁。叛逃的弟子，宗门会启动追踪阵法，配合追杀令，让叛逃者无处可逃。

修仙界的刑罚与追杀，本质上是维护秩序的手段。没有这些威慑，修仙界将彻底沦为弱肉强食的丛林。但这些手段也充满了残酷和不公，低阶修士往往没有反抗的余地，只能任人宰割。
