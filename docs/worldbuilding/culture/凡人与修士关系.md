# 凡人与修士关系

凡人与修士共存于同一片天地，却生活在截然不同的世界里。两者之间的关系复杂而微妙。

## 凡人国度与修仙界的关系

凡人国度与修仙势力之间存在着复杂的互动。

| 关系类型 | 说明 | 典型案例 |
|----------|------|----------|
| 依附关系 | 凡人国度依附于大宗门，提供资源和人口 | 赵国依附于掩月宗 |
| 独立关系 | 凡人国度独立存在，与修仙界互不干涉 | 凡人小国自生自灭 |
| 控制关系 | 修仙势力直接控制凡人国度 | 魔道宗门控制傀儡政权 |
| 合作关系 | 双方平等合作，互通有无 | 大型坊市与周边国家 |

### 凡人国度对修仙界的贡献

| 贡献类型 | 说明 | 回报 |
|----------|------|------|
| 人口基数 | 提供灵根检测的候选者 | 修士庇护 |
| 凡人资源 | 粮食、矿产、手工制品 | 灵石、丹药 |
| 世俗权力 | 管理凡人社会秩序 | 政治支持 |
| 情报网络 | 凡人遍布各地，可收集信息 | 保护与赏赐 |

## 凡人对修士的态度

凡人对修士的态度复杂多样。

| 态度类型 | 表现 | 占比 | 说明 |
|----------|------|------|------|
| 敬畏 | 虔诚供奉、不敢冒犯 | 60% | 大多数凡人的态度 |
| 恐惧 | 躲避修士、不敢靠近 | 25% | 见识过修士力量的凡人 |
| 利用 | 试图借助修士力量 | 10% | 皇室、贵族的态度 |
| 仇恨 | 憎恨修士的傲慢与残忍 | 5% | 受过修士迫害的凡人 |

### 敬畏心理的表现

| 表现 | 说明 |
|------|------|
| 建立庙宇 | 供奉修士为仙人 |
| 主动献祭 | 献上子女、财物 |
| 绝对服从 | 不敢违抗修士命令 |
| 传说美化 | 将修士事迹神化 |

## 修士对凡人的态度

修士对凡人的态度同样复杂。

| 态度类型 | 表现 | 占比 | 说明 |
|----------|------|------|------|
| 漠视 | 视凡人为蝼蚁 | 50% | 大多数修士的态度 |
| 庇护 | 保护凡人不受妖兽侵害 | 20% | 有责任感的修士 |
| 利用 | 利用凡人收集资源、情报 | 20% | 务实的修士 |
| 尊重 | 平等对待凡人 | 10% | 出身凡人的修士 |

### 漠视心理的表现

| 表现 | 说明 |
|------|------|
| 视而不见 | 凡人苦难与己无关 |
| 随意处置 | 为了方便可以牺牲凡人 |
| 高高在上 | 认为凡人不配与修士平等 |
| 缺乏共情 | 无法理解凡人的生死悲欢 |

## 凡人修仙的可能性

凡人想要修仙，必须具备灵根。

| 灵根检测方式 | 成本 | 准确率 | 可检测年龄 |
|--------------|------|--------|------------|
| 宗门定期检测 | 免费 | 99% | 6-16岁 |
| 坊市散修检测 | 10下品灵石 | 90% | 任意年龄 |
| 世家内部检测 | 无 | 99% | 任意年龄 |
| 自行感应 | 无 | 30% | 任意年龄 |

### 灵根出现概率

| 群体 | 有灵根概率 | 说明 |
|------|------------|------|
| 普通凡人 | 1% | 大多数灵根修士的来源 |
| 修士后代 | 10% | 血脉传承的优势 |
| 世家子弟 | 30% | 多代血脉优化的结果 |
| 特殊体质者 | 50% | 体质与灵根相关 |

## 凡人国度中的修仙者势力

修仙者在凡人国度中往往拥有超然地位。

| 势力类型 | 说明 | 典型表现 |
|----------|------|----------|
| 国师 | 凡人国度的修仙顾问 | 指导皇室修炼、处理修士事务 |
| 护国修士 | 保护凡人国度的修仙者 | 抵御妖兽、击退敌国修士 |
| 世家分支 | 修仙世家在凡人国度的分支 | 经营产业、收集资源 |
| 隐居散修 | 隐居在凡人国度的散修 | 不问世事、独自修炼 |

### 国师的权力与责任

| 权力 | 责任 |
|------|------|
| 参与国家决策 | 保护皇室安全 |
| 调动世俗资源 | 处理修士纠纷 |
| 免除法律约束 | 抵御外部修士威胁 |
| 享受皇室供奉 | 指导皇室子弟修炼 |

## 凡人战争与修士干预

凡人之间的战争，有时会受到修士的干预。

| 干预类型 | 说明 | 动机 | 后果 |
|----------|------|------|------|
| 直接参战 | 修士加入凡人军队 | 收取报酬或政治目的 | 战争迅速结束 |
| 提供武器 | 提供法器、符箓 | 交易或支持 | 战争天平倾斜 |
| 情报支持 | 提供敌方情报 | 收取报酬 | 战略优势 |
| 调停战争 | 修士出面调停 | 维护地区稳定 | 战争停止 |

### 修士干预的规则

| 规则 | 说明 |
|------|------|
| 不得屠戮凡人 | 修士不得对凡人大规模杀伤 |
| 不得毁灭国度 | 修士不得毁灭凡人国家 |
| 适度干预 | 干预程度应有限度 |
| 收取报酬 | 干预应获得相应报酬 |

这些规则并非铁律，强大的修士可以无视这些规矩。但大多数修士还是会遵守，因为屠戮凡人会引来其他修士的讨伐。

凡人与修士的关系，本质上是强者与弱者的关系。凡人没有选择的权利，只能仰望修士的背影。而修士则掌握着绝对的权力，可以随意决定凡人的命运。这种关系冷酷而现实，却是修仙界的基本法则。
