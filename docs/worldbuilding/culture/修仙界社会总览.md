# 修仙界社会总览

修仙界并非铁板一块的统一社会，而是由无数宗门、世家、散修势力交织而成的松散联盟。修士与凡人共存于同一片天地，却生活在截然不同的世界里。

## 社会阶层划分

修仙界的社会结构呈金字塔形，越往上人数越少，权力越大。

| 阶层 | 占比 | 典型境界 | 社会地位 |
|------|------|----------|----------|
| 顶层大能 | 0.01% | 大乘期以上 | 一言可定宗门兴衰 |
| 宗门高层 | 0.1% | 元婴期以上 | 掌控一方势力 |
| 核心弟子 | 1% | 金丹期 | 宗门未来栋梁 |
| 内门弟子 | 5% | 筑基期 | 宗门中坚力量 |
| 外门弟子 | 15% | 练气期 | 宗门基础力量 |
| 散修 | 30% | 练气期为主 | 无依无靠，自生自灭 |
| 凡人 | 53% | 无修为 | 修士眼中的蝼蚁 |

### 四大群体特征

**散修**：无宗门庇护的独立修士，占修士总数过半。资源匮乏，功法残缺，寿命短暂。多数散修终生困于练气期，能筑基者百中无一。

**宗门修士**：依附于宗门的正式弟子，享受系统培养。从外门到核心，层层筛选，优胜劣汰。

**世家子弟**：传承数百上千年的修仙家族后裔，血脉中可能蕴含特殊体质或灵根。起点高于常人，但内部竞争残酷。

**皇族修士**：凡人国度中皇室血脉的修仙者，数量稀少，但掌握世俗权力与修仙资源的双重优势。

## 修士与凡人的关系

修士与凡人之间存在一道无形的鸿沟。

| 方面 | 修士视角 | 凡人视角 |
|------|----------|----------|
| 寿命 | 数百年起步 | 不过百年 |
| 力量 | 移山填海 | 手无缚鸡之力 |
| 资源 | 灵石丹药 | 金银粮食 |
| 社会地位 | 高高在上 | 仰望敬畏 |

修仙界有条不成文的规矩：修士不得随意屠戮凡人。这条规矩并非出于善意，而是因为凡人是修士的根基，没有凡人繁衍，灵根修士的出现概率将大幅降低。

## 社会流动渠道

修仙界的阶层并非完全固化，存在几条上升通道：

| 渠道 | 成功率 | 风险 | 说明 |
|------|--------|------|------|
| 灵根检测 | 1% | 低 | 凡人中检测出灵根者可入宗门 |
| 宗门试炼 | 10% | 中 | 外门晋升内门的常规途径 |
| 秘境探险 | 5% | 高 | 高风险高回报，可能一步登天 |
| 师徒传承 | 30% | 低 | 被高人看中收为弟子 |
| 联姻攀附 | 20% | 中 | 通过婚姻进入更高阶层 |

## 修仙界法律与执法

修仙界没有统一的法律体系，但各势力都有自己的规矩。

**执法堂**：大宗门内部设立的执法机构，负责维护宗门秩序。执法堂弟子通常修为精深，铁面无私，直接听命于掌门或太上长老。

**赏罚制度**：

| 行为 | 奖励 | 惩罚 |
|------|------|------|
| 完成宗门任务 | 贡献点、灵石 | 无 |
| 发现重大矿脉 | 直接晋升、重赏 | 无 |
| 泄露宗门秘密 | 无 | 废修为、逐出宗门 |
| 残害同门 | 无 | 处死或永久囚禁 |
| 叛逃投敌 | 无 | 全界追杀、格杀勿论 |

## 基本社交礼仪

修仙界的社交规则简单粗暴：实力为尊。

| 场景 | 礼仪规范 |
|------|----------|
| 初次见面 | 以境界高低决定称呼，低阶修士需主动行礼 |
| 论道交流 | 平辈论交，不得随意打断 |
| 拜访宗门 | 需提前通报，不得擅闯山门 |
| 交易买卖 | 买卖公平，强买强卖者遭坊市执法队驱逐 |
| 结伴探险 | 提前约定分配方式，事后不得反悔 |

低阶修士在高阶修士面前需保持恭敬，这不是礼貌问题，而是生存问题。冒犯高阶修士，轻则被教训，重则被灭杀，无人会为你主持公道。

修仙界的本质是弱肉强食，所有的礼仪规矩都建立在实力对等的前提下。两个金丹修士可以平起平坐，但一个练气修士在金丹修士面前，连说话的资格都没有。
