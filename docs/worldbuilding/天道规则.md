# 天道规则

> 天道是修仙世界的最高法则，无形无质却无处不在。天道维持世界平衡，干预重大事件，考验修仙者。

## 天道的本质

### 什么是天道？

天道是修仙世界的最高法则，是宇宙运行的根本规律。天道无形无质，却无处不在，它维持着世界的平衡，干预着重大的事件，考验着每一个修仙者。

### 天道的特点

| 特点 | 说明 |
|------|------|
| 无形无质 | 天道没有具体的形态 |
| 无处不在 | 天道存在于世界的每一个角落 |
| 公正无私 | 天道对所有生灵一视同仁 |
| 维持平衡 | 天道维持世界的平衡 |
| 干预重大事件 | 天道会在关键时刻干预 |

4. **业障**：天道对恶行的惩罚
5. **气运**：天道对生灵的眷顾
