# 修仙禁忌与规则完整体系

> 本文档详细定义修仙世界的天道规则、修炼禁忌、因果法则与功德体系。

---

## 一、天道规则概述

### 天道的本质

天道是修仙世界的最高法则，无形无质，却无处不在。它维持着世界的运行，对一切违规行为进行惩罚。

**表现形式**：
- 天劫：对修炼者的考验与惩罚
- 功德：对善行的奖励
- 因果：对行为的记录与回报
- 天道意志：特殊情况下的直接干预

### 天道意志的表现

| 表现 | 触发条件 | 效果 |
|------|----------|------|
| 天劫降临 | 修为突破 | 考验/惩罚 |
| 功德金光 | 大善之举 | 加持保护 |
| 天罚 | 大恶之举 | 直接惩罚 |
| 天机示警 | 重大变故 | 预兆 |

### 天道干预的机制

天道并非直接干预一切，而是通过法则自动运行：
1. **因果自动记录**：无需天道主动
2. **天劫自动降临**：境界提升时自动触发
3. **功德自动累积**：行善自动记录
4. **特殊干预**：仅极少数情况天道主动介入

## 二、修炼禁忌

### 禁止屠城灭国

**规则**：修士不得以法力屠灭凡人城池或国家
**原因**：凡人是天地的根基，大量杀戮破坏天地平衡
**违反后果**：
- 天劫强度×3
- 功德清零
- 气运暴跌
- 可能被天道直接惩罚

### 禁止大规模吸取凡人精血

**规则**：不得以凡人精血修炼
**原因**：侵害无辜，违反天道仁慈
**违反后果**：心魔滋生、天劫加重

### 禁止破坏天地根基

**规则**：不得毁坏灵脉源眼、破坏天地框架
**原因**：灵脉是天地灵气的根本
**违反后果**：天道追杀

### 禁止窥探天机

**规则**：不得过度窥探未来
**原因**：泄露天机会导致因果紊乱
**例外**：天机阁有部分权限
**违反后果**：反噬、修为受损

### 禁止逆天改命（特定情况）

**规则**：不得强行改变他人命格
**原因**：命格由因果决定
**例外**：以自身功德/寿命为代价
**违反后果**：天道惩罚

## 四、违反规则的后果

### 天谴（雷劫加重）

| 原违规行为 | 天劫加重程度 |
|------------|--------------|
| 杀戮凡人 | +1-2道天雷 |
| 大规模破坏 | +3-5道天雷 |
| 逆天之举 | 天劫无法度过 |

### 功德扣除

| 违规行为 | 功德扣除 |
|----------|----------|
| 轻微违规 | 10-100 |
| 中等违规 | 100-1000 |
| 严重违规 | 全部扣除 |

### 气运下降

| 违规程度 | 气运影响 |
|----------|----------|
| 轻微 | 运势略差 |
| 中等 | 修炼困难 |
| 严重 | 灾祸不断 |

### 心魔滋生

恶行越多，心魔越强：
- 轻度：修炼分心
- 中度：突破困难
- 重度：可能走火入魔

### 天道追杀

仅针对极端违规者：
- 天道降下雷罚
- 无处可逃
- 形神俱灭

## 五、特殊豁免

### 天命之人的豁免

- 天道选定的"天命之子"
- 部分规则不适用
- 有特定的使命
- 豁免范围有限

### 大功德者的豁免

- 功德极高者可抵消部分惩罚
- 但不是无限的
- 功德与业力可以互相抵消

### 天庭授权的豁免

- 仙界天帝可授权特定行为
- 如：下界除魔
- 需要正式授权

### 因果循环中的豁免

- 如果对方先欠因果
- 反击不算违规
- 因果了结为止

## 六、因果法则

### 因果的产生

**善因**：救人、行善、积累功德
**恶因**：杀人、作恶、破坏天地

### 因果的偿还

| 因果类型 | 偿还方式 |
|----------|----------|
| 善因未报 | 突破顺利、遇贵人 |
| 恶因未报 | 修炼困难、遇劫难 |
| 因果已了 | 互不相欠 |

### 因果与修行的关系

- 因果越重，突破越难
- 大因果需要在渡劫时偿还
- 了结因果是一种修行

### 斩断因果的方法

1. **自然了结**：因果自行了结
2. **功德抵消**：用功德抵消
3. **因果术**：高深法术斩断（极危险）
4. **天劫中偿还**：在渡劫时清算
