# 功德业力气运量化体系

> 天道至公，赏善罚恶。功德、业力、气运三者相互交织，共同构成修仙世界的因果法则。功德多则天道眷顾，业力重则天道惩罚，气运则是二者博弈的外在体现。

---

## 功德量化表

功德是修士行善积德、维护天道秩序所获得的天道奖赏。功德越高，天道眷顾越深。

### 功德获取途径

| 善行类型 | 功德值范围 | 说明 |
|----------|------------|------|
| 救人一命 | 100-500 | 视被救者身份与因果而定，救凡人100，救修士300，救大能500 |
| 护佑百姓 | 1000-5000 | 长期庇护一方百姓，视庇护范围与时间而定 |
| 斩杀邪修 | 200-2000 | 视邪修恶业深浅而定，恶业越重，功德越多 |
| 著书立说 | 500-3000 | 传播修炼心得或天道感悟，视影响力而定 |
| 修复灵脉 | 1000-5000 | 修复受损灵脉，恢复天地灵气循环 |
| 平息小劫 | 5000-20000 | 化解小规模天地劫难，如妖兽潮、魔气爆发 |
| 平息大劫 | 50000+ | 化解大规模天地劫难，如域外天魔入侵、天地大劫 |
| 拯救一界 | 100000+ | 拯救整个界面免于毁灭，功德无量 |

### 功德获取注意事项

- **动机纯正**：为求功德而行善，功德减半；发自本心行善，功德全得
- **因果了结**：善行需了结因果，若行善后又反悔或索取回报，功德扣除
- **量力而行**：超出自身能力范围的善行，若失败则不扣功德，若成功则功德加倍
- **天道记录**：功德由天道自动记录，无法伪造或篡改

## 业力量化表

业力是修士作恶多端、破坏天道秩序所获得的天道惩罚。业力越重，天道惩罚越严厉。

### 业力获取途径

| 恶行类型 | 业力值范围 | 说明 |
|----------|------------|------|
| 杀戮无辜凡人 | 500/人 | 每杀一个无辜凡人，增加500点业力 |
| 杀修士 | 200-1000 | 视被杀修士身份与因果而定，杀无辜修士业力更重 |
| 夺舍 | 3000-10000 | 夺舍他人肉身，视夺舍次数与对象而定 |
| 毁坏灵脉 | 5000-50000 | 毁坏灵脉会导致天地灵气紊乱，业力极重 |
| 修炼禁术 | 1000-10000 | 修炼天道禁止的功法，视禁术等级而定 |
| 欺骗天道 | 10000+ | 试图欺骗天道或篡改天道规则，业力极重 |
| 屠城灭族 | 50000+ | 大规模屠杀，业力滔天，天道必降天劫 |
| 篡改因果 | 20000-100000 | 强行篡改他人命运或因果线，视影响范围而定 |

### 业力特殊规则

- **因果牵连**：若因你的行为间接导致他人死亡，业力减半计入
- **不知者不罪**：若确实不知行为会造成恶果，业力减半
- **主动赎罪**：若主动弥补过错，可减少对应业力
- **业力转移**：某些禁术可将业力转移给他人，但会增加额外业力
- **业力累积**：业力不会随时间消散，只会越积越多

## 业力反噬机制

业力积累到一定程度，会触发天道惩罚，即"业力反噬"。业力越高，反噬越严重。

| 业力等级 | 业力值范围 | 反噬表现 | 持续效果 |
|----------|------------|----------|----------|
| 轻微 | 0-1000 | 轻微霉运 | 偶尔遭遇小麻烦，如法宝失灵、丹药炼废 |
| 中等 | 1000-5000 | 修炼瓶颈 | 修炼速度下降30%-50%，突破概率降低 |
| 严重 | 5000-20000 | 心魔入侵 | 心魔频繁出现，道心不稳，容易走火入魔 |
| 重大 | 20000-50000 | 天劫加重 | 天劫威力增强50%-200%，渡劫成功率大幅下降 |
| 滔天 | 50000+ | 因果大劫 | 天道直接降下因果大劫，几乎无生还可能 |

### 业力反噬具体表现

**轻微霉运（0-1000）**
- 炼丹时偶尔炸炉
- 出门容易遇到小妖兽
- 法宝使用时偶尔失灵
- 与人交易时容易被坑

**修炼瓶颈（1000-5000）**
- 修炼速度下降30%-50%
- 突破境界的概率降低
- 功法运转时偶尔出现滞涩
- 灵力吸收效率下降

**心魔入侵（5000-20000）**
- 心魔频繁出现，干扰修炼
- 道心不稳，容易产生杂念
- 修炼时容易走火入魔
- 睡眠时容易被噩梦侵扰

**天劫加重（20000-50000）**
- 天劫威力增强50%-200%
- 渡劫成功率大幅下降
- 天劫中会出现额外考验
- 渡劫失败后修为跌落更严重

**因果大劫（50000+）**
- 天道直接降下因果大劫
- 因果大劫威力远超普通天劫
- 几乎无生还可能
- 即使侥幸存活，也会被打落凡尘

## 气运等级表

气运是修士命运走向的外在体现，受功德与业力共同影响。气运越高，机缘越多；气运越低，灾祸越多。

| 气运等级 | 气运值范围 | 外在表现 | 对修炼的影响 |
|----------|------------|----------|--------------|
| 天弃 | 0-10 | 诸事不顺，灾祸连连 | 修炼速度-80%，突破概率-90% |
| 低迷 | 10-30 | 偶有小成，多有挫折 | 修炼速度-50%，突破概率-60% |
| 平平 | 30-50 | 无功无过，平淡度日 | 修炼速度正常，突破概率正常 |
| 旺盛 | 50-70 | 机缘不断，顺风顺水 | 修炼速度+30%，突破概率+40% |
| 昌隆 | 70-90 | 天道眷顾，万事亨通 | 修炼速度+60%，突破概率+80% |
| 应劫之人 | 90-100 | 天命所归，气运之子 | 修炼速度+100%，突破概率+150% |

### 气运外在表现

**天弃之人气运（0-10）**
- 出门必遇妖兽
- 炼丹必定炸炉
- 寻宝必定空手而归
- 与人交易必定被坑
- 修炼时容易走火入魔

**低迷之人气运（10-30）**
- 偶尔能获得小机缘
- 但总是在关键时刻功亏一篑
- 容易被卷入麻烦之中
- 修炼时偶尔遭遇瓶颈

**平平之人气运（30-50）**
- 生活平淡，无大起大落
- 修炼按部就班，无特殊机缘
- 偶尔能获得小宝物
- 突破境界需靠自身努力

**旺盛之人气运（50-70）**
- 经常能发现宝物
- 修炼时容易进入顿悟状态
- 突破境界时概率提升
- 遇到危险时总能化险为夷

**昌隆之人气运（70-90）**
- 天道眷顾，机缘不断
- 修炼速度远超常人
- 突破境界如同喝水般简单
- 遇到危险时总能获得意外帮助

**应劫之人气运（90-100）**
- 天命所归，气运之子
- 修炼速度是常人的数倍
- 突破境界几乎没有瓶颈
- 即使身处绝境也能逢凶化吉

## 气运规则

### 承载上限

- 气运承载上限为100点
- 超过100点的气运会自动溢出，化为"气运之光"护持修士
- 气运之光可抵挡一次致命攻击，消耗后气运值降为90

### 气运消耗

| 行为类型 | 气运消耗 | 说明 |
|----------|----------|------|
| 寻找机缘 | 5-10 | 主动寻找机缘会消耗气运 |
| 化解危机 | 10-20 | 遭遇危险时，气运会自动消耗以化解 |
| 突破境界 | 15-20 | 突破境界时消耗气运以增加成功率 |
| 逆天改命 | 20+ | 强行改变命运轨迹，消耗大量气运 |

### 气运恢复

| 恢复方式 | 恢复速度 | 说明 |
|----------|----------|------|
| 自然恢复 | 1点/年 | 气运会随时间缓慢恢复 |
| 功德转化 | 500功德/点 | 消耗功德可快速恢复气运 |
| 天材地宝 | 视宝物而定 | 某些天材地宝可恢复气运 |
| 机缘巧合 | 不定 | 遇到特殊机缘可恢复气运 |

### 气运转移

- **主动转移**：修士可主动将气运转移给他人，但会消耗自身气运
- **被动转移**：被夺舍时，气运会被夺舍者继承
- **气运掠夺**：某些禁术可掠夺他人气运，但会增加业力
- **气运嫁接**：某些大能可将气运嫁接给他人，但需付出代价

### 先天气运与后天气运

| 类型 | 来源 | 特点 | 可变性 |
|------|------|------|--------|
| 先天气运 | 出生时天道赐予 | 决定修士初始命运走向 | 极难改变，需逆天改命 |
| 后天气运 | 功德业力转化 | 随修士行为动态变化 | 可通过行善积德提升 |

## 三者关系图

```
功德 ←→ 业力
   ↓         ↓
   → 气运 ←
      ↓
   天道眷顾/惩罚
```

### 核心关系

**功德与业力：此消彼长**
- 功德与业力是天道的一体两面
- 行善积德增加功德，同时减少业力
- 作恶多端增加业力，同时减少功德
- 功德与业力不能同时为正，但可以同时为零

**功德与气运：正相关**
- 功德越高，气运越旺
- 功德可直接转化为气运
- 高气运可带来更多机缘，从而获得更多功德

**业力与气运：负相关**
- 业力越重，气运越低
- 业力会持续消耗气运
- 低气运会带来更多灾祸，从而增加业力

**气运与天道：外在体现**
- 气运是天道对修士态度的外在体现
- 高气运代表天道眷顾，低气运代表天道惩罚
- 气运影响修士的一切，包括修炼、机缘、生死

### 量化关系

- 1点功德 → +0.002气运
- 1点业力 → -0.002气运
- 气运 = 基础气运 + (功德 - 业力) × 0.002
- 基础气运由先天气运决定，范围0-50

## 写作参考：量化体系在剧情中的应用

### 功德业力在剧情中的应用

**主角初期**
- 功德：0-1000
- 业力：0-500
- 气运：30-50（平平）

主角初入修仙界，功德业力都不高，气运平平。此时剧情应以"积累"为主，让主角通过小善行积累功德，同时避免作恶增加业力。

**主角中期**
- 功德：1000-10000
- 业力：0-2000
- 气运：50-70（旺盛）

主角已有所成就，功德积累到一定程度，气运开始旺盛。此时剧情应以"考验"为主，让主角面临善恶抉择，考验其道心。

**主角后期**
- 功德：10000+
- 业力：0-5000
- 气运：70-90（昌隆）

主角已成为大能，功德深厚，气运昌隆。此时剧情应以"责任"为主，让主角承担更大的责任，守护更多的人。

**主角巅峰**
- 功德：50000+
- 业力：0-10000
- 气运：90-100（应劫之人）

主角已站在修仙界巅峰，功德无量，气运之子。此时剧情应以"抉择"为主，让主角面临终极抉择，决定自己的道。

### 业力反噬在剧情中的应用

**制造冲突**
- 主角因救人而误杀无辜，业力增加
- 业力反噬导致修炼瓶颈，剧情陷入低谷
- 主角需找到化解业力的方法，推动剧情发展

**推动成长**
- 主角因业力反噬而走火入魔，道心崩溃
- 在绝望中重新找回道心，突破心魔
- 业力反噬成为主角成长的催化剂

**设置悬念**
- 主角发现自己业力突然增加，却不知原因
- 调查发现是有人暗中陷害，嫁祸业力
- 引出幕后黑手，推动主线剧情

### 气运在剧情中的应用

**机缘设计**
- 高气运时：主角随手捡到宝物，误入秘境获得传承
- 低气运时：主角寻宝空手而归，炼丹必定炸炉
- 气运变化可制造剧情起伏

**危机设计**
- 主角气运突然下降，预示即将遭遇危机
- 主角需找到气运下降的原因，化解危机
- 危机过后气运恢复，甚至提升

**对手设计**
- 反派气运极高，天道眷顾，主角屡战屡败
- 主角需找到克制反派气运的方法
- 最终击败反派，掠夺其气运

### 量化数据在剧情中的具体应用

**修炼突破**
- 主角功德10000，业力500，气运75
- 突破境界时，功德减轻天劫10档，气运增加突破概率
- 渡劫成功率 = 基础成功率 + 功德加成 + 气运加成

**寻宝冒险**
- 主角气运70，进入秘境寻宝
- 气运70对应"昌隆"等级，机缘不断
- 主角在秘境中连续获得宝物，推进剧情

**善恶抉择**
- 主角面临抉择：救一人还是救十人
- 救一人：功德+300，业力+0
- 救十人：功德+3000，但需牺牲一人，业力+500
- 主角的选择会影响功德业力平衡，进而影响后续剧情
