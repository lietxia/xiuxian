## 一、阴阳概述

### 1.1 什么是阴阳

阴阳，是天地间最基本的两种力量。阴代表黑暗、寒冷、柔弱、下降；阳代表光明、温暖、刚强、上升。阴阳相互对立、相互依存、相互转化，是万物生成变化的根本。

### 1.2 阴阳的特点

| 特点 | 说明 |
|:---:|:---|
| 对立统一 | 阴阳相互对立，又相互依存 |
| 相互转化 | 阴阳可相互转化，物极必反 |
| 动态平衡 | 阴阳需保持动态平衡 |
| 万物之本 | 阴阳是万物生成变化的根本 |

### 1.3 阴阳在修仙中的应用

| 应用领域 | 说明 |
|:---:|:---|
| 修炼 | 修炼需调和阴阳，保持平衡 |
| 法术 | 法术分阴阳，相生相克 |
| 炼丹 | 炼丹需调和阴阳，方可成丹 |
| 炼器 | 炼器需调和阴阳，方可成器 |
| 阵法 | 阵法需调和阴阳，方可运转 |

## 二、五行概述

### 2.1 什么是五行

五行，即金、木、水、火、土，是天地间最基本的五种元素。五行相生相克，循环往复，是万物生成变化的基础。

### 2.2 五行相生

| 相生关系 | 说明 | 代表 |
|:---:|:---|:---|
| 木生火 | 木燃烧生火 | 木火相生 |
| 火生土 | 火燃烧生土 | 火土相生 |
| 土生金 | 土中生金 | 土金相生 |
| 金生水 | 金熔化生水 | 金水相生 |
| 水生木 | 水滋养生木 | 水木相生 |

### 2.3 五行相克

| 相克关系 | 说明 | 代表 |
|:---:|:---|:---|
| 木克土 | 木破土而出 | 木克土 |
| 土克水 | 土挡水势 | 土克水 |
| 水克火 | 水灭火势 | 水克火 |
| 火克金 | 火熔金器 | 火克金 |
| 金克木 | 金伐木器 | 金克木 |

### 2.4 五行属性

| 五行 | 属性 | 代表 | 特点 |
|:---:|:---:|:---|:---|
| 金 | 金属 | 刚强、锐利 | 攻击力强 |
| 木 | 木属 | 生长、柔和 | 恢复力强 |
| 水 | 水属 | 流动、寒冷 | 防御力强 |
| 火 | 火属 | 炎热、猛烈 | 破坏力强 |
| 土 | 土属 | 厚重、稳固 | 稳定性强 |
