## 一、防御符箓概述

### 1.1 什么是防御符箓

防御符箓是以保护己方、抵挡攻击为主要目的的符箓。它将各种防御性法术封印于符纸之上，使用时瞬间激发，形成护盾、结界或屏障，抵挡来自外界的攻击和伤害。防御符箓的特点是瞬发、持久、覆盖面广，是修士外出历练时保命的重要手段。

### 1.2 防御符箓的分类

防御符箓按功能可分为护体符、法防符、净化符、五行防御符四大类。护体符抵挡物理攻击，法防符抵挡法术攻击，净化符解除负面状态，五行防御符则针对特定属性进行防御。每类都有从一阶到九阶的完整体系。

## 七、防御符箓使用技巧

### 7.1 实战应用

| 场景 | 推荐符箓 | 说明 |
|:---:|:---|:---|
| 遭遇强敌 | 金钟护身符、万法护盾符 | 多重护盾叠加，提升防御 |
| 毒雾环境 | 祛毒符、净化符 | 提前使用，预防中毒 |
| 雷雨天劫 | 避雷符、天雷护符 | 专门抵御雷系攻击 |
| 火域历练 | 避火符、万火不侵符 | 抵抗高温火焰 |
| 灵魂攻击 | 守魂符、万邪不侵符 | 保护灵魂安全 |

### 7.2 符箓组合技

| 组合 | 效果 | 说明 |
|:---:|:---|:---|
| 护体符+法防符 | 双重防御 | 物理法术双重防护 |
| 净化符+护体符 | 净化护体 | 先净化负面状态再防御 |
| 避火符+避雷符 | 五行双防 | 同时防御两种属性 |
| 生死符+金身护体符 | 绝对保命 | 生死关头双重保险 |

### 7.3 使用注意事项

1. **提前准备**：防御符箓应在战斗前提前激发，不要等到攻击临身才使用
2. **灵气储备**：防御符箓需要持续消耗灵气维持，应确保灵气充足
3. **叠加使用**：多张防御符箓可叠加使用，但会增加灵气消耗
4. **时效性**：防御符箓有持续时间，需注意时效，及时补充
5. **属性克制**：五行防御符需根据敌人属性选择，用错则效果大减
