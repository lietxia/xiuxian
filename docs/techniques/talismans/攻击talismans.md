## 一、攻击符箓概述

### 1.1 什么是攻击符箓

攻击符箓是符箓体系中最为常见、使用最为广泛的一类。它将各种攻击性法术封印于符纸之上，使用时只需注入灵气激发，便可瞬间释放出强大的攻击威力。攻击符箓的特点是瞬发、便携、威力可控，是修士外出历练时的必备之物。

### 1.2 攻击符箓的分类

攻击符箓按属性可分为火系、雷系、剑系、冰系、爆系五大类，每类都有从一阶到九阶的完整体系。火系灼烧范围广，雷系伤害高且附带麻痹，剑系穿透力强，冰系可冻结减速，爆系则以范围爆炸见长。

## 八、攻击符箓使用技巧

### 8.1 实战应用

| 场景 | 推荐符箓 | 说明 |
|:---:|:---|:---|
| 单挑强敌 | 雷系符、剑系符 | 高伤害，可破防 |
| 群战围攻 | 火系符、爆系符 | 范围广大，可同时伤敌多人 |
| 追击逃敌 | 风系符、冰系符 | 减速冻结，限制行动 |
| 偷袭暗杀 | 剑系符、毒系符 | 穿透力强，无声无息 |

### 8.2 符箓组合技

| 组合 | 效果 | 说明 |
|:---:|:---|:---|
| 冰符+雷符 | 冰雷交击 | 先冻结再雷击，伤害翻倍 |
| 火符+风符 | 风助火势 | 火焰范围扩大，威力增强 |
| 毒符+火符 | 毒火焚身 | 毒素被火焰激发，伤害倍增 |
| 剑符+雷符 | 雷霆剑气 | 剑气附带雷电，穿透麻痹 |

### 8.3 使用注意事项

1. **距离把控**：攻击符箓有有效距离，过远则威力大减
2. **灵气注入**：注入灵气量影响符箓威力，但不可超过符纸承受上限
3. **连续使用**：连续使用同类型符箓会产生抗性，应交替使用
4. **保存条件**：攻击符箓需在灵气环境中保存，否则会逐渐失效
5. **误伤防范**：范围攻击符箓需注意友军位置，避免误伤
