# 属性克制

> 属性克制是修仙战斗中的重要因素，包括五行相克、阴阳相克、特殊属性克制等。

---

## 属性概述

### 什么是属性？

属性是修仙世界中万物的基本特性，包括五行属性（金木水火土）、阴阳属性、特殊属性等。不同的属性之间存在克制关系。

### 属性的分类

| 分类 | 说明 | 代表 |
|------|------|------|
| 五行属性 | 金木水火土 | 基础属性 |
| 阴阳属性 | 阴阳 | 对立属性 |
| 特殊属性 | 雷风冰暗光 | 特殊属性 |
| 混合属性 | 多种属性混合 | 复合属性 |

---

## 五行相克

### 五行属性

| 属性 | 特点 | 代表 |
|------|------|------|
| 金 | 锐利、坚硬 | 金刃、金剑 |
| 木 | 生机、生长 | 木藤、木刺 |
| 水 | 柔和、变化 | 水箭、水幕 |
| 火 | 炽热、暴烈 | 火球、火海 |
| 土 | 厚重、稳定 | 土墙、土刺 |

### 五行相克关系

```
金 → 木 → 土 → 水 → 火 → 金
```

| 克制关系 | 说明 | 效果 |
|----------|------|------|
| 金克木 | 金刃切割木 | 伤害加成 |
| 木克土 | 木根破土 | 伤害加成 |
| 土克水 | 土堤挡水 | 伤害加成 |
| 水克火 | 水灭火焰 | 伤害加成 |
| 火克金 | 火熔金属 | 伤害加成 |

### 五行相生关系

```
木 → 火 → 土 → 金 → 水 → 木
```

| 相生关系 | 说明 | 效果 |
|----------|------|------|
| 木生火 | 木燃烧生火 | 增强效果 |
| 火生土 | 火燃烧成土 | 增强效果 |
| 土生金 | 土中生金属 | 增强效果 |
| 金生水 | 金熔化成水 | 增强效果 |
| 水生木 | 水滋养木 | 增强效果 |

| 属性 | 特点 | 代表 |

| 阴 | 柔和、阴冷 | 阴气、阴雷 |

| 克制关系 | 说明 | 效果 |

| 阳克阴 | 阳气克制阴气 | 伤害加成 |
| 阴克阳 | 阴气克制阳气 | 伤害加成 |

### 阴阳平衡

阴阳平衡是最佳状态：

| 状态 | 说明 | 效果 |
|------|------|------|
| 纯阳 | 只有阳属性 | 攻击力强，防御弱 |
| 纯阴 | 只有阴属性 | 防御力强，攻击弱 |
| 阴阳平衡 | 阴阳平衡 | 攻防平衡 |

## 属性克制的突破

### 如何突破属性克制

| 方法 | 说明 | 效果 |
|------|------|------|
| 提升境界 | 提升境界可以减弱克制 | 减少克制效果 |
| 特殊体质 | 特殊体质可以免疫克制 | 免疫克制 |
| 特殊功法 | 特殊功法可以减弱克制 | 减少克制效果 |
| 特殊法宝 | 特殊法宝可以减弱克制 | 减少克制效果 |
| 混合属性 | 混合属性可以减弱克制 | 减少克制效果 |

### 克制系数

| 克制程度 | 系数 | 说明 |
|----------|------|------|
| 完全克制 | 2.0 | 伤害翻倍 |
| 强烈克制 | 1.5 | 伤害增加50% |
| 普通克制 | 1.2 | 伤害增加20% |
| 轻微克制 | 1.1 | 伤害增加10% |
| 无克制 | 1.0 | 伤害正常 |
| 被克制 | 0.8 | 伤害减少20% |

## 属性与灵根

### 灵根属性

| 灵根 | 属性 | 特点 |
|------|------|------|
| 金灵根 | 金属性 | 攻击力强 |
| 木灵根 | 木属性 | 恢复力强 |
| 水灵根 | 水属性 | 变化多端 |
| 火灵根 | 火属性 | 爆发力强 |
| 土灵根 | 土属性 | 防御力强 |
| 雷灵根 | 雷属性 | 速度极快 |
| 风灵根 | 风属性 | 灵活多变 |
| 冰灵根 | 冰属性 | 控制力强 |
| 暗灵根 | 暗属性 | 隐蔽性强 |
| 光灵根 | 光属性 | 净化力强 |

### 灵根与克制

灵根决定了修仙者的属性，也决定了修仙者在属性克制中的位置：

1. **单灵根**：属性单一，克制关系明确
2. **双灵根**：属性两种，可以切换
3. **多灵根**：属性多种，灵活应变
4. **变异灵根**：特殊属性，克制特殊
