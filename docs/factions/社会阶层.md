# 社会阶层

> 社会阶层是修仙世界中社会等级的划分，包括阶层划分、阶层流动、阶层冲突等。

---

## 社会阶层概述

### 什么是社会阶层？

社会阶层是修仙世界中社会等级的划分，反映了不同群体在社会中的地位和权力。

### 社会阶层的特点

| 特点 | 说明 |
|------|------|
| 等级性 | 社会等级分明 |
| 流动性 | 阶层可以流动 |
| 差异性 | 不同阶层差异大 |
| 稳定性 | 阶层相对稳定 |
| 冲突性 | 阶层之间有冲突 |
