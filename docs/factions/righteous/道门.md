# 道门

> 道门是修仙界正道势力的核心，以道法自然、顺应天道为修行准则。本文详细介绍道门三宗的组织结构与特点。

---

## 道门概述

道门是修仙界最正统的修炼体系，以"道法自然"为核心理念，追求与天地和谐、顺应天道。道门三宗是正道势力中最强大的存在。

### 道门核心理念

| 理念 | 说明 |
|------|------|
| 道法自然 | 顺应天地自然法则修炼 |
| 无为而治 | 不强求，顺其自然 |
| 清静无为 | 心境清净，不被外物干扰 |
| 天人合一 | 与天地合一，感悟大道 |

## 道门三宗

### 太上道宗

| 项目 | 内容 |
|------|------|
| 地位 | 道门之首，正道第一宗门 |
| 宗主 | 太上道人（合体期） |
| 弟子数量 | 约三万人 |
| 势力范围 | 修真界中部，方圆万里 |
| 主修功法 | 太上无极功（八品） |
| 特色 | 最全面的道门功法，讲究道法自然 |
| 镇宗法宝 | 太上无极镜（仙器级） |

### 玄天道宗

| 项目 | 内容 |
|------|------|
| 地位 | 道门第二，擅长阵法 |
| 宗主 | 玄天道人（合体期） |
| 弟子数量 | 约两万人 |
| 势力范围 | 修真界东部 |
| 主修功法 | 玄天大阵诀（七品） |
| 特色 | 阵法之道登峰造极 |
| 镇宗法宝 | 玄天大阵盘（仙器级） |

### 青虚道宗

| 项目 | 内容 |
|------|------|
| 地位 | 道门第三，擅长炼丹 |
| 宗主 | 青虚道人（炼虚期巅峰） |
| 弟子数量 | 约一万五千人 |
| 势力范围 | 修真界南部 |
| 主修功法 | 青虚丹元功（七品） |
| 特色 | 炼丹术天下第一 |
| 镇宗法宝 | 青虚丹炉（仙器级） |

| 功法等级 | 品级 | 修炼条件 | 威力 |
