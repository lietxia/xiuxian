# 幽冥界

> 幽冥界是鬼修势力的核心组织，掌控冥界与轮回。本文详细介绍幽冥界的组织结构与特点。

---

## 幽冥界概述

幽冥界是鬼修势力的统称，掌控冥界、轮回和亡灵事务。幽冥界相对独立，不常干涉修仙界事务。

### 幽冥界特点

| 特点 | 说明 |
|------|------|
| 掌控轮回 | 掌管六道轮回 |
| 独立运作 | 相对独立于各界 |
| 阴气极重 | 冥界阴气浓郁 |
| 灵魂法则 | 以灵魂法则为核心 |
