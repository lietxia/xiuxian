# 宗门内部结构

> 宗门乃修仙界立身之本。一个运转良好的宗门，其内部结构如同精密法阵，环环相扣，缺一不可。本文详细阐述宗门的权力层级、部门架构、弟子体系与经济命脉。

---

## 一、权力层级体系

宗门权力自上而下，层级分明。修为越高，话语权越重，此乃修仙界亘古不变之法则。

### 1.1 太上长老（老祖）

太上长老是宗门的定海神针，通常为化神期以上的老怪物，寿元将尽或闭关苦修，不问世事。

| 职责 | 说明 |
|------|------|
| 镇守宗门 | 以自身修为震慑宵小，令敌不敢轻犯 |
| 最终裁决 | 宗门生死存亡之际，出关主持大局 |
| 传授大道 | 偶尔开坛讲道，指点核心弟子迷津 |
| 守护底蕴 | 守护宗门镇派之宝、上古传承 |

**典型设定：** 太上长老常年闭关，宗门弟子只闻其名不见其人。唯有掌门持信物求见，方能叩开洞府大门。一位太上长老的存在，往往比千军万马更具威慑力。

### 1.2 掌门（宗主）

掌门是宗门的最高实际掌权者，统管宗门内外一切事务。

| 职责 | 说明 |
|------|------|
| 统领全局 | 主持宗门日常运转，决策重大事务 |
| 对外代表 | 代表宗门参与仙盟会议、与其他势力交涉 |
| 传承衣钵 | 选定下一任掌门人选 |
| 调解纷争 | 裁决宗门内部矛盾与利益分配 |

**掌门遴选：** 通常由前任掌门指定，或由长老会推举。需满足两个硬性条件——修为不低于元婴期，且在宗门威望足以服众。有些宗门设有"掌门试炼"，候选者需通过秘境考验方可继位。

### 1.3 副宗主与大长老

副宗主协助掌门处理宗门事务，大长老则是长老会的核心人物。

| 职位 | 人数 | 职责 |
|------|------|------|
| 副宗主 | 1-2人 | 代行掌门职权，分管具体事务 |
| 大长老 | 3-5人 | 长老会核心，拥有重大事务投票权 |

**权力制衡：** 副宗主与大长老之间往往存在微妙的平衡。掌门在位时二者各司其职，掌门闭关或外出时，副宗主代行职权，但重大决策仍需长老会过半数同意。

### 1.4 长老会

长老会是宗门的最高决策机构，由金丹期以上的长老组成。

| 类别 | 职责 | 人数 |
|------|------|------|
| 内务长老 | 管理宗门内部事务、资源分配 | 3-5人 |
| 外务长老 | 处理对外关系、情报收集 | 2-3人 |
| 执法长老 | 维护门规、审判违规弟子 | 2-3人 |
| 传功长老 | 传授功法、指导弟子修炼 | 3-5人 |
| 护道长老 | 率领护道军、镇守宗门要地 | 2-3人 |

**议事规则：** 一般事务由分管长老自行决断；重大事务需长老会过半数通过；涉及宗门根基的生死大事，需太上长老首肯。

### 1.5 峰主、堂主、殿主

此层级是宗门的中坚管理层，各司一方，直接对长老会负责。

| 职位 | 职责 | 典型设置 |
|------|------|----------|
| 峰主 | 统管各分峰事务 | 七脉各设一峰主 |
| 堂主 | 统管各堂口事务 | 执法堂、炼丹堂等 |
| 殿主 | 统管各殿堂事务 | 传功殿、藏宝阁等 |

### 1.6 护法、执事、供奉

此层级负责具体执行，是宗门运转的润滑剂。

| 职位 | 职责 | 身份 |
|------|------|------|
| 护法 | 护卫宗门重要人物或设施 | 通常为筑基后期以上 |
| 执事 | 处理日常事务、分派任务 | 通常为筑基中期以上 |
| 供奉 | 宗门聘请的外来高手 | 不限修为，以实力论 |

**供奉制度：** 供奉是宗门的特殊存在。他们不受宗门门规约束，不参与宗门内政，但享受宗门资源供奉，战时需为宗门出力。散修高手、隐世老怪皆可成为供奉。

## 二、弟子等级体系

弟子是宗门的根基，其等级划分直接关系到资源分配与培养重点。

### 2.1 弟子等级划分

| 等级 | 典型修为 | 人数占比 | 地位说明 |
|------|----------|----------|----------|
| 圣子/圣女 | 筑基后期-金丹 | 极少（1-2人） | 宗门未来掌门候选人，万众瞩目 |
| 真传弟子 | 筑基中期以上 | 1%-3% | 峰主或大长老亲传，重点培养 |
| 亲传弟子 | 筑基初期以上 | 3%-5% | 长老亲传弟子，资源优厚 |
| 内门弟子 | 练气后期以上 | 15%-20% | 正式弟子，可学习中阶功法 |
| 外门弟子 | 练气中期以上 | 30%-40% | 普通弟子，仅有基础资源 |
| 杂役弟子 | 练气初期或无修为 | 40%-50% | 从事杂务，勉强温饱 |

### 2.2 弟子待遇差异

| 项目 | 圣子/真传 | 亲传 | 内门 | 外门 | 杂役 |
|------|-----------|------|------|------|------|
| 月例灵石 | 中品灵石50枚 | 下品500枚 | 下品100枚 | 下品20枚 | 下品5枚 |
| 丹药供给 | 筑基丹、金丹期丹药 | 练气丹药优先 | 基础丹药 | 无 | 无 |
| 功法权限 | 可阅藏经阁顶层 | 可阅第三层 | 可阅第二层 | 仅第一层 | 无 |
| 洞府等级 | 灵脉核心独院 | 灵脉次核心区 | 内门区域 | 外门区域 | 杂役区 |
| 指导频率 | 峰主亲自指点 | 长老每月一次 | 执事季度指导 | 无固定指导 | 无 |
| 秘境名额 | 优先入选 | 有名额 | 需竞争 | 极少 | 无 |

### 2.3 晋升与淘汰

| 晋升路径 | 条件 | 难度 |
|----------|------|------|
| 杂役→外门 | 突破练气中期，通过考核 | 较易 |
| 外门→内门 | 突破练气后期，通过大比 | 中等 |
| 内门→亲传 | 被长老看中收为弟子 | 较难 |
| 亲传→真传 | 资质出众，峰主亲传 | 困难 |
| 真传→圣子 | 宗门大比夺魁或立下大功 | 极难 |

**淘汰机制：** 外门弟子若三十年内未能突破练气后期，将被降为杂役或劝退。内门弟子若百年内未能筑基，将失去内门资格。此乃宗门保持活力之残酷法则。

## 三、关键部门架构

### 3.1 传功殿

传功殿是宗门传承功法的核心所在。

| 职能 | 说明 |
|------|------|
| 功法传授 | 定期开坛讲道，传授基础功法 |
| 藏经管理 | 管理藏经阁，维护功法典籍 |
| 考核评定 | 考核弟子修为进度，评定等级 |
| 创法研修 | 研究改良功法，推陈出新 |

**藏经阁层级：** 通常分为四层——第一层存放基础功法，外门弟子可入；第二层存放中阶功法，内门弟子可入；第三层存放高阶功法，亲传以上弟子可入；顶层存放镇派绝学，唯有掌门与太上长老可入。

### 3.2 执法堂

执法堂是宗门的"铁面判官"，维护门规纲纪。

| 职能 | 说明 |
|------|------|
| 巡查执法 | 巡查宗门各处，制止违规行为 |
| 审判断罪 | 审判违规弟子，量刑定罪 |
| 缉拿追捕 | 追捕叛逃弟子、通缉犯 |
| 牢狱管理 | 管理宗门地牢、禁闭室 |

**执法堂权柄：** 执法堂拥有先斩后奏之权，执法长老可直接拿下违规弟子，无需请示掌门。但若涉及长老以上人物，需报请掌门或长老会裁决。

### 3.3 炼丹堂

炼丹堂负责炼制宗门所需的各种丹药。

| 职能 | 说明 |
|------|------|
| 丹药炼制 | 炼制弟子修炼所需丹药 |
| 灵药种植 | 管理灵药园，种植炼丹材料 |
| 丹方研究 | 研究新丹方，改良炼丹技艺 |
| 对外销售 | 将多余丹药对外出售，赚取灵石 |

**炼丹师等级：** 学徒→初级炼丹师→中级炼丹师→高级炼丹师→大师级炼丹师→宗师级炼丹师。炼丹师在宗门地位极高，即便修为不高，也备受尊重。

### 3.4 炼器堂

炼器堂负责炼制法器、法宝。

| 职能 | 说明 |
|------|------|
| 法器炼制 | 炼制弟子使用的法器 |
| 法宝修复 | 修复损坏的法宝 |
| 材料提炼 | 提炼矿石、妖兽材料 |
| 阵法刻录 | 在法器上刻录阵法 |

### 3.5 灵兽堂

灵兽堂负责驯养、管理宗门的灵兽。

| 职能 | 说明 |
|------|------|
| 灵兽驯养 | 驯养灵兽，供弟子使用 |
| 灵兽配种 | 培育优良灵兽品种 |
| 妖兽研究 | 研究妖兽习性、弱点 |
| 骑乘运输 | 提供飞行灵兽，用于运输 |

### 3.6 藏书阁

藏书阁收藏宗门的典籍、功法、秘闻。

| 层级 | 内容 | 权限 |
|------|------|------|
| 第一层 | 基础功法、修仙常识 | 所有弟子 |
| 第二层 | 中阶功法、秘闻杂记 | 内门以上 |
| 第三层 | 高阶功法、宗门秘史 | 亲传以上 |
| 顶层 | 镇派绝学、上古秘典 | 掌门、太上长老 |

### 3.7 藏宝阁

藏宝阁收藏宗门的珍稀宝物、灵材、法器。

| 职能 | 说明 |
|------|------|
| 宝物收藏 | 收藏宗门积累的珍稀宝物 |
| 资源分配 | 根据弟子贡献分配资源 |
| 兑换交易 | 弟子以贡献点兑换宝物 |
| 鉴定估价 | 鉴定弟子带回的不明宝物 |

### 3.8 悬赏台

悬赏台是宗门发布任务、弟子接取任务的场所。

| 任务类型 | 说明 | 奖励 |
|----------|------|------|
| 采集任务 | 采集灵药、矿石 | 灵石+贡献点 |
| 清剿任务 | 清剿妖兽、邪修 | 灵石+战利品 |
| 护送任务 | 护送物资、人员 | 灵石+丹药 |
| 探索任务 | 探索秘境、遗迹 | 灵石+机缘 |
| 特殊任务 | 宗门紧急事务 | 丰厚奖励 |

### 3.9 综务殿

综务殿是宗门的大管家，负责后勤保障。

| 职能 | 说明 |
|------|------|
| 物资管理 | 管理宗门日常物资 |
| 财务收支 | 管理宗门灵石收支 |
| 建设维护 | 维护宗门建筑、阵法 |
| 弟子档案 | 管理弟子信息、功勋记录 |

### 3.10 护道军

护道军是宗门的武装力量，负责保卫宗门安全。

| 编制 | 人数 | 职责 |
|------|------|------|
| 护道统领 | 1人 | 统领全军，通常为金丹期以上 |
| 护道队长 | 5-10人 | 各领一队，筑基期以上 |
| 护道军士 | 数百人 | 执行巡逻、护卫、作战任务 |

## 四、七脉分峰制度

七脉分峰是大型宗门常见的组织形式，参考青云门模式。

### 4.1 制度概述

宗门主峰之外，另设六座分峰，各峰由一位峰主统管，形成"一主六副"或"七脉并立"的格局。

### 4.2 各峰职能

| 峰名 | 职能 | 特色 |
|------|------|------|
| 主峰（掌门峰） | 宗门中枢，掌门驻地 | 决策中心，藏经阁所在 |
| 剑峰 | 剑修培养 | 以剑道传承为主，战力最强 |
| 丹峰 | 炼丹传承 | 灵药园所在，炼丹堂驻地 |
| 器峰 | 炼器传承 | 矿脉所在，炼器堂驻地 |
| 兽峰 | 灵兽驯养 | 灵兽园所在，灵兽堂驻地 |
| 符峰 | 符箓传承 | 符纸制作，阵法研究 |
| 律峰 | 执法监察 | 执法堂所在，维护门规 |

### 4.3 各峰关系

各峰之间既有合作又有竞争。每十年举行一次"七脉大比"，各峰弟子同台竞技，胜者可获得额外资源分配。峰主之间虽有竞争，但在宗门大事上必须一致对外。

### 4.4 弟子归属

新入门弟子通常先在主峰修习基础功法，一年后根据资质与兴趣选择分峰。各峰也会主动招收弟子，形成良性竞争。弟子选定分峰后，即为该峰弟子，享受该峰资源，也受该峰峰主管辖。

## 五、宗门经济模型

宗门的运转离不开灵石，其经济来源与支出构成一个完整的体系。

### 5.1 收入来源

| 来源 | 说明 | 占比 |
|------|------|------|
| 灵石矿脉 | 宗门领地内的灵石矿脉开采 | 30%-40% |
| 坊市税收 | 宗门庇护的坊市收取税收 | 15%-20% |
| 丹药法器销售 | 炼丹堂、炼器堂对外销售 | 15%-20% |
| 灵田灵植 | 种植灵药、灵谷出售 | 5%-10% |
| 拍卖会 | 宗门定期举办拍卖会抽成 | 5%-10% |
| 附属势力贡品 | 附属家族、小宗门缴纳贡品 | 10%-15% |

### 5.2 支出项目

| 项目 | 说明 | 占比 |
|------|------|------|
| 弟子月例 | 每月发放给弟子的灵石、丹药 | 30%-40% |
| 阵法维护 | 维护宗门护山大阵、聚灵阵 | 10%-15% |
| 建设修缮 | 宗门建筑、洞府的建设与修缮 | 5%-10% |
| 战争消耗 | 战时物资、抚恤、奖赏 | 不定 |
| 采购物资 | 购买宗门无法自产的物资 | 10%-15% |
| 储备金 | 应对突发情况的储备 | 10%-20% |

### 5.3 贡献点体系

贡献点是宗门内部的"货币"，弟子以贡献点兑换资源。

| 获取方式 | 贡献点 | 说明 |
|----------|--------|------|
| 完成悬赏任务 | 10-1000点 | 根据任务难度 |
| 缴纳灵材宝物 | 按价值折算 | 由藏宝阁估价 |
| 传授功法心得 | 50-500点 | 需经传功殿认可 |
| 立下战功 | 100-10000点 | 根据战功大小 |
| 发现矿脉秘境 | 5000-50000点 | 重大发现重赏 |

## 六、写作参考：宗门剧情设计

### 6.1 经典宗门剧情模式

| 剧情模式 | 说明 | 代表作品 |
|----------|------|----------|
| 废柴逆袭 | 主角从杂役弟子崛起 | 《凡人修仙传》 |
| 宗门大比 | 各峰弟子同台竞技 | 《诛仙》 |
| 宗门危机 | 外敌入侵，宗门生死存亡 | 多部作品 |
| 权力斗争 | 掌门之争、峰主之位 | 《玄鉴仙族》 |
| 秘境探索 | 宗门组织弟子探索秘境 | 常见桥段 |

### 6.2 宗门剧情设计要点

**层级冲突：** 利用弟子等级差异制造冲突。外门弟子被内门弟子欺压，杂役弟子被外门弟子看不起，这种层级矛盾是天然的剧情驱动力。

**资源争夺：** 灵石、丹药、功法、秘境名额——资源永远是稀缺的。围绕资源分配展开的争夺，能够自然地推动剧情发展。

**师徒传承：** 师父传功、师兄弟之间的恩怨情仇，是宗门剧情的重要组成部分。一个好师父可以改变弟子的命运，一个坏师父也可能毁掉弟子的前程。

**宗门兴衰：** 宗门从鼎盛走向衰落，或从衰落走向复兴，是宏大的叙事框架。主角可以是宗门复兴的关键人物，也可以是宗门覆灭的见证者。

### 6.3 宗门设定注意事项

| 要点 | 说明 |
|------|------|
| 权力制衡 | 避免掌门一人独大，设置合理的制衡机制 |
| 利益纠葛 | 宗门内部必然存在利益纷争，避免过于理想化 |
| 历史传承 | 宗门应有历史感，有传承、有底蕴、有故事 |
| 外部压力 | 宗门面临的外部威胁，是推动剧情的重要因素 |
| 人物群像 | 塑造不同层级、不同立场的人物，丰富宗门生态 |
