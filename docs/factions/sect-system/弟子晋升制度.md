# 弟子晋升制度

> 本文档详细说明宗门弟子的晋升条件、考核方式、晋升流程等。

---

## 一、弟子等级与晋升路径

### 1.1 晋升路径图

```
杂役弟子 → 记名弟子 → 外门弟子 → 内门弟子 → 核心弟子 → 亲传弟子 → 真传弟子
    │           │           │           │           │           │           │
    │           │           │           │           │           │           │
    ▼           ▼           ▼           ▼           ▼           ▼           ▼
  宗门杂务    考察期      基础修炼    精英修炼    重点培养    长老亲传    宗门未来
```

---

## 二、各等级晋升条件

### 2.1 杂役弟子 → 记名弟子

| 条件 | 要求 | 说明 |
|------|------|------|
| 服务年限 | 满1年 | 在宗门服务满1年 |
| 修为要求 | 练气期一层 | 初步感应灵气 |
| 表现评价 | 良好以上 | 工作态度端正 |
| 考核方式 | 基础测试 | 测试灵根资质 |
| 推荐人 | 杂役管事 | 需要管事推荐 |

**晋升流程：**
1. 杂役管事提交推荐
2. 参加基础测试
3. 测试通过后登记为记名弟子
4. 领取记名弟子令牌

**晋升奖励：**
- 记名弟子令牌
- 基础功法一部
- 基础灵石10枚

---

### 2.2 记名弟子 → 外门弟子

| 条件 | 要求 | 说明 |
|------|------|------|
| 服务年限 | 满2年 | 在宗门服务满2年 |
| 修为要求 | 练气期三层以上 | 基础修为稳固 |
| 功法掌握 | 掌握1门基础功法 | 功法熟练度达标 |
| 考核方式 | 外门考核 | 通过外门考核 |
| 推荐人 | 记名弟子管事 | 需要管事推荐 |

**外门考核内容：**
- 灵力测试：测试灵力储量和纯度
- 功法演示：演示所学功法
- 实战测试：与同阶修士战斗
- 心性测试：测试道心稳固程度

**晋升流程：**
1. 记名弟子管事提交推荐
2. 参加外门考核
3. 考核通过后登记为外门弟子
4. 领取外门弟子令牌

**晋升奖励：**
- 外门弟子令牌
- 外门弟子服饰
- 基础灵石100枚
- 基础丹药若干

---

### 2.3 外门弟子 → 内门弟子

| 条件 | 要求 | 说明 |
|------|------|------|
| 入门年限 | 满3年 | 成为外门弟子满3年 |
| 修为要求 | 练气期七层以上 | 修为达到要求 |
| 功法掌握 | 掌握3门以上功法 | 功法掌握达标 |
| 贡献值 | 1000点以上 | 为宗门做出贡献 |
| 考核方式 | 内门考核 | 通过内门考核 |

**内门考核内容：**
- 灵力测试：测试灵力储量和纯度
- 功法演示：演示所学功法
- 实战测试：与同阶修士战斗（需胜利）
- 心性测试：测试道心稳固程度
- 贡献审核：审核贡献值

**晋升流程：**
1. 外门弟子提交晋升申请
2. 审核资格条件
3. 参加内门考核
4. 考核通过后登记为内门弟子
5. 领取内门弟子令牌

**晋升奖励：**
- 内门弟子令牌
- 内门弟子服饰
- 中品灵石10枚
- 中级丹药若干
- 内门弟子洞府

---

### 2.4 内门弟子 → 核心弟子

| 条件 | 要求 | 说明 |
|------|------|------|
| 入门年限 | 满5年 | 成为内门弟子满5年 |
| 修为要求 | 筑基期以上 | 修为达到要求 |
| 功法掌握 | 掌握5门以上功法 | 功法掌握达标 |
| 贡献值 | 5000点以上 | 为宗门做出贡献 |
| 资质要求 | 双灵根以上 | 资质优秀 |
| 考核方式 | 核心弟子考核 | 通过核心弟子考核 |

**核心弟子考核内容：**
- 灵力测试：测试灵力储量和纯度
- 功法演示：演示所学功法
- 实战测试：与同阶修士战斗（需连胜三场）
- 心性测试：测试道心稳固程度
- 贡献审核：审核贡献值
- 资质审核：审核灵根资质

**晋升流程：**
1. 内门弟子提交晋升申请
2. 审核资格条件
3. 参加核心弟子考核
4. 考核通过后长老会审核
5. 审核通过后登记为核心弟子
6. 领取核心弟子令牌

**晋升奖励：**
- 核心弟子令牌
- 核心弟子服饰
- 上品灵石10枚
- 高级丹药若干
- 核心弟子洞府
- 可进入藏经阁第二层

---

### 2.5 核心弟子 → 亲传弟子

| 条件 | 要求 | 说明 |
|------|------|------|
| 入门年限 | 满10年 | 成为核心弟子满10年 |
| 修为要求 | 金丹期以上 | 修为达到要求 |
| 贡献值 | 10000点以上 | 为宗门做出贡献 |
| 资质要求 | 天灵根或特殊体质 | 资质优秀 |
| 推荐人 | 长老亲荐 | 需要长老亲自推荐 |
| 考核方式 | 长老会审核 | 长老会审核通过 |

**亲传弟子审核内容：**
- 修为审核：审核修为是否达标
- 贡献审核：审核贡献值
- 资质审核：审核灵根资质
- 心性审核：审核道心稳固程度
- 长老推荐：需要长老亲自推荐

**晋升流程：**
1. 核心弟子获得长老推荐
2. 提交晋升申请
3. 长老会审核
4. 审核通过后登记为亲传弟子
5. 领取亲传弟子令牌

**晋升奖励：**
- 亲传弟子令牌
- 亲传弟子服饰
- 极品灵石10枚
- 顶级丹药若干
- 亲传弟子洞府
- 可进入藏经阁第三层
- 获得长老亲自指导

---

### 2.6 亲传弟子 → 真传弟子

| 条件 | 要求 | 说明 |
|------|------|------|
| 入门年限 | 满20年 | 成为亲传弟子满20年 |
| 修为要求 | 元婴期以上 | 修为达到要求 |
| 贡献值 | 50000点以上 | 为宗门做出贡献 |
| 资质要求 | 天灵根或特殊体质 | 资质优秀 |
| 推荐人 | 宗主或太上长老 | 需要宗主或太上长老推荐 |
| 考核方式 | 太上长老会审核 | 太上长老会审核通过 |

**真传弟子审核内容：**
- 修为审核：审核修为是否达标
- 贡献审核：审核贡献值
- 资质审核：审核灵根资质
- 心性审核：审核道心稳固程度
- 功法审核：审核功法掌握程度
- 太上长老会审核：太上长老会审核通过

**晋升流程：**
1. 亲传弟子获得宗主或太上长老推荐
2. 提交晋升申请
3. 太上长老会审核
4. 审核通过后登记为真传弟子
5. 领取真传弟子令牌

**晋升奖励：**
- 真传弟子令牌
- 真传弟子服饰
- 仙石1枚
- 仙丹若干
- 真传弟子洞府
- 可进入藏经阁最高层
- 可使用宗门秘境
- 参与宗门决策

## 三、特殊晋升途径

### 3.1 破格晋升

| 条件 | 说明 |
|------|------|
| 天才弟子 | 天灵根或特殊体质 |
| 大功弟子 | 为宗门立下大功 |

**破格晋升流程：**
1. 宗主或太上长老提名
2. 长老会审核
3. 审核通过后破格晋升

---

| 条件 | 说明 |
|------|------|

| 其他宗门弟子 | 其他宗门叛逃弟子 |

**特招流程：**
1. 长老推荐
2. 审核资质和修为

## 四、晋升考核详解

### 4.1 灵力测试

| 测试项目 | 测试方法 | 达标标准 |
|----------|----------|----------|
| 灵力储量 | 灵力测试石 | 达到同阶平均水平 |
| 灵力纯度 | 灵力纯度仪 | 纯度达到70%以上 |
| 灵力控制 | 灵力控制测试 | 控制精度达到80%以上 |

### 4.2 功法演示

| 测试项目 | 测试方法 | 达标标准 |
|----------|----------|----------|
| 功法熟练度 | 演示功法 | 熟练度达到大成以上 |
| 功法威力 | 功法威力测试 | 威力达到同阶平均水平 |
| 功法变化 | 功法变化演示 | 掌握3种以上变化 |

### 4.3 实战测试

| 测试项目 | 测试方法 | 达标标准 |
|----------|----------|----------|
| 战斗技巧 | 与同阶修士战斗 | 胜率达到60%以上 |
| 应变能力 | 应对突发情况 | 能够灵活应对 |
| 团队配合 | 团队战斗 | 能够与队友配合 |

### 4.4 心性测试

| 测试项目 | 测试方法 | 达标标准 |
|----------|----------|----------|
| 道心稳固 | 心魔幻境 | 能够通过心魔考验 |
| 意志坚定 | 意志考验 | 意志坚定不动摇 |
| 品德端正 | 品德考察 | 品德端正无劣迹 |

## 五、晋升失败处理

### 5.1 晋升失败原因

| 原因 | 说明 | 处理方式 |
|------|------|----------|
| 修为不足 | 修为未达标 | 继续修炼 |
| 考核未通过 | 考核成绩不合格 | 3年后可再次申请 |
| 资质不足 | 灵根资质不够 | 可通过特殊贡献弥补 |
| 品德问题 | 品德有劣迹 | 取消晋升资格 |

### 5.2 晋升失败惩罚

| 情况 | 惩罚 |
|------|------|
| 一般失败 | 3年内不得再次申请 |
| 严重失败 | 5年内不得再次申请 |
| 舞弊行为 | 取消晋升资格，降级处理 |

---

## 六、晋升仪式

### 6.1 晋升仪式流程

| 步骤 | 内容 | 说明 |
|------|------|------|
| 1 | 宣读晋升名单 | 由传功殿殿主宣读 |
| 2 | 颁发令牌 | 由宗主或长老颁发 |
| 3 | 更换服饰 | 更换新等级服饰 |
| 4 | 宣誓 | 宣读宗门誓词 |
| 5 | 叩拜宗门 | 向宗门祖师叩拜 |
| 6 | 领取奖励 | 领取晋升奖励 |

### 6.2 晋升仪式时间

| 等级 | 仪式时间 | 说明 |
|------|----------|------|
| 杂役→记名 | 每月一次 | 小型仪式 |
| 记名→外门 | 每季度一次 | 中型仪式 |
| 外门→内门 | 每年一次 | 大型仪式 |
| 内门→核心 | 每3年一次 | 大型仪式 |
| 核心→亲传 | 每5年一次 | 大型仪式 |
| 亲传→真传 | 每10年一次 | 盛大仪式 |
