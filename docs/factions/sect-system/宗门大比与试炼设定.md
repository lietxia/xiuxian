# 宗门大比与试炼设定

> 本文档详细说明宗门内部大比、试炼的具体规则。

---

## 一、宗门大比

### 1.1 大比类型

| 大比类型 | 周期 | 参与者 | 奖励 |
|----------|------|--------|------|
| 外门大比 | 每年一次 | 外门弟子 | 灵石+贡献+晋升机会 |
| 内门大比 | 每3年一次 | 内门弟子 | 灵石+贡献+晋升机会 |
| 核心弟子大比 | 每5年一次 | 核心弟子 | 灵石+贡献+晋升机会 |
| 宗门大比 | 每10年一次 | 全部弟子 | 灵石+贡献+晋升机会 |

### 1.2 大比规则

| 规则 | 说明 |
|------|------|
| 报名条件 | 符合弟子等级要求 |
| 比赛形式 | 淘汰赛制 |
| 比赛场地 | 宗门演武场 |
| 裁判 | 长老担任裁判 |
| 禁止事项 | 禁止使用禁术、禁止杀人 |

### 1.3 大比奖励

| 名次 | 奖励 | 说明 |
|------|------|------|
| 第一名 | 灵石10000枚+贡献10000点+晋升机会 | 最高奖励 |
| 第二名 | 灵石5000枚+贡献5000点 | 次高奖励 |
| 第三名 | 灵石3000枚+贡献3000点 | 第三奖励 |
| 第四-十名 | 灵石1000枚+贡献1000点 | 优秀奖励 |
| 参与奖 | 灵石100枚+贡献100点 | 参与奖励 |

## 三、宗门试炼

### 3.1 试炼类型

| 试炼类型 | 目的 | 参与者 | 奖励 |
|----------|------|--------|------|
| 新弟子试炼 | 考核新弟子 | 新弟子 | 弟子等级 |
| 晋升试炼 | 考核晋升资格 | 待晋升弟子 | 弟子等级 |
| 特殊试炼 | 特殊目的 | 特定弟子 | 特殊奖励 |

### 3.2 新弟子试炼

| 试炼内容 | 说明 | 通过标准 |
|----------|------|----------|
| 灵力测试 | 测试灵力储量 | 达到同阶平均水平 |
| 功法演示 | 演示所学功法 | 熟练度达到大成 |
| 实战测试 | 与同阶修士战斗 | 胜率达到60% |
| 心性测试 | 测试道心稳固 | 通过心魔考验 |

### 3.3 晋升试炼

| 试炼内容 | 说明 | 通过标准 |
|----------|------|----------|
| 修为测试 | 测试修为是否达标 | 达到晋升要求 |
| 功法测试 | 测试功法掌握程度 | 掌握足够功法 |
| 实战测试 | 与同阶修士战斗 | 胜率达到70% |
| 心性测试 | 测试道心稳固 | 通过心魔考验 |
| 贡献审核 | 审核贡献值 | 达到晋升要求 |
