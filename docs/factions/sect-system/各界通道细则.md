# 各界通道细则

> 本文档详细说明九大界域之间的通道系统，包括通道位置、通行条件、费用等。

---

## 一、通道概述

### 1.1 通道类型

| 类型 | 说明 | 特点 |
|------|------|------|
| 天然通道 | 天然形成的通道 | 稳定，但有限 |
| 传送阵 | 人工建造的传送阵 | 方便，但有成本 |
| 破界符 | 强行突破界面壁垒 | 危险，但无限制 |
| 飞升通道 | 飞升专用通道 | 特殊，需渡劫成功 |

### 1.2 通道等级

| 等级 | 说明 | 稳定性 | 成本 |
|------|------|--------|------|
| 一级 | 天然通道 | 极高 | 低 |
| 二级 | 传送阵 | 高 | 中等 |
| 三级 | 破界符 | 中等 | 高 |
| 四级 | 强行突破 | 低 | 极高 |
