# 宗门戒律与奖惩制度

> 本文档详细说明宗门内部戒律、奖惩的具体规则。

---

## 一、宗门戒律

### 1.1 基本戒律

| 戒律 | 内容 | 违反后果 |
|------|------|----------|
| 第一条 | 忠于宗门，不得背叛 | 废除修为，逐出宗门 |
| 第二条 | 尊重师长，不得欺师灭祖 | 废除修为，逐出宗门 |
| 第三条 | 团结同门，不得残害同门 | 废除修为，逐出宗门 |
| 第四条 | 遵守规矩，不得违反门规 | 视情节轻重处罚 |
| 第五条 | 勤奋修炼，不得懈怠 | 警告或处罚 |

### 1.2 严重禁令

| 禁令 | 内容 | 违反后果 |
|------|------|----------|
| 禁令一 | 禁止同门相残 | 废除修为，逐出宗门 |
| 禁令二 | 禁止偷盗宗门财物 | 废除修为，逐出宗门 |
| 禁令三 | 禁止泄露宗门机密 | 废除修为，逐出宗门 |
| 禁令四 | 禁止勾结外敌 | 废除修为，逐出宗门 |
| 禁令五 | 禁止修炼禁术 | 废除修为，逐出宗门 |

## 二、处罚等级

### 2.1 处罚等级表

| 处罚等级 | 适用情况 | 处罚内容 |
|----------|----------|----------|
| 警告 | 轻微违规 | 口头警告，记录在案 |
| 禁闭 | 一般违规 | 关闭禁闭室7-30天 |
| 扣除贡献 | 一般违规 | 扣除一定贡献值 |
| 降级 | 严重违规 | 降低弟子等级 |
| 废除修为 | 严重违规 | 废除修为，逐出宗门 |
| 处死 | 极其严重违规 | 处死 |

### 2.2 处罚标准

| 违规行为 | 处罚等级 | 说明 |
|----------|----------|------|
| 不敬师长 | 警告 | 口头警告 |
| 懈怠修炼 | 警告 | 口头警告 |
| 违反门规 | 禁闭 | 关闭禁闭室7天 |
| 打架斗殴 | 禁闭 | 关闭禁闭室15天 |
| 偷盗财物 | 扣除贡献 | 扣除贡献值 |
| 残害同门 | 废除修为 | 废除修为，逐出宗门 |
| 背叛宗门 | 废除修为 | 废除修为，逐出宗门 |
| 欺师灭祖 | 处死 | 处死 |

## 三、奖励制度

| 奖励等级 | 适用情况 | 奖励内容 |

| 拯救同门 | 大功 | 贡献点+灵石+晋升机会 |
| 击杀强敌 | 大功 | 贡献点+灵石+晋升机会 |
| 为宗门立下大功 | 特等功 | 贡献点+灵石+晋升机会+特殊奖励 |

## 四、执法堂

### 4.1 执法堂职责

| 职责 | 说明 |
|------|------|
| 执行戒律 | 执行宗门戒律 |
| 调查违规 | 调查违规行为 |
| 审判违规 | 审判违规弟子 |
| 执行处罚 | 执行处罚决定 |

### 4.2 执法堂组织

| 职位 | 人数 | 职责 |
|------|------|------|
| 堂主 | 1位 | 负责执法堂全面工作 |
| 副堂主 | 1-2位 | 辅助堂主工作 |
| 执法长老 | 3-6位 | 负责审判 |
| 执法弟子 | 12-36位 | 负责执行 |

### 4.3 执法流程

| 步骤 | 内容 | 负责人 |
|------|------|--------|
| 1 | 接收举报 | 执法弟子 |
| 2 | 调查取证 | 执法弟子 |
| 3 | 审理案件 | 执法长老 |
| 4 | 作出判决 | 执法长老 |
| 5 | 执行处罚 | 执法弟子 |

## 六、申诉制度

### 6.1 申诉条件

| 条件 | 说明 |
|------|------|
| 事实不清 | 案件事实不清 |
| 证据不足 | 证据不足 |
| 处罚过重 | 处罚明显过重 |
| 程序违法 | 执法程序违法 |

### 6.2 申诉流程

| 步骤 | 内容 | 负责人 |
|------|------|--------|
| 1 | 提交申诉书 | 当事人 |
| 2 | 审核申诉 | 执法堂 |
| 3 | 重新审理 | 长老会 |
| 4 | 作出裁决 | 长老会 |
| 5 | 执行裁决 | 执法堂 |

---

## 七、特殊情况处理

### 7.1 紧急情况

| 情况 | 处理方式 |
|------|----------|
| 同门相残 | 立即制止，严肃处理 |
| 背叛宗门 | 立即抓捕，严肃处理 |
| 外敌入侵 | 全力应对，保护宗门 |

### 7.2 复杂情况

| 情况 | 处理方式 |
|------|----------|
| 证据不足 | 继续调查，谨慎处理 |
| 双方各执一词 | 调查取证，公正处理 |
| 涉及长老 | 太上长老会审理 |
