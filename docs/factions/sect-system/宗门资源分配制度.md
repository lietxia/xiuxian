## 五、洞府分配

### 5.1 洞府等级

| 洞府等级 | 灵气浓度 | 设施 | 分配对象 |
|----------|----------|------|----------|
| 普通洞府 | 1倍 | 基础设施 | 杂役、记名弟子 |
| 外门洞府 | 2倍 | 基础设施 | 外门弟子 |
| 内门洞府 | 5倍 | 中级设施 | 内门弟子 |
| 核心洞府 | 10倍 | 高级设施 | 核心弟子 |
| 亲传洞府 | 20倍 | 顶级设施 | 亲传弟子 |
| 真传洞府 | 50倍 | 顶级设施 | 真传弟子 |
| 长老洞府 | 100倍 | 顶级设施 | 长老 |
| 宗主洞府 | 200倍 | 顶级设施 | 宗主 |

### 5.2 洞府分配规则

| 规则 | 说明 |
|------|------|
| 按等级分配 | 弟子等级决定洞府等级 |
| 先到先得 | 同等级洞府先到先得 |
| 贡献优先 | 贡献值高的弟子优先 |
| 特殊分配 | 特殊贡献者可获得更高级洞府 |
