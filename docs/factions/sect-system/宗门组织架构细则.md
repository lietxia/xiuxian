## 二、各职位详解

### 2.1 太上长老/老祖

| 项目 | 说明 |
|------|------|
| **人数** | 1-12位（视宗门等级） |
| **修为要求** | 大乘期以上（一品宗门） |
| **选拔方式** | 前任宗主或长老晋升 |
| **任期** | 无固定任期，终身制 |
| **权力** | 最高决策权，可罢免宗主 |
| **职责** | 宗门生死关头出手，守护宗门传承 |
| **日常** | 闭关修炼，不问俗世 |

**太上长老会议：**
- 召集条件：宗门生死存亡、宗主更替、重大决策
- 表决规则：三分之二以上同意方可执行
- 一票否决权：太上长老拥有一票否决权

---

### 2.2 宗主/掌门

| 项目 | 说明 |
|------|------|
| **人数** | 1位 |
| **修为要求** | 化神期以上（一品宗门） |
| **选拔方式** | 太上长老指定或长老会推举 |
| **任期** | 100-500年（可连任） |
| **权力** | 最高行政权、决策权 |
| **职责** | 统筹宗门全局，代表宗门对外交往 |

**宗主权力：**
- 人事任免权：任命副宗主、长老、峰主等
- 资源调配权：调配宗门资源
- 任务发布权：发布宗门任务
- 对外代表权：代表宗门对外交往
- 最终决策权：重大事务最终决定权

**宗主更替：**
- 正常更替：任期届满，由太上长老指定新宗主
- 非正常更替：宗主陨落、叛宗、失德等，由太上长老紧急指定
- 禅让制：宗主可主动禅让给继承人

---

### 2.3 副宗主

| 项目 | 说明 |
|------|------|
| **人数** | 1-3位 |
| **修为要求** | 化神期以上 |
| **选拔方式** | 宗主任命，太上长老同意 |
| **任期** | 与宗主同期 |
| **权力** | 辅助宗主，分管部分事务 |
| **职责** | 协助宗主管理宗门，分管不同领域 |

**副宗主分工：**
- 内务副宗主：负责宗门内部事务（弟子管理、资源分配等）
- 外务副宗主：负责宗门对外事务（外交、贸易、战争等）
- 传功副宗主：负责功法传承（藏经阁、传功殿等）

---

### 2.4 大长老

| 项目 | 说明 |
|------|------|
| **人数** | 3-12位 |
| **修为要求** | 炼虚期以上 |
| **选拔方式** | 长老中资历最深、修为最高者 |
| **任期** | 终身制 |
| **权力** | 决策建议权、监督权 |
| **职责** | 辅助宗主决策，监督宗主行为 |

**大长老职责：**
- 参与宗门重大决策
- 监督宗主行为是否失当
- 调解宗门内部矛盾
- 指导后辈修炼

---

### 2.5 长老

| 项目 | 说明 |
|------|------|
| **人数** | 12-36位（视宗门规模） |
| **修为要求** | 元婴期以上 |
| **选拔方式** | 核心弟子晋升或外聘 |
| **任期** | 无固定任期 |
| **权力** | 事务管理权 |
| **职责** | 分管宗门某方面事务 |

**长老分类：**
- 内部长老：从宗门内部晋升
- 客卿长老：外聘高手，享受长老待遇
- 荣誉长老：对宗门有大贡献者

**长老职责分工：**
- 传功长老：负责功法传承
- 执法长老：负责宗门戒律
- 丹药长老：负责炼丹事务
- 炼器长老：负责炼器事务
- 阵法长老：负责阵法事务
- 外事长老：负责对外事务
- 内务长老：负责内部事务

---

### 2.6 峰主

| 项目 | 说明 |
|------|------|
| **人数** | 与山峰数量相同 |
| **修为要求** | 元婴期以上 |
| **选拔方式** | 宗主任命或山峰内推举 |
| **任期** | 无固定任期 |
| **权力** | 山峰管理权 |
| **职责** | 管理山峰一切事务 |

**峰主职责：**
- 管理山峰弟子
- 分配山峰资源
- 维护山峰设施
- 指导山峰弟子修炼

---

### 2.7 堂主

| 项目 | 说明 |
|------|------|
| **人数** | 与堂口数量相同 |
| **修为要求** | 金丹期以上 |
| **选拔方式** | 宗主任命 |
| **任期** | 100年（可连任） |
| **权力** | 专项事务管理权 |
| **职责** | 管理专项事务 |

**堂口设置：**
- 藏经堂：管理功法秘籍
- 丹药堂：管理炼丹事务
- 炼器堂：管理炼器事务
- 阵法堂：管理阵法事务
- 灵兽堂：管理灵兽事务
- 任务堂：管理宗门任务
- 执法堂：执行宗门戒律
- 传功堂：传授功法
- 外事堂：处理对外事务
- 内务堂：处理内部事务

---

| **人数** | 与殿堂数量相同 |
| **修为要求** | 金丹期以上 |
| **选拔方式** | 宗主任命 |

### 2.9 护法

| 项目 | 说明 |
|------|------|
| **人数** | 12-72位 |
| **修为要求** | 金丹期以上 |
| **选拔方式** | 核心弟子晋升 |
| **任期** | 无固定任期 |
| **权力** | 武力执行权 |
| **职责** | 维护宗门安全，执行战斗任务 |

**护法分类：**

| 项目 | 说明 |
|------|------|
| **人数** | 数十至数百位 |
| **修为要求** | 筑基期以上 |
| **选拔方式** | 内门弟子晋升 |
| **任期** | 50年（可连任） |
| **权力** | 事务执行权 |
| **职责** | 负责具体事务 |

**执事分级：**
- 高级执事：负责重要事务
- 中级执事：负责一般事务
- 初级执事：负责基础事务

## 三、弟子等级详解

### 3.1 真传弟子/圣子/道子

| 项目 | 说明 |
|------|------|
| **人数** | 3-12位 |
| **资质要求** | 天灵根或特殊体质 |
| **修为要求** | 金丹期以上 |
| **选拔方式** | 宗主或太上长老亲定 |
| **待遇** | 最高弟子待遇 |
| **特权** | 可参与宗门决策 |

**真传弟子特权：**
- 使用宗门最高级修炼资源
- 可进入藏经阁最高层
- 可使用宗门秘境
- 参与宗门重大决策
- 未来宗主候选人

---

## 五、宗门会议制度

### 5.1 长老会议

| 项目 | 说明 |
|------|------|
| **召集人** | 宗主或大长老 |
| **参会人员** | 全部长老 |
| **召开频率** | 每月一次 |
| **议事内容** | 宗门日常事务 |
| **表决规则** | 多数同意 |

### 5.2 峰主会议

| 项目 | 说明 |
|------|------|
| **召集人** | 宗主 |
| **参会人员** | 全部峰主 |
| **召开频率** | 每季度一次 |
| **议事内容** | 各峰事务协调 |
| **表决规则** | 多数同意 |

### 5.3 紧急会议

| 项目 | 说明 |
|------|------|
| **召集人** | 宗主或太上长老 |
| **参会人员** | 相关人员 |
| **召开频率** | 随时 |
| **议事内容** | 紧急事务 |
| **表决规则** | 宗主决定 |
