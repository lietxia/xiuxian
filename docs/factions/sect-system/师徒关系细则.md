# 师徒关系细则

> 本文档详细说明修仙界师徒关系的具体规则、流程、义务等。

---

## 一、拜师条件

### 1.1 弟子条件

| 条件 | 要求 | 说明 |
|------|------|------|
| 灵根资质 | 有灵根即可 | 最基本条件 |
| 年龄限制 | 不超过30岁 | 练气期以下 |
| 品行端正 | 无劣迹 | 品德要求 |
| 心性坚定 | 道心稳固 | 心性要求 |
| 机缘巧合 | 师傅认可 | 缘分要求 |

### 1.2 师傅条件

| 条件 | 要求 | 说明 |
|------|------|------|
| 修为要求 | 高于弟子两个大境界以上 | 确保指导能力 |
| 收徒名额 | 每位师傅最多3-5名亲传弟子 | 确保教学质量 |
| 品德要求 | 无重大劣迹 | 师德要求 |
| 宗门许可 | 需宗门批准 | 正式收徒需报备 |

---

## 二、拜师流程

### 2.1 拜师前准备

| 步骤 | 内容 | 说明 |
|------|------|------|
| 1 | 寻找师傅 | 通过宗门安排或自行寻找 |
| 2 | 了解师傅 | 了解师傅的修炼方向、性格、教学风格 |
| 3 | 表达意愿 | 向师傅表达拜师意愿 |
| 4 | 师傅考察 | 师傅考察弟子资质、心性、品德 |
| 5 | 双方确认 | 双方确认师徒关系 |

### 2.2 拜师仪式

| 步骤 | 内容 | 说明 |
|------|------|------|
| 1 | 净身沐浴 | 弟子净身沐浴，表示虔诚 |
| 2 | 敬茶 | 向师傅敬茶，表示尊敬 |
| 3 | 叩拜 | 向师傅三叩九拜，表示敬意 |
| 4 | 赐名 | 师傅赐予道号，正式入门 |
| 5 | 赐物 | 师傅赐予入门礼物（功法、法器等） |
| 6 | 立誓 | 弟子立下心魔誓言，承诺尊师重道 |
| 7 | 天地见证 | 请天地见证师徒关系 |
| 8 | 登记备案 | 在宗门登记备案 |

### 2.3 拜师仪式流程图

```
净身沐浴 → 敬茶 → 叩拜 → 赐名 → 赐物 → 立誓 → 天地见证 → 登记备案
```

## 三、师徒义务

### 3.1 师傅义务

| 义务 | 内容 | 要求 |
|------|------|------|
| 传授功法 | 传授修炼功法 | 倾囊相授，不得藏私 |
| 指导修炼 | 指导弟子修炼 | 耐心指导，因材施教 |
| 保护弟子 | 保护弟子安全 | 尽力保护，不得见死不救 |
| 提供资源 | 提供修炼资源 | 量力提供，不得吝啬 |
| 教育品德 | 教育弟子品德 | 言传身教，以身作则 |
| 解答疑惑 | 解答弟子疑惑 | 耐心解答，不得敷衍 |

### 3.2 弟子义务

| 义务 | 内容 | 要求 |
|------|------|------|
| 尊师重道 | 尊重师傅 | 必须遵守，不得违逆 |
| 勤奋修炼 | 努力修炼 | 必须做到，不得懈怠 |
| 孝敬师傅 | 孝敬师傅 | 必须遵守，不得忘恩负义 |
| 传承功法 | 传承师傅功法 | 必须做到，不得断绝传承 |
| 维护师门 | 维护师门荣誉 | 必须做到，不得损害师门 |
| 侍奉师傅 | 侍奉师傅起居 | 量力而行，不得推诿 |

## 四、师徒类型

### 4.1 按关系亲疏分类

| 类型 | 说明 | 特点 | 数量限制 |
|------|------|------|----------|
| 亲传弟子 | 师傅亲自传授 | 关系最亲密，获得资源最多 | 每位师傅最多3-5名 |
| 内门弟子 | 师门内门弟子 | 关系较亲密，获得资源较多 | 不限 |
| 外门弟子 | 师门外门弟子 | 关系一般，获得资源一般 | 不限 |
| 记名弟子 | 师傅记名弟子 | 关系较疏远，获得资源较少 | 不限 |

### 4.2 按传承方式分类

| 类型 | 说明 | 特点 | 适用情况 |
|------|------|------|----------|
| 单传 | 只收一个徒弟 | 传承集中，资源集中 | 核心功法传承 |
| 少传 | 收少数徒弟 | 传承集中，资源较集中 | 重要功法传承 |
| 多传 | 收多个徒弟 | 传承分散，资源分散 | 普通功法传承 |
| 广传 | 广泛传授 | 传承广泛，资源分散 | 基础功法传承 |

## 五、师徒情感

### 5.1 师徒情感建立

| 阶段 | 情感 | 说明 | 时间 |
|------|------|------|------|
| 初期 | 尊敬 | 弟子尊敬师傅 | 0-10年 |
| 中期 | 信任 | 建立信任关系 | 10-50年 |
| 后期 | 亲情 | 如同父子亲情 | 50-100年 |
| 长期 | 羁绊 | 建立深厚羁绊 | 100年以上 |

### 5.2 师徒情感表现

| 表现 | 说明 | 例子 |
|------|------|------|
| 关心爱护 | 互相关心爱护 | 师傅关心弟子修炼进度 |
| 舍身相救 | 舍身相救对方 | 师傅为弟子挡下致命一击 |
| 共同进退 | 共同面对困难 | 师徒共同对抗敌人 |
| 传承意志 | 传承意志信念 | 弟子继承师傅遗志 |

## 六、师徒禁忌

### 6.1 师徒禁忌

| 禁忌 | 说明 | 后果 |
|------|------|------|
| 欺师灭祖 | 欺骗或杀害师傅 | 修仙界共诛，废除修为 |
| 背叛师门 | 背叛师门 | 驱逐出师门，废除修为 |
| 不敬师长 | 不尊敬师傅 | 警告或惩罚 |
| 泄露功法 | 泄露师门功法 | 废除修为，逐出师门 |
| 忘恩负义 | 忘记师傅恩情 | 修仙界唾弃 |

### 6.2 师徒冲突处理

| 冲突类型 | 说明 | 处理方式 |
|----------|------|----------|
| 修炼分歧 | 修炼理念不同 | 沟通解决，尊重弟子选择 |
| 资源分配 | 资源分配不均 | 协商解决，公平分配 |
| 功法传承 | 功法传承问题 | 按规矩处理，不得偏袒 |
| 性格冲突 | 性格不合 | 相互理解，求同存异 |

## 七、师徒关系解除

### 7.1 解除原因

| 原因 | 说明 | 常见程度 |
|------|------|----------|
| 师傅陨落 | 师傅去世 | 常见 |
| 弟子叛逆 | 弟子背叛师门 | 较少 |
| 师徒反目 | 师徒关系破裂 | 较少 |
| 功法冲突 | 功法修炼冲突 | 罕见 |
| 其他原因 | 其他原因 | 不定 |

### 7.2 解除流程

| 步骤 | 内容 | 说明 |
|------|------|------|
| 1 | 提出解除 | 一方提出解除师徒关系 |
| 2 | 对方同意 | 对方同意解除 |
| 3 | 资源分割 | 分割共同资源 |
| 4 | 举行仪式 | 举行解除仪式 |
| 5 | 公告天下 | 公告修仙界 |

### 7.3 解除后果

| 后果 | 说明 | 影响 |
|------|------|------|
| 资源分割 | 分割共同资源 | 经济影响 |
| 感情伤害 | 双方感情受伤 | 心理影响 |
| 社会影响 | 社会观感变化 | 社会影响 |
| 修炼影响 | 修炼可能受影响 | 修炼影响 |
