# 道侣/婚姻制度细则

> 本文档详细说明修仙界道侣/婚姻制度的具体规则、流程、义务等。

---

## 一、道侣条件

### 1.1 基本条件

| 条件 | 要求 | 说明 |
|------|------|------|
| 情投意合 | 双方情投意合 | 必须满足 |
| 境界相近 | 境界相差不超过两个大境界 | 建议满足 |
| 功法匹配 | 功法可以匹配 | 建议满足 |
| 性格相合 | 性格相合 | 建议满足 |
| 背景合适 | 背景合适 | 可选条件 |

### 1.2 特殊条件

| 条件 | 要求 | 说明 |
|------|------|------|
| 宗门许可 | 需宗门批准 | 宗门弟子需报备 |
| 师傅同意 | 需师傅同意 | 亲传弟子需师傅同意 |
| 家族同意 | 需家族同意 | 家族弟子需家族同意 |
| 道心考验 | 通过道心考验 | 确保道心稳固 |

## 二、结为道侣流程

### 2.1 结侣前准备

| 步骤 | 内容 | 说明 |
|------|------|------|
| 1 | 相识相知 | 相识并了解对方 |
| 2 | 表明心意 | 表明结为道侣的心意 |
| 3 | 对方同意 | 对方同意结为道侣 |
| 4 | 双方家长/师门会面 | 双方家长或师门会面 |
| 5 | 确定聘礼/嫁妆 | 确定聘礼或嫁妆 |
| 6 | 选择结侣日期 | 选择吉日 |

### 2.2 结侣仪式

| 步骤 | 内容 | 说明 |
|------|------|------|
| 1 | 净身沐浴 | 双方净身沐浴 |
| 2 | 交换信物 | 交换定情信物 |
| 3 | 立下誓言 | 立下道侣誓言 |
| 4 | 天地见证 | 请天地见证 |
| 5 | 宴请宾客 | 宴请亲朋好友 |
| 6 | 共同修炼 | 共同修炼 |
| 7 | 登记备案 | 在宗门或家族登记备案 |

### 2.3 结侣仪式流程图

```
净身沐浴 → 交换信物 → 立下誓言 → 天地见证 → 宴请宾客 → 共同修炼 → 登记备案
```

## 四、双修之法

### 4.1 双修类型

| 类型 | 说明 | 特点 | 适用情况 |
|------|------|------|----------|
| 灵气双修 | 通过灵气互通修炼 | 最常见 | 所有道侣 |

| 功法双修 | 通过功法配合修炼 | 特殊双修 | 功法匹配道侣 |

| 共同战斗 | 共同战斗能力增强 | 战斗力提升30% |
| 延长寿命 | 可能延长寿命 | 寿命延长10%-20% |

### 4.3 双修注意事项

| 注意事项 | 说明 | 重要性 |
|----------|------|--------|
| 功法匹配 | 功法要匹配 | 高 |
| 境界相近 | 境界要相近 | 中等 |
| 互相信任 | 互相信任 | 高 |
| 循序渐进 | 循序渐进 | 中等 |
| 注意安全 | 注意安全 | 高 |

| 原因 | 说明 | 常见程度 |
|------|------|----------|
| 感情破裂 | 感情破裂 | 常见 |
