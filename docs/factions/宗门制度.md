# 宗门制度

> 宗门是修仙世界中最重要的组织形式，是修仙者修炼和生活的主要场所。

---

## 宗门概述

### 什么是宗门？

宗门是修仙世界中由修仙者组成的组织，是修仙者修炼和生活的主要场所。宗门为弟子提供功法传承、资源支持、安全保障等服务。

### 宗门的特点

| 特点 | 说明 |
|------|------|
| 组织性 | 有组织的修仙团体 |
| 传承性 | 有功法传承体系 |
| 等级性 | 有明确的等级制度 |
| 规范性 | 有严格的门规戒律 |
| 服务性 | 为弟子提供服务 |

## 贡献体系

### 贡献的定义

贡献是弟子为宗门做出的贡献，是获取资源的重要途径。

### 贡献来源

| 来源 | 说明 | 贡献值 |
|------|------|--------|
| 完成任务 | 完成宗门任务 | 根据难度 |
| 缴纳资源 | 缴纳灵石、材料 | 根据价值 |
| 立下功劳 | 为宗门立功 | 根据功劳 |
| 授课传道 | 传授功法知识 | 根据效果 |
| 探索秘境 | 探索秘境带回资源 | 根据收获 |

### 贡献用途

| 用途 | 说明 | 消耗 |
|------|------|------|
| 兑换功法 | 兑换宗门功法 | 根据等级 |
| 兑换丹药 | 兑换宗门丹药 | 根据品质 |
| 兑换法宝 | 兑换宗门法宝 | 根据品质 |
| 使用设施 | 使用修炼设施 | 根据设施 |
| 获得指导 | 获得长老指导 | 根据时间 |

## 门规戒律

### 基本门规

| 门规 | 说明 | 违规后果 |
|------|------|----------|
| 忠于宗门 | 忠于宗门利益 | 驱逐出宗 |
| 尊重师长 | 尊重师门长辈 | 警告或惩罚 |
| 团结同门 | 与同门团结 | 警告或惩罚 |
| 遵守规矩 | 遵守宗门规矩 | 相应惩罚 |
| 不得叛宗 | 不得背叛宗门 | 废除修为 |

### 严重禁令

| 禁令 | 说明 | 违规后果 |
|------|------|----------|
| 禁止同门相残 | 不得杀害同门 | 废除修为 |
| 禁止偷盗 | 不得偷盗宗门财物 | 废除修为 |
| 禁止泄密 | 不得泄露宗门机密 | 废除修为 |
| 禁止欺师灭祖 | 不得欺师灭祖 | 废除修为 |
| 禁止勾结外敌 | 不得勾结外敌 | 废除修为 |
