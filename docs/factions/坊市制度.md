# 坊市制度

> 坊市乃修仙界的交易枢纽，是散修赖以生存、宗门获取资源、商盟赚取利润的核心场所。一个繁荣的坊市，往往是一座修仙城市的命脉所在。

---

## 一、坊市概述

### 1.1 什么是坊市

坊市是修仙界中专门为修士提供交易服务的场所，类似于凡人世界的集市与商街。坊市通常由某个宗门、家族或商盟建立并管理，为修士提供买卖灵材、丹药、法器、功法等修仙资源的平台。

### 1.2 坊市的等级

| 等级 | 规模 | 管理者 | 说明 |
|------|------|--------|------|
| 顶级坊市 | 方圆百里 | 大宗门或商盟总部 | 修仙界交易中心，汇聚天下奇珍 |
| 一流坊市 | 方圆十里 | 中型宗门或大型商盟 | 区域性交易中心 |
| 二流坊市 | 方圆数里 | 小宗门或家族 | 本地交易为主 |
| 临时坊市 | 不固定 | 散修自发组织 | 秘境开启时临时形成 |

## 二、坊市结构

### 2.1 内域与外域

大型坊市通常分为内域和外域两个区域，以灵气浓度和交易内容区分。

| 区域 | 灵气浓度 | 交易内容 | 进入门槛 |
|------|----------|----------|----------|
| 内域 | 极高（设有聚灵阵） | 高阶灵材、金丹期以上丹药、法宝 | 需缴纳入域费或持有令牌 |
| 外域 | 一般 | 中低阶灵材、练气筑基期丹药、法器 | 自由进出 |

**内域特点：**
- 灵气浓度是外域的数倍，适合修士临时修炼
- 店铺多为知名商盟分号，信誉有保障
- 交易物品档次更高，价格也更昂贵
- 治安更好，有专门的护卫巡逻

**外域特点：**
- 灵气浓度普通，与外界无异
- 店铺良莠不齐，需自行辨别真伪
- 交易物品覆盖面广，从低阶到中阶皆有
- 人员混杂，需提防扒手与骗子

### 2.2 功能分区

| 区域 | 功能 | 说明 |
|------|------|------|
| 商业街 | 店铺林立 | 各类固定店铺，买卖灵材丹药 |
| 自由交易区 | 摆摊交易 | 散修自由摆摊，收取摊位费 |
| 拍卖行 | 拍卖珍品 | 定期举办拍卖会 |
| 任务大厅 | 发布任务 | 类似宗门悬赏台，面向所有修士 |
| 斗法场 | 比武切磋 | 解决纷争或赌斗 |
| 灵食楼 | 餐饮休憩 | 提供灵食灵酒，修士社交场所 |
| 客栈 | 住宿休息 | 提供带聚灵阵的客房 |

## 三、宅院制度

### 3.1 宅院概述

坊市内域设有专供修士长期居住的宅院，这些宅院建在灵脉之上，灵气浓郁，是修士修炼的理想场所。

### 3.2 只租不卖原则

坊市宅院遵循"只租不卖"的原则，这是坊市管理者维护自身利益的核心制度。

| 原因 | 说明 |
|------|------|
| 灵脉归属 | 灵脉属于坊市管理者，宅院只是附着物 |
| 长期收益 | 租金是坊市的重要收入来源 |
| 控制权 | 保持管理者对内域的绝对控制 |
| 防止垄断 | 避免某个势力买断所有宅院 |

### 3.3 宅院等级与租金

| 等级 | 面积 | 灵气浓度 | 年租金 | 说明 |
|------|------|----------|--------|------|
| 甲等宅院 | 独立院落，百丈见方 | 灵脉核心 | 1000中品灵石 | 通常为金丹期修士租用 |
| 乙等宅院 | 独立院落，五十丈见方 | 灵脉次核心 | 500中品灵石 | 筑基后期修士租用 |
| 丙等宅院 | 独立院落，三十丈见方 | 灵脉中段 | 100中品灵石 | 筑基中期修士租用 |
| 丁等宅院 | 联排小院，十丈见方 | 灵脉末段 | 50中品灵石 | 筑基初期修士租用 |

### 3.4 内域名额拍卖

由于内域宅院数量有限，而需求旺盛，坊市管理者通常采用拍卖方式分配新空出的宅院。

**拍卖规则：**
- 每年定期举行一次宅院拍卖会
- 竞拍者需缴纳保证金（100中品灵石）
- 价高者得，租期通常为十年至百年
- 租期届满后，原租户有优先续租权
- 若原租户放弃续租，则重新拍卖

**竞拍资格：**
- 修为不低于筑基期
- 无坊市违规记录
- 持有坊市认可的身份令牌

## 四、散修交易日

### 4.1 制度概述

坊市每月设有固定的散修交易日，专门为散修提供低价交易的机会。这一制度既照顾了散修的利益，也为坊市带来了人气与税收。

### 4.2 交易日安排

| 日期 | 名称 | 说明 |
|------|------|------|
| 每月十号 | 灵材交易日 | 以灵药、矿石、妖兽材料为主 |
| 每月二十号 | 丹药法器交易日 | 以丹药、法器、符箓为主 |
| 每月三十号 | 自由交易日 | 不限品类，自由交易 |

### 4.3 散修交易日规则

| 规则 | 说明 |
|------|------|
| 摊位费减免 | 散修交易日摊位费减半 |
| 税率优惠 | 交易税从5%降至3% |
| 禁止大宗收购 | 禁止商盟大宗收购，保护散修利益 |
| 设立鉴定台 | 坊市免费提供物品鉴定服务 |
| 禁止斗法 | 交易日期间严禁斗法，违者重罚 |

### 4.4 散修交易日的意义

散修是修仙界的底层力量，他们没有宗门庇护，资源匮乏，生存艰难。散修交易日为他们提供了一个相对公平的交易环境，让他们能够以合理的价格出售自己辛苦采集的灵材，换取修炼所需的资源。

## 五、拍卖会规则

### 5.1 拍卖会等级

| 等级 | 频率 | 拍品档次 | 参与资格 |
|------|------|----------|----------|
| 年度大拍 | 每年一次 | 元婴期以上宝物 | 需请柬或金丹期以上修为 |
| 季度中拍 | 每季一次 | 金丹期宝物 | 需请柬或筑基期以上修为 |
| 月度小拍 | 每月一次 | 筑基期宝物 | 自由参加 |

### 5.2 请柬制度

年度大拍和季度中拍采用请柬制度，只有收到请柬的修士方可参加。

**请柬获取方式：**
- 坊市VIP客户自动获得
- 在坊市消费达到一定额度
- 由宗门长老或商盟高层推荐
- 拍卖行特别邀请

### 5.3 席位设置

| 席位 | 位置 | 费用 | 特权 |
|------|------|------|------|
| 贵宾室 | 二楼包厢，视野最佳 | 100中品灵石 | 优先竞拍、免费茶水灵果 |
| 普通席 | 一楼大厅 | 免费（需请柬） | 正常竞拍 |
| 站席 | 大厅后方 | 免费 | 仅可观看，不可竞拍 |

### 5.4 拍卖规则

| 规则 | 说明 |
|------|------|
| 竞价方式 | 举牌竞价，每次加价不低于起拍价的10% |
| 成交确认 | 三次无人加价后落槌成交 |
| 付款方式 | 现场支付灵石，或以物抵价（需鉴定） |
| 违约惩罚 | 竞拍成功不付款者，没收保证金并列入黑名单 |
| 保密条款 | 拍卖行对买家身份严格保密 |

## 六、黑市规则

### 6.1 黑市概述

黑市是坊市中隐藏的灰色交易场所，专门交易来路不明或被禁止的物品。

### 6.2 黑市交易物品

| 物品类别 | 说明 | 风险 |
|----------|------|------|
| 赃物 | 偷盗、抢劫所得 | 可能被追查 |
| 禁药 | 宗门禁止的邪丹、毒药 | 可能被执法堂追究 |
| 邪功 | 魔道、邪道功法 | 修炼可能走火入魔 |
| 走私品 | 逃避关税的正规货物 | 被查获将被没收 |
| 情报 | 各势力的机密情报 | 泄露可能招来杀身之祸 |

### 6.3 商盟抽成

黑市交易由特定商盟担保，商盟从中抽取高额佣金。

| 交易类型 | 抽成比例 | 说明 |
|----------|----------|------|
| 普通赃物 | 70% | 卖家仅得30% |
| 禁药邪功 | 70% | 风险越高抽成越重 |
| 走私品 | 50% | 相对正规，抽成略低 |
| 情报交易 | 80% | 情报价值极高，抽成最重 |

**为何抽成如此之高？** 黑市商盟承担着巨大的风险——一旦被宗门或仙盟查获，商盟将面临灭顶之灾。高抽成是对风险的补偿，也是商盟维持黑市运转的动力。

### 6.4 黑市规则

| 规则 | 说明 |
|------|------|
| 匿名交易 | 买卖双方均以代号相称，不问真名 |
| 现货现付 | 不接受赊账，当场银货两讫 |
| 禁止追踪 | 禁止追踪买家去向，违者格杀勿论 |
| 保密义务 | 参与者不得泄露黑市信息 |
| 自担风险 | 购买物品后发现有问题，黑市概不负责 |

## 七、税率差异

### 7.1 税率概述

坊市对所有交易收取一定比例的交易税，这是坊市管理者的重要收入来源。

### 7.2 不同宗门店铺的税率

| 店铺归属 | 税率 | 说明 |
|----------|------|------|
| 坊市自营店铺 | 免税 | 坊市管理者自己的店铺 |
| 宗门附属店铺 | 3% | 与坊市管理者有合作关系的宗门 |
| 普通修士店铺 | 5% | 在坊市租用店铺的普通修士 |
| 外来商队 | 8% | 从外地前来的商队 |
| 散修摊位 | 5%（交易日3%） | 散修临时摊位 |

### 7.3 税率差异的原因

税率差异体现了坊市管理者的利益考量：
- 自营店铺免税，是为了最大化自身利润
- 宗门附属店铺低税，是为了维护合作关系
- 外来商队高税，是为了保护本地商户利益
- 散修交易日减税，是为了吸引更多散修前来交易

### 7.4 逃税的后果

| 行为 | 惩罚 |
|------|------|
| 隐瞒交易金额 | 补缴税款并处以三倍罚款 |
| 私下交易逃避税收 | 没收货物并处以十倍罚款 |
| 多次逃税 | 列入坊市黑名单，禁止进入 |
| 组织逃税 | 由执法队缉拿，交由宗门处置 |

## 八、经典商会

### 8.1 万宝阁

万宝阁是修仙界最大的商会之一，以"万宝齐聚"闻名天下。

| 项目 | 说明 |
|------|------|
| 总部 | 位于修仙界最大的坊市——天宝城 |
| 经营范围 | 灵材、丹药、法器、功法、情报，无所不包 |
| 分号分布 | 修仙界各大坊市均有分号 |
| 特色服务 | 鉴定宝物、寄售拍卖、定制法器 |
| 信誉等级 | 修仙界公认的金字招牌，信誉极高 |

### 8.2 摘星楼

摘星楼以会员制著称，只接待有身份、有地位的修士。

| 项目 | 说明 |
|------|------|
| 总部 | 位于摘星峰，常年云雾缭绕 |
| 会员门槛 | 金丹期以上修士，或有元婴期修士推荐 |
| 经营范围 | 高阶灵材、珍稀丹药、上古法宝 |
| 特色服务 | 私人定制、专属顾问、优先竞拍 |
| 会员等级 | 铜牌→银牌→金牌→紫金→至尊 |

**摘星楼会员权益：**

| 等级 | 年消费要求 | 权益 |
|------|------------|------|
| 铜牌 | 100中品灵石 | 九五折优惠 |
| 银牌 | 1000中品灵石 | 九折优惠+优先竞拍 |
| 金牌 | 10000中品灵石 | 八五折+专属顾问 |
| 紫金 | 100000中品灵石 | 八折+私人定制 |
| 至尊 | 邀请制 | 七折+全方位服务 |

### 8.3 青鸾商盟

青鸾商盟是修仙界最具传奇色彩的商会，据说其背后有化神期老怪撑腰。

| 项目 | 说明 |
|------|------|
| 总部 | 位置神秘，无人知晓 |
| 经营范围 | 禁品、情报、暗杀、走私，灰色地带为主 |
| 特色服务 | 代办棘手之事、提供避难所 |
| 信誉等级 | 黑白通吃，承诺必践 |
| 风险提示 | 与青鸾商盟交易需谨慎，可能招来麻烦 |

**青鸾商盟的规矩：**
- 不问来路，不问去向
- 一手交钱，一手交货
- 任务完成，概不负责
- 泄露商盟信息，格杀勿论

## 九、写作参考：坊市剧情设计

### 9.1 经典坊市剧情模式

| 剧情模式 | 说明 | 代表作品 |
|----------|------|----------|
| 淘宝捡漏 | 主角在摊位上发现被低估的宝物 | 《凡人修仙传》 |
| 拍卖争夺 | 拍卖会上与人竞价争夺宝物 | 常见桥段 |
| 黑市交易 | 在黑市购买禁品或出售赃物 | 多部作品 |
| 坊市斗法 | 在坊市中与人发生冲突 | 常见桥段 |
| 商盟任务 | 接取商盟任务赚取灵石 | 常见桥段 |

### 9.2 坊市剧情设计要点

**阶层碰撞：** 坊市是修仙界各阶层交汇的场所——宗门弟子、家族子弟、散修、商人都在这里。利用阶层差异制造冲突，是坊市剧情的核心魅力。

**利益纠葛：** 坊市中的一切都围绕利益展开。买卖、竞争、欺诈、抢劫——利益驱动着每一个人的行为。

**信息流通：** 坊市是信息汇聚之地。主角可以在坊市中获取关键情报，推动剧情发展。

**社会百态：** 坊市是修仙界社会的缩影。在这里，可以看到散修的艰辛、宗门弟子的傲慢、商人的精明、黑市的险恶。

### 9.3 坊市设定注意事项

| 要点 | 说明 |
|------|------|
| 经济合理性 | 坊市的运转必须有合理的经济逻辑 |
| 安全保障 | 坊市必须有足够的安全保障，否则无人敢来 |
| 规则执行 | 规则必须被执行，否则形同虚设 |
| 利益平衡 | 管理者、商户、顾客之间的利益必须平衡 |
| 历史感 | 好的坊市应该有历史，有故事，有传说 |
