# 散修联盟

> 散修联盟是散修自发组成的互助组织，为散修提供资源交流和安全保障。本文详细介绍散修联盟的组织与功能。

---

## 散修联盟概述

散修联盟是由散修自发组建的互助组织，为散修提供资源交易、任务对接、安全互助等服务。

### 联盟特点

| 特点 | 说明 |
|------|------|
| 松散组织 | 非严格的宗门制度 |
| 互助为主 | 以互助互利为宗旨 |
| 自愿加入 | 来去自由 |
| 资源共享 | 共享部分修炼资源 |

---

## 组织结构

| 职位 | 实力 | 职责 |
|------|------|------|
| 盟主 | 化神期以上 | 统领联盟 |
| 长老 | 元婴期 | 管理各地分部 |
| 执事 | 金丹期 | 处理日常事务 |
| 成员 | 金丹以下 | 普通散修成员 |

## 联盟服务

| 服务 | 费用 | 说明 |
