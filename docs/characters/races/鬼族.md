# 鬼族

> 鬼族是修仙界中以灵魂形态存在的种族，与冥界关系密切。本文详细介绍鬼族的起源、修炼体系与特点。

---

## 鬼族概述

鬼族是由亡灵修炼而成的种族，以灵魂形态存在，与冥界关系密切。鬼族修炼阴属性功法，擅长灵魂攻击和幻术。

### 鬼族特点

| 特点 | 说明 |
|------|------|
| 灵魂形态 | 以灵魂形态存在，无实体肉身 |
| 阴气亲和 | 天生亲和阴气 |
| 灵魂攻击 | 擅长灵魂类攻击 |
| 幻术精通 | 精通幻术和迷惑之术 |
| 隐匿能力 | 隐匿能力极强 |

---

| 境界 | 对应人族 | 特征 | 能力 |

| 鬼将 | 金丹期 | 鬼体凝实 | 灵魂攻击 |
| 鬼王 | 元婴期 | 鬼体完善 | 掌控阴气 |
| 鬼帝 | 化神期 | 鬼域自成 | 鬼域降临 |

## 鬼族与人族关系

| 关系 | 说明 |
|------|------|
| 疏远 | 鬼族多居于冥界，与人族交集少 |
| 恐惧 | 人族普遍对鬼族有恐惧心理 |
| 利用 | 某些修士利用鬼族功法修炼 |
| 轮回关联 | 鬼族掌管轮回，与人族死后相关 |
