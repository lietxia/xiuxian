# 人族

> 人族是修仙界数量最多、分布最广的种族，以其适应性和创造力著称。本文详细介绍人族的起源、特点、修炼体系与社会结构。

---

## 人族概述

人族是修仙界的主体种族，数量最多、分布最广。人族虽天生体质不如妖族、法力不如魔族，但凭借悟性、适应性和创造力，成为修仙界最强大的种族之一。

### 人族特点

| 特点 | 说明 |
|------|------|
| 悟性极高 | 人族悟性为各族之首 |
| 适应性极强 | 能在各种环境修炼 |
| 创造力突出 | 创造了最多的功法、阵法、丹药 |
| 数量庞大 | 人口为各族最多 |
| 社会复杂 | 社会结构最为复杂 |

---

## 人族起源

人族由女娲以五彩石和混沌灵泥仿照神族形态创造，先天不足但后天潜力无限。人族先天体质弱于妖族和魔族，但悟性和适应性强于其他种族。

### 起源传说

| 传说 | 内容 |
|------|------|
| 女娲造人 | 女娲以灵泥造人，赋予灵智 |
| 三皇教化 | 燧人、伏羲、神农教导人族文明 |
| 五帝传承 | 黄帝、颛顼、帝喾、尧、舜传承修炼之法 |
| 人族自立 | 人族自强不息，建立修仙文明 |

## 人族修炼体系

### 灵根系统

| 灵根类型 | 稀有度 | 修炼速度 | 上限 |
|---------|--------|---------|------|
| 天灵根（单属性） | 极稀有 | 极快 | 大乘以上 |
| 双灵根 | 稀有 | 快 | 合体以上 |
| 三灵根 | 较常见 | 中等 | 元婴/化神 |
| 四灵根 | 常见 | 慢 | 金丹 |
| 五灵根（废灵根） | 较常见 | 极慢 | 筑基 |
| 变异灵根 | 极稀有 | 极快 | 大乘以上 |

### 修炼优势

| 优势 | 说明 |
|------|------|
| 功法选择多 | 人族创造了最多种类的功法 |
| 丹药辅助强 | 人族炼丹术最为发达 |
| 阵法运用好 | 人族阵法体系最完善 |
| 法宝使用灵活 | 人族炼器术最为精妙 |
| 悟道速度快 | 悟性高，领悟法则更快 |

### 修炼劣势

| 劣势 | 说明 |
|------|------|
| 体质较弱 | 天生体质不如妖族 |
| 寿元较短 | 同境界寿元短于妖族 |
| 无天赋神通 | 需要通过功法获得能力 |
| 灵根限制 | 不是所有人都有灵根 |
