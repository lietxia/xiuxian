# 神族

> 神族是修仙界最尊贵的种族，居于神界，受天道庇护。本文详细介绍神族的起源、修炼体系与特点。

---

## 神族概述

神族是天地初开时由混沌之气孕育的种族，天生拥有神力，居于神界，地位超然。神族是修仙界最强的种族，实力远超其他种族。

### 神族特点

| 特点 | 说明 |
|------|------|
| 天生神力 | 出生即拥有强大力量 |
| 神格传承 | 拥有神格，可掌管天地法则 |
| 寿元无限 | 天生不死（除非被杀） |
| 地位超然 | 超脱各界纷争 |
| 实力最强 | 单体实力最强种族 |

## 神族修炼体系

| 境界 | 特征 | 能力 | 说明 |
|------|------|------|------|
| 下位神 | 刚觉醒神力 | 掌控一种法则 | 新生神族 |
| 中位神 | 神力增强 | 掌控两种法则 | 普通神族 |
| 上位神 | 神力大成 | 掌控多种法则 | 强大神族 |
| 主神 | 掌管一域 | 执掌天地一域 | 神界管理者 |
| 至高神 | 掌管天道一环 | 执掌天道之力 | 最顶级存在 |
| 天帝 | 执掌天界 | 统御万神 | 神族之主 |
