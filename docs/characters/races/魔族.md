# 魔族

> 魔族是修仙界最令人畏惧的种族之一，以魔气修炼，崇尚力量与征服。本文详细介绍魔族的起源、修炼体系与社会结构。

---

## 魔族概述

魔族是以魔气修炼的种族，天生拥有魔气亲和力，修炼速度快但易失控。魔族崇尚力量与征服，与正道长期对立。

### 魔族特点

| 特点 | 说明 |
|------|------|
| 魔气亲和 | 天生亲和魔气，修炼速度快 |
| 力量崇拜 | 崇尚绝对力量 |
| 肉身强悍 | 魔气淬体，肉身极强 |
| 战斗狂热 | 天生好战 |
| 性格霸道 | 霸道强势，不择手段 |

---

## 魔族修炼体系

| 境界 | 对应人族 | 特征 | 寿元 |
|------|---------|------|------|
| 炼魔期 | 练气期 | 吸收魔气，淬炼肉身 | 150-200年 |
| 魔光期 | 筑基期 | 魔气化光，实力初显 | 300-400年 |
| 魔丹期 | 金丹期 | 凝聚魔丹，实力大增 | 600-1000年 |
| 魔婴期 | 元婴期 | 魔丹化婴，神通初显 | 1500-3000年 |
| 混沌期 | 化神期 | 混沌之力，掌控魔域 | 5000-8000年 |
| 魔渡期 | 大乘期 | 魔道大成，可渡劫飞升 | 80000年以上 |

## 魔族与人族关系

魔族与人族长期对立，正邪之争从未停歇。但魔族并非纯粹的恶，也有自己的生存逻辑和追求。

| 关系 | 说明 |
|------|------|
| 正邪对立 | 正道与魔道长期对抗 |
| 偶尔合作 | 面对外敌（如妖族入侵）时偶尔联手 |
| 暗中交易 | 某些正道修士私下与魔族交易 |
| 互相渗透 | 双方都有间谍潜伏 |
