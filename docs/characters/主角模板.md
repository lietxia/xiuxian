# 主角模板

> 主角是修仙小说的核心，承载读者代入感与故事主线。本文提供主角类型分类、金手指设计、道心设定、成长曲线、常见剧情线、性格模板及写作参考。

---

## 主角类型分类

### 废柴逆袭流

| 特征 | 说明 |
|------|------|
| 初始状态 | 灵根低劣、修为垫底、被人轻视 |
| 核心看点 | 从底层一步步崛起，打脸看不起自己的人 |
| 代表设定 | 废灵根、无灵根、丹田破碎 |
| 情感基调 | 隐忍、厚积薄发、逆袭爽感 |
| 经典桥段 | 测灵根被嘲笑、宗门大比逆袭、打脸曾经欺辱自己的人 |

### 天才流

| 特征 | 说明 |
|------|------|
| 初始状态 | 天赋异禀、灵根上佳、自带光环 |
| 核心看点 | 一路高歌猛进，碾压同辈，越级挑战 |
| 代表设定 | 天灵根、先天道体、混沌体质 |
| 情感基调 | 自信、霸气、睥睨天下 |
| 经典桥段 | 测灵根震惊全场、越级斩杀强敌、天才之间的对决 |

### 重生流

| 特征 | 说明 |
|------|------|
| 初始状态 | 前世大能重生回低境界，保留前世记忆 |
| 核心看点 | 利用前世经验与见识，避开前世遗憾，快速崛起 |
| 代表设定 | 渡劫失败重生、被挚爱背叛陨落后重生 |
| 情感基调 | 复仇、弥补遗憾、运筹帷幄 |
| 经典桥段 | 提前获取前世机缘、识破前世仇人伪装、对前世敌人降维打击 |

### 穿越流

| 特征 | 说明 |
|------|------|
| 初始状态 | 现代人穿越到修仙世界，拥有现代知识 |
| 核心看点 | 用现代思维玩转修仙世界，降维打击 |
| 代表设定 | 科学家穿越、程序员穿越、普通人穿越 |
| 情感基调 | 新奇、探索、以智取胜 |
| 经典桥段 | 用科学解释修仙、现代知识炼丹炼器、对修仙世界的吐槽 |

### 系统流

| 特征 | 说明 |
|------|------|
| 初始状态 | 获得神秘系统辅助，通过完成任务获取奖励 |
| 核心看点 | 系统任务推动剧情，奖励加速成长 |
| 代表设定 | 签到系统、任务系统、抽奖系统、模拟器 |
| 情感基调 | 任务驱动、数值成长、稳定升级 |
| 经典桥段 | 每日签到获取功法、完成隐藏任务获得神器、系统商城兑换 |

## 金手指设计

### 外挂类型

| 类型 | 说明 | 常见形式 |
|------|------|---------|
| 功法类 | 获得逆天功法或传承 | 上古功法、混沌功法、禁忌秘术 |
| 法宝类 | 获得强大法宝 | 混沌至宝、先天灵宝、残破神器 |
| 体质类 | 觉醒特殊体质 | 先天道体、万毒不侵体、圣体 |
| 空间类 | 获得独立空间 | 仙府、洞天福地、小世界 |
| 灵宠类 | 获得强大灵宠 | 上古神兽后裔、变异灵兽 |
| 灵药类 | 获得逆天丹药或灵药 | 万年灵药、洗髓丹、破境丹 |
| 系统类 | 获得神秘系统辅助 | 各类系统（见系统流） |

### 获取方式

| 方式 | 说明 |
|------|------|
| 偶然获得 | 探险中意外发现，跳崖获得传承 |
| 家族传承 | 家族血脉中隐藏的秘密 |
| 师尊传授 | 师尊赐予的功法或法宝 |
| 拍卖获取 | 拍卖会上低价获得至宝 |
| 敌人身上 | 击杀敌人后获得其传承或法宝 |
| 穿越带来 | 穿越时携带的金手指 |

### 成长曲线

| 曲线类型 | 特点 | 适用主角 |
|---------|------|---------|
| 线性成长 | 稳步提升，循序渐进 | 努力型主角 |
| 阶梯式 | 关键节点大幅提升 | 机缘型主角 |
| 先弱后强 | 前期弱小，后期爆发 | 废柴逆袭型 |
| 持续碾压 | 一直很强，越来越强 | 天才流主角 |

## 道心设定

### 道心类型

| 道心 | 特点 | 代表 |
|------|------|------|
| 剑道之心 | 以剑入道，一往无前 | 剑修主角 |
| 杀伐之心 | 杀伐果断，不惧生死 | 魔修主角 |
| 守护之心 | 守护所爱，守护苍生 | 仁义型主角 |
| 自由之心 | 不受束缚，逍遥天地 | 散修主角 |
| 复仇之心 | 以仇恨为动力，誓要复仇 | 复仇型主角 |
| 长生之心 | 追求长生，探索天道 | 求道型主角 |
| 众生之心 | 心怀天下，为苍生请命 | 圣人型主角 |

### 道心破碎与重塑

| 阶段 | 说明 | 效果 |
|------|------|------|
| 道心动摇 | 遭遇重大打击，信念开始动摇 | 修为停滞，心境不稳 |
| 道心破碎 | 彻底失去信念，走火入魔边缘 | 实力暴跌，险些陨落 |
| 道心重塑 | 在绝望中重新找到方向 | 突破瓶颈，实力更进一步 |
| 道心升华 | 经历磨难后道心更加坚定 | 境界突破，实力大增 |

## 成长曲线设计

### 境界提升节奏

| 阶段 | 境界范围 | 时间跨度 | 节奏建议 |
|------|---------|---------|---------|
| 初期 | 练气→筑基 | 数年 | 快节奏，快速建立基础 |
| 中期 | 金丹→元婴 | 数十年 | 中等节奏，穿插机缘与战斗 |
| 后期 | 化神→大乘 | 数百年 | 慢节奏，注重心境描写 |
| 终章 | 渡劫→飞升 | 视剧情而定 | 高潮节奏，集中爆发 |

### 机缘安排

| 机缘类型 | 说明 | 安排建议 |
|---------|------|---------|
| 小机缘 | 低级法宝、灵药、功法 | 频繁出现，推动日常剧情 |
| 中机缘 | 中级法宝、秘境探索、传承 | 适量出现，作为剧情节点 |
| 大机缘 | 高级法宝、逆天功法、上古遗迹 | 稀少出现，作为重要转折点 |
| 终极机缘 | 飞升机缘、天道感悟 | 故事高潮，仅出现一次 |

## 常见剧情线

### 复仇线

| 阶段 | 内容 |
|------|------|
| 起因 | 家族被灭、师门遭难、挚爱被害 |
| 隐忍 | 韬光养晦，积蓄实力 |
| 复仇 | 逐个击败仇人，揭开真相 |
| 终结 | 击杀幕后黑手，大仇得报 |

### 寻亲线

| 阶段 | 内容 |
|------|------|
| 失散 | 幼年与亲人失散 |
| 寻找 | 修炼有成后开始寻找亲人 |
| 发现 | 发现身世之谜，揭开隐藏真相 |
| 团聚 | 与亲人团聚，或发现亲人已陨落 |

### 飞升线

| 阶段 | 内容 |
|------|------|
| 起步 | 凡人界开始修炼 |
| 崛起 | 修真界崭露头角 |
| 争霸 | 仙界与各方势力角逐 |
| 飞升 | 突破极限，飞升更高位面 |

### 争霸线

| 阶段 | 内容 |
|------|------|
| 初露锋芒 | 在宗门或区域中崭露头角 |
| 逐鹿天下 | 与各方势力角逐资源 |
| 君临天下 | 成为一方霸主 |
| 超越世俗 | 发现更高层次的争斗 |

## 性格模板

### 隐忍型

| 特征 | 说明 |
|------|------|
| 核心 | 韬光养晦，不鸣则已一鸣惊人 |
| 表现 | 低调行事，忍辱负重 |
| 优点 | 冷静理性，善于谋划 |
| 弱点 | 可能显得压抑，缺乏爽感 |
| 适用 | 废柴逆袭流、重生流 |

### 张扬型

| 特征 | 说明 |
|------|------|
| 核心 | 我辈修士，何惧一战 |
| 表现 | 高调行事，敢打敢拼 |
| 优点 | 热血激情，代入感强 |
| 弱点 | 可能显得鲁莽 |
| 适用 | 天才流、系统流 |

### 腹黑型

| 特征 | 说明 |
|------|------|
| 核心 | 笑里藏刀，绵里藏针 |
| 表面 | 温和有礼，人畜无害 |
| 实际 | 心思缜密，手段狠辣 |
| 优点 | 智商在线，计谋精彩 |
| 弱点 | 可能显得阴险 |
| 适用 | 重生流、穿越流 |

### 热血型

| 特征 | 说明 |
|------|------|
| 核心 | 一腔热血，勇往直前 |
| 表现 | 重情重义，路见不平拔刀相助 |
| 优点 | 感情真挚，容易共鸣 |
| 弱点 | 可能显得冲动 |
| 适用 | 废柴逆袭流、天才流 |

## 写作参考

### 主角塑造要点

| 要点 | 说明 |
|------|------|
| 代入感强 | 读者能与主角产生共鸣 |
| 成长明显 | 主角应有清晰的成长轨迹 |
| 性格一致 | 性格设定前后一致，不崩人设 |
| 有血有肉 | 有优点也有缺点，不是完人 |
| 动机明确 | 行事有明确的动机和目标 |
| 情感真实 | 情感描写真实可信 |

### 常见误区

| 误区 | 说明 | 改进建议 |
|------|------|---------|
| 无脑开挂 | 金手指太强，毫无挑战 | 给金手指设置限制和代价 |
| 圣母过度 | 对敌人心慈手软 | 保持合理的人性底线 |
| 无情无义 | 完全没有感情描写 | 适当加入情感线 |
| 智商下线 | 关键时刻犯蠢推进剧情 | 保持主角智商在线 |
| 人设崩塌 | 前后性格不一致 | 建立性格档案，保持一致 |
| 节奏失控 | 升级太快或太慢 | 合理规划升级节奏 |

## 主角模板示例

### 示例：废柴逆袭型主角

| 项目 | 内容 |
|------|------|
| 姓名 | 林尘 |
| 类型 | 废柴逆袭流 |
| 初始状态 | 废灵根，被宗门弟子欺辱 |
| 金手指 | 偶得上古混沌功法，可将废灵根转化为混沌灵根 |
| 道心 | 守护之心，誓要保护身边之人 |
| 性格 | 外表隐忍，内心坚韧，重情重义 |
| 成长线 | 废灵根→混沌灵根→越级挑战→一飞冲天 |
| 主要剧情 | 宗门大比逆袭、探索上古遗迹、为师门复仇 |
| 最终结局 | 飞升仙界，成为一代传奇 |
