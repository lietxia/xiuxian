# 人物模板

> 人物模板是创建修仙人物的标准化框架，包含基础信息、外貌、性格、资质、修炼经历、人际关系等完整设定。

---

## 模板说明

本模板适用于创建各类修仙人物，包括正道修士、魔修、散修、妖修等。可根据人物类型调整各项内容。

## 性格特征模板

### 性格关键词

选择3-5个关键词描述人物性格：

| 性格类型 | 可选关键词 |
|---------|-----------|
| 正面性格 | 坚毅、仁慈、稳重、果敢、聪慧、宽厚、正义 |
| 中性性格 | 冷静、谨慎、独立、务实、淡泊、随性 |
| 负面性格 | 阴沉、狡诈、残忍、贪婪、傲慢、偏执 |
| 特殊性格 | 亦正亦邪、多疑、寡言、矛盾、执着 |
