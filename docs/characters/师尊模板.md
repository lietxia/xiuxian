# 师尊模板

> 师尊是修仙小说中重要的引导型角色，为主角提供功法传承、人生指引和情感羁绊。本文提供师尊类型分类、背景设计、传承设定、师徒关系、剧情线及写作参考。

---

## 师尊类型分类

### 严厉型

| 特征 | 说明 |
|------|------|
| 性格 | 冷面严师，要求苛刻 |
| 教导方式 | 高压训练，动辄惩罚 |
| 内心 | 外冷内热，默默关心弟子 |
| 常见台词 | "不够，再来！""这点苦都吃不了，如何成大器？" |
| 经典形象 | 冷面剑仙、铁血长老 |

### 慈祥型

| 特征 | 说明 |
|------|------|
| 性格 | 和蔼可亲，如父如母 |
| 教导方式 | 循循善诱，耐心指导 |
| 内心 | 真心疼爱弟子，视如己出 |
| 常见台词 | "不急，慢慢来。""为师相信你。" |
| 经典形象 | 慈祥老者、温婉女修 |

### 神秘型

| 特征 | 说明 |
|------|------|
| 性格 | 深不可测，行事莫测 |
| 教导方式 | 点到为止，让弟子自行领悟 |
| 内心 | 有深意，考验弟子心性 |
| 常见台词 | "日后你自会明白。""天机不可泄露。" |
| 经典形象 | 隐世大能、神秘老者 |

### 陨落型

| 特征 | 说明 |
|------|------|
| 性格 | 为弟子不惜一切 |
| 教导方式 | 倾囊相授，甚至牺牲自我 |
| 内心 | 将所有希望寄托在弟子身上 |
| 常见台词 | "为师等不到那一天了。""你要替为师活下去。" |
| 经典形象 | 受伤濒死的强者、为弟子挡劫的大能 |

### 亦师亦友型

| 特征 | 说明 |
|------|------|
| 性格 | 平易近人，没有架子 |
| 教导方式 | 共同修炼，互相切磋 |
| 内心 | 将弟子视为平等的存在 |
| 常见台词 | "来，陪为师喝一杯。""你我切磋一番。" |
| 经典形象 | 年轻天才师尊、洒脱散修 |

## 师尊背景设计

### 隐世大能

| 项目 | 说明 |
|------|------|
| 身份 | 曾经的绝世强者，隐居避世 |
| 实力 | 深不可测，至少化神以上 |
| 隐世原因 | 厌倦纷争、受伤退隐、等待时机 |
| 收徒动机 | 看中弟子资质、有缘相遇、完成某种使命 |
| 常见设定 | 退隐的宗主、上古大能转世、受伤的仙人 |

### 受伤退隐

| 项目 | 说明 |
|------|------|
| 身份 | 曾经叱咤风云的强者 |
| 实力 | 受伤后实力大减，但仍远超弟子 |
| 受伤原因 | 大战受伤、渡劫失败、被人暗算 |
| 收徒动机 | 传承衣钵、寻找治愈之法、了却心愿 |
| 常见设定 | 被人暗算的宗主、渡劫失败的大能、中毒的高手 |

### 宗门长老

| 项目 | 说明 |
|------|------|
| 身份 | 宗门内德高望重的长老 |
| 实力 | 宗门顶尖，但非最强 |
| 收徒原因 | 看中弟子资质、完成任务、有缘 |
| 教导方式 | 正统教导，按部就班 |
| 常见设定 | 宗门护法长老、传功长老、执事长老 |

### 散修高人

| 项目 | 说明 |
|------|------|
| 身份 | 不属于任何势力的强者 |
| 实力 | 参差不齐，但往往有独到之处 |
| 收徒原因 | 有缘、看中弟子、排遣寂寞 |
| 教导方式 | 因材施教，不拘一格 |
| 常见设定 | 游历天下的散修、隐居山野的高人、独行侠客 |

## 传承设定

### 功法传承

| 传承类型 | 说明 | 常见形式 |
|---------|------|---------|
| 本门功法 | 宗门正统功法 | 按部就班传授，需考核 |
| 独门秘技 | 师尊独创或获得的秘技 | 亲传弟子才能学习 |
| 上古功法 | 师尊获得的上古传承 | 作为压箱底传授 |
| 禁忌功法 | 有副作用的强大功法 | 特殊情况下传授 |

### 法宝传承

| 法宝类型 | 说明 | 传承方式 |
|---------|------|---------|
| 本命法宝 | 师尊的本命法宝 | 临终传授或特殊时机赐予 |
| 师门传承 | 代代相传的法宝 | 弟子出师时传承 |
| 收集法宝 | 师尊收集的法宝 | 根据弟子表现赐予 |
| 残破法宝 | 受损的强大法宝 | 作为考验，修复后归弟子 |

### 秘境传承

| 秘境类型 | 说明 | 传承方式 |
|---------|------|---------|
| 师门秘境 | 师门代代相传的秘境 | 弟子达到要求后进入 |
| 师尊私有秘境 | 师尊发现的秘境 | 作为弟子机缘 |
| 上古遗迹 | 师尊发现的上古遗迹 | 带领弟子探索 |
| 试炼秘境 | 专门用于试炼弟子的秘境 | 定期让弟子进入历练 |

## 师徒关系设计

### 师徒情深

| 特征 | 说明 |
|------|------|
| 关系 | 亲如父子/母女 |
| 互动 | 师尊悉心教导，弟子尊敬孝顺 |
| 情感 | 深厚的师徒感情 |
| 经典桥段 | 师尊为弟子挡劫、弟子为师尊报仇 |
| 适用 | 慈祥型、亦师亦友型师尊 |

### 师徒反目

| 特征 | 说明 |
|------|------|
| 关系 | 从亲密到对立 |
| 原因 | 理念冲突、误会、利益冲突、被人挑拨 |
| 发展 | 渐行渐远→冲突爆发→决裂 |
| 经典桥段 | 弟子发现师尊的秘密、师徒理念之争 |
| 适用 | 严厉型、神秘型师尊 |

### 师徒恋

| 特征 | 说明 |
|------|------|
| 关系 | 从师徒发展为恋人 |
| 障碍 | 身份差异、世俗眼光、门规限制 |
| 发展 | 敬仰→爱慕→突破阻碍 |
| 经典桥段 | 朝夕相处生情、患难与共 |
| 适用 | 年轻型师尊、亦师亦友型 |

### 多弟子争宠

| 特征 | 说明 |
|------|------|
| 关系 | 多个弟子争夺师尊关注 |
| 冲突 | 嫉妒、竞争、暗中较劲 |
| 发展 | 表面和谐→暗流涌动→矛盾爆发 |
| 经典桥段 | 师尊偏心、资源分配不均、弟子内斗 |
| 适用 | 各类型师尊 |

## 师尊剧情线

### 保护弟子

| 阶段 | 内容 |
|------|------|
| 起因 | 弟子遭遇强敌或危机 |
| 过程 | 师尊出手相救，展现强大实力 |
| 结果 | 弟子获救，师尊可能受伤 |
| 意义 | 展现师徒情深，推动弟子成长 |

### 为弟子牺牲

| 阶段 | 内容 |
|------|------|
| 起因 | 弟子陷入绝境，无人能救 |
| 过程 | 师尊以命换命，或牺牲修为 |
| 结果 | 师尊陨落或重伤，弟子获救 |
| 意义 | 情感高潮，推动弟子复仇或成长 |

### 师尊有难

| 阶段 | 内容 |
|------|------|
| 起因 | 师尊被强敌围困、中毒、受伤 |
| 过程 | 弟子得知后前往救援 |
| 结果 | 弟子成功救出师尊 |
| 意义 | 展现弟子成长，反哺师恩 |

### 师尊黑化

| 阶段 | 内容 |
|------|------|
| 起因 | 师尊遭遇重大打击或发现真相 |
| 过程 | 师尊性情大变，走上邪路 |
| 结果 | 弟子与师尊对立，最终师徒对决 |
| 意义 | 制造情感冲突，增加故事深度 |

## 写作参考

### 师尊角色塑造要点

| 要点 | 说明 |
|------|------|
| 有深度 | 师尊应有自己的背景故事和动机 |
| 有魅力 | 师尊应有让读者喜欢的特质 |
| 有缺点 | 不要塑造完美师尊，适当加入缺点 |
| 有成长 | 师尊也应有成长或变化 |
| 有互动 | 师徒互动要真实自然 |
| 有张力 | 师徒关系应有戏剧张力 |

### 常见误区

| 误区 | 说明 | 改进建议 |
|------|------|---------|
| 工具人师尊 | 只在传授功法时出现 | 赋予师尊独立的故事线 |
| 实力崩塌 | 师尊实力忽强忽弱 | 保持实力设定一致 |
| 过度保护 | 师尊总是出手相救 | 让弟子独立面对困难 |
| 缺乏互动 | 师徒互动太少 | 增加日常互动描写 |
| 人设单薄 | 只有"严厉"或"慈祥"一个标签 | 给师尊多面性格 |
| 背景缺失 | 师尊没有背景故事 | 补充师尊的过去 |

## 师尊模板示例

### 示例：神秘型师尊

| 项目 | 内容 |
|------|------|
| 姓名 | 无崖子 |
| 类型 | 神秘型 |
| 身份 | 隐世散修，疑似上古大能转世 |
| 实力 | 深不可测，从不出手 |
| 性格 | 看似疯癫，实则洞察世事 |
| 教导方式 | 点到为止，让弟子自行领悟 |
| 背景 | 曾是绝世强者，因某种原因隐居 |
| 与主角关系 | 在主角落魄时收其为徒，传授独特功法 |
| 经典台词 | "天道无常，人心难测。""你自己悟吧。" |
| 最终结局 | 在主角危机时刻出手，揭示真实身份后离去 |
