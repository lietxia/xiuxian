# 散修人物

> 散修是没有宗门归属的独立修士，是修仙界中一个特殊而重要的群体。本文详细介绍散修的类型、生存方式与代表人物。

---

## 散修概述

散修，即没有宗门归属的独立修士。散修没有宗门庇护，修炼资源全靠自力更生，生存艰难但也有自由自在的优势。

### 散修特点

| 特点 | 说明 |
|------|------|
| 自由自在 | 不受宗门规矩束缚 |
| 资源匮乏 | 没有宗门资源支持 |
| 经验丰富 | 历经磨难，实战经验丰富 |
| 适应力强 | 适应各种环境 |
| 关系松散 | 没有固定的组织关系 |

---

| 类型 | 特点 | 修炼方式 | 经济来源 |
|------|------|---------|---------|
| 隐居散修 | 隐居山林，不问世事 | 自行修炼 | 采药、偶尔交易 |
| 坊市散修 | 在坊市活动 | 坊市交流修炼 | 交易、佣兵任务 |
| 游历散修 | 四处游历 | 边走边修 | 历练中获取资源 |

| 落魄散修 | 从宗门脱离 | 保留部分功法 | 打零工、佣兵 |
| 天才散修 | 天赋极高 | 自悟为主 | 奇遇获取 |
