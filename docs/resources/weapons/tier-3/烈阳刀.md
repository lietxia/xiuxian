# 烈阳刀

## 法宝概述

**法宝名称：** 烈阳刀  
**品级：** 三阶法宝  
**类型：** 刀类  

## 材料

烈阳刀以烈阳精铁为刃，取自火山深处的熔岩矿脉，需以地脉真火煅烧七七四十九日方能提炼出足够的精铁；以赤炎玉为柄，触手温热，能在战斗中为修士补充火属灵力；刀身上还镶嵌了三枚火鸦内丹，令刀身长年保持炽热。

## 功能/神通

**刀一·烈阳斩**：全力催动时，刀身会燃起熊熊烈焰，一刀劈出，仿佛一轮小太阳砸向敌人，威力惊人。

**刀二·火鸦飞舞**：刀身携带的火鸦内丹可分离出三道火焰虚影，虚影会自动追踪敌人进行攻击，持续约一刻钟。

**刀三·烈阳护体**：激活后，修士周身会环绕一圈炽热烈焰，令近身战斗的敌人不敢轻易靠近。

**刀四·破甲斩**：针对穿戴护甲的敌人，此招将火属灵力凝聚于刀尖一点，破坏力集中，能轻易破开各种防御类法宝。

**刀五·火中取栗**：此刀内置一道"避火诀"，持刀者在火焰环境中不仅不受伤害，反而能吸收火焰灵气补充自身。

## 历史/来历

一千五百年前，东海之滨曾有一位刀法大家"赤阳刀王"横空出世，以一套"烈阳刀法"杀得妖魔闻风丧胆。赤阳刀王性情刚烈，最看不惯恃强凌弱之辈，每逢不平之事必拔刀相助。

晚年时，赤阳刀王自觉时日无多，决定将毕生刀道精华封入此刀，流传后世。他以烈阳精铁为骨、火鸦内丹为魂、赤阳刀王的毕生刀意为核心，历时十年方才铸成此刀。为防刀落入邪修之手，赤阳刀王在刀中设下禁制——非心性纯正者持之，刀身会自行发出警示高温，拒绝配合。

此后，此刀在坊间流转百年，数次易手，每当持有者心生邪念时，刀身便滚烫难耐；而持刀者一旦改邪归正，刀便恢复温顺。此刀被视为"试心刀"，在散修中颇有名气。

## 持有者/下落

此刀现为散修联盟中一位德高望重的长老"烈火真人"所持有。烈火真人以此刀行侠仗义多年，从不滥杀无辜，深得同道敬重。
