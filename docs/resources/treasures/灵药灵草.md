## 灵药采摘与保存

### 采摘要点

| 要点 | 说明 |
|------|------|
| 时辰选择 | 特定时辰采摘药效最佳 |
| 保存灵性 | 采摘后需立即保存，灵性流失快 |
| 根系完整 | 尽量保持根系完整可移植 |
| 避免损伤 | 损伤会大幅降低药效 |
| 特殊手法 | 部分灵药需特殊手法采摘 |

### 保存方式

| 方式 | 适用灵药 | 保存时间 |
|------|----------|----------|
| 玉盒密封 | 普通灵药 | 数月至数年 |
| 灵玉封存 | 高阶灵药 | 数十年 |
| 空间法宝 | 稀有灵药 | 无限期 |
| 活体移植 | 可移植灵药 | 长期存活 |
