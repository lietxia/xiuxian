# 丹师等级

> 炼丹师是修仙界最受尊崇的职业之一，等级森严，每晋升一级都需极大的天赋与努力。

---

## 丹师等级划分

| 等级 | 称号 | 可炼丹药 | 成功率 | 火候要求 | 社会地位 |
|------|------|----------|--------|----------|----------|
| 一品 | 丹童 | 基础丹药（一阶） | 30%以下 | 文火 | 入门弟子 |
| 二品 | 丹徒 | 普通丹药（一至二阶） | 30%-50% | 文火/武火 | 外门弟子 |
| 三品 | 丹师 | 中级丹药（二至三阶） | 50%-70% | 武火 | 内门弟子 |
| 四品 | 大丹师 | 高级丹药（三至四阶） | 70%-85% | 丹火 | 长老级 |
| 五品 | 丹宗 | 顶级丹药（四至五阶） | 85%-95% | 丹火/天火 | 宗门客卿 |
| 六品 | 丹尊 | 传说丹药（五至六阶） | 95%以上 | 天火 | 一方势力的座上宾 |
| 七品 | 丹圣 | 仙丹（六至七阶） | 极高 | 天火/异火 | 传说级存在 |
| 八品 | 丹仙 | 神丹（七至八阶） | 几乎完美 | 异火 | 万年难遇 |
| 九品 | 丹神 | 禁忌丹药（九阶） | 完美 | 先天之火 | 传说中的传说 |

## 晋升条件

### 一品→二品 丹童→丹徒

| 条件 | 要求 |
|------|------|
| 修为 | 练气三层以上 |
| 灵根 | 具备火属性灵根 |
| 理论 | 掌握至少10种灵草药性 |
| 实践 | 成功炼制至少50炉一阶丹药 |
| 考核 | 通过丹盟入门考试 |

### 二品→三品 丹徒→丹师

| 条件 | 要求 |
|------|------|
| 修为 | 筑基期 |
| 火候 | 掌握武火控制 |
| 理论 | 掌握至少50种灵草药性 |
| 实践 | 成功炼制至少100炉二阶丹药 |
| 考核 | 炼制指定三阶丹药，品质达到中品以上 |

### 三品→四品 丹师→大丹师

| 条件 | 要求 |
|------|------|
| 修为 | 金丹期 |
| 火候 | 凝聚自身丹火 |
| 理论 | 自创或改良至少一条丹方 |
| 实践 | 成功炼制至少50炉三阶丹药 |
| 考核 | 炼制指定四阶丹药，品质达到上品以上 |

### 四品→五品 大丹师→丹宗

| 条件 | 要求 |
|------|------|
| 修为 | 元婴期 |
| 火候 | 丹火纯度达到极致 |
| 理论 | 自创至少三条以上丹方 |
| 实践 | 成功炼制至少30炉四阶丹药 |
| 考核 | 炼制指定五阶丹药，引发丹成异象 |

### 五品以上

五品以上丹师晋升条件极其苛刻，不仅需要极高的丹道天赋，还需要机缘际遇。六品丹尊以上更是万年难遇，每一位都是修仙界的传奇。

## 丹师天赋要求

### 灵根要求

| 灵根类型 | 适合炼丹程度 | 说明 |
|----------|-------------|------|
| 火木双灵根 | 最佳 | 火控温，木调药 |
| 火属性灵根 | 优秀 | 火候控制天然优势 |
| 木属性灵根 | 良好 | 药理理解能力强 |
| 单灵根 | 良好 | 灵力精纯，火候稳定 |
| 双灵根 | 一般 | 可成为丹师但成就有限 |
| 三灵根以上 | 较差 | 灵力驳杂，难以精进 |

### 精神力要求

| 精神力等级 | 对应丹师等级 | 说明 |
|------------|-------------|------|
| 普通 | 一至二品 | 可同时控制1-2种药材 |
| 强大 | 三至四品 | 可同时控制3-5种药材 |
| 超强 | 五至六品 | 可同时控制5-10种药材 |
| 恐怖 | 七品以上 | 精神力分化，一心多用 |
