## 丹药服用注意事项

| 注意事项 | 说明 |
|----------|------|
| 境界匹配 | 只能服用对应境界或以下品阶的丹药，越阶服用可能爆体 |
| 服用间隔 | 同类丹药需间隔一定时间服用，避免药力冲突 |
| 抗药性 | 长期服用同类丹药会降低药效 |
| 副作用 | 部分丹药有副作用，需配合其他丹药抵消 |
| 服用环境 | 突破类丹药需在安全环境中服用 |
| 配伍禁忌 | 某些丹药不可同时服用，会产生剧毒 |
