## 禁忌丹药的炼制条件

| 条件 | 说明 |
|------|------|
| 特殊丹方 | 禁忌丹方多为上古遗留，极少流传 |
| 祭炼之法 | 需以特殊仪式祭炼，非正常炼丹流程 |
| 活祭材料 | 部分禁忌丹药需活物祭祀 |
| 阴煞之地 | 需在阴气极重之地炼制 |
| 特定时辰 | 需在特定天象、时辰方可开炉 |
