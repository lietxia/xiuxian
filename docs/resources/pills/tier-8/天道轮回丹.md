# 天道轮回丹

## 丹药概述

**丹药名称：** 天道轮回丹  
**品级：** 八阶极品  
**类型：** 天道证道类丹药  

## 主治/效果

天道轮回丹位列八阶丹药之巅，是令无数合道境大能趋之若鹜的绝世神丹。其核心理念在于**令服用者窥见天道的终极奥义，在轮回与永恒之间找到那条通往至高境界的道路**。

此丹并非直接助人突破，而是给予服用者一次"窥见天道本原"的机会。天道轮回丹服用后，修士将进入一种前所未有的悟道状态——时间、空间、因果、命运、生死，一切法则在眼前交织成一幅完整的画卷。那是天道运转的本质，是宇宙诞生的源流，是万法归宗的终极奥义。

在此状态下，服用者将：

- **窥见天机**：亲眼见证天道的运转规律，获得远超常人的法则感悟
- **轮回感悟**：经历无数次轮回转世，在轮回中积累无上智慧
- **因果通明**：彻底看透因果法则的前因后果，再无迷惑
- **天道共鸣**：与天道产生短暂共鸣，感受天道意志的脉动
- **证道契机**：窥见天道的刹那，若能把握那一线灵感，可能一举突破到传说中从未有人达到过的境界

服用此丹，是迈向天道的第一步，也是最后一步。

## 炼制材料

**主药：** 天道本源碎片一枚（天地诞生之初，天道第一次显化时遗留的本源碎片）  
**辅药：** 轮回法则结晶、因果法则结晶、命运法则结晶各一份  
**药引：** 天道意志的一缕投影（需以无上秘法，在天道显化之时收取其投影）  

炼制需在天道显化的特殊时刻进行，整个炼制过程可能跨越数个纪元。

## 炼制难度

八阶极品，材料几乎不可能收集齐全。天道本源碎片自天地诞生以来只出现一次，且被谁获得已不可考证。整个炼制过程可能需要数千年乃至数万年。成功率极低，且无人知晓具体概率。

## 副作用/注意事项

1. **服用此丹后的状态无法预测**——每个人的悟道体验都不相同，有人收获颇丰，有人一无所获。
2. 窥见天道后，服用者可能会陷入长时间的悟道状态，可能长达数百年乃至数千年。
3. 服用后若无法承受天道的威压，神魂可能崩溃。
4. 天道意志并非善意——窥见天道本身可能招来天道的"注视"，日后可能面临天道的考验。
5. 服用此丹后，服用者可能会对世间一切失去兴趣，追求更高的超脱。
6. 每名修士仅有一次服用机会，第二次服用将毫无效果。
