# 特殊类丹药

特殊类丹药不直接用于战斗或修炼突破，却在修仙生活中扮演着不可或缺的角色。从日常辟谷到改变灵根属性，从驻颜不老到延寿续命，特殊类丹药覆盖了修士生活的方方面面。

## 一、避毒丹

避毒丹与解毒丹不同，它并非清除已有的毒素，而是在服用后的一段时间内使修士对毒物免疫。

| 属性 | 详情 |
|------|------|
| **品阶** | 一至五阶（对应免疫同阶以下毒素） |
| **有效时间** | 一阶：一个时辰；三阶：半日；五阶：三日 |
| **免疫机制** | 在体表与经脉外层形成药力屏障，毒素接触即被分解 |
| **限制** | 不可与解毒丹同时服用（药力冲突）；对已中毒者无效 |
| **常见用途** | 探索毒沼秘境、闯荡瘴气之地、对抗毒修 |
| **三阶避毒丹材料** | 百毒草×3、清灵花×2、三阶蛇胆×1、灵泉液 |

## 二、辟谷丹

修士闭关修炼时，肉身仍需基本的水谷精气维持。辟谷丹可替代食物与饮水，让修士安心闭关。

| 属性 | 详情 |
|------|------|
| **品阶** | 一阶（无境界限制） |
| **有效时间** | 一枚辟谷七日 |
| **药理** | 丹中蕴含浓缩的天地精气，可直接被肉身吸收，替代食物消化产生的能量 |
| **限制** | 不可连续服用超过十枚（约七十日），之后需正常进食恢复脏腑功能 |
| **副作用** | 长期依赖辟谷丹会导致消化功能退化 |
| **材料成本** | 每枚5灵石，是最廉价的丹药之一 |
| **批量需求** | 宗门闭关弟子每人每月需备3-4枚 |

## 三、易容丹

易容丹可改变修士的外貌，包括五官、身形、声音，甚至年龄表象。

| 属性 | 详情 |
|------|------|
| **品阶** | 二阶（基础易容）、四阶（高阶易容） |
| **有效时间** | 二阶：三日；四阶：一月 |
| **效果范围** | 二阶：改变五官与体型，变化幅度30%以内；四阶：完全改变外貌，可模拟他人 |
| **识别方式** | 高两个品阶以上的修士，以神识探查可识破 |
| **限制** | 无法改变灵根属性与修为气息；神识强度不变 |
| **常见用途** | 隐藏身份、潜入敌方势力、躲避追杀 |
| **二阶材料** | 幻容草×2、灵蝶粉×1、玉露×3 |

## 四、隐息丹

隐息丹可隐藏修士的修为气息，使他人无法以神识或探查术感知其实力。

| 属性 | 详情 |
|------|------|
| **品阶** | 二至六阶（品阶越高，隐藏越深） |
| **有效时间** | 二阶：两个时辰；四阶：一日；六阶：一月 |
| **隐藏效果** | 服用后修为显示为比实际低一到三个大境界，或完全隐匿 |
| **识别方式** | 高三个品阶以上的修士可感知异常（但无法确定真实修为） |
| **限制** | 战斗时药效大幅衰减，全力出手后气息暴露 |
| **四阶材料** | 隐灵草×3、虚空石碎片×1、四阶妖兽蜕皮×1 |

## 五、易灵丹

易灵丹是极为罕见的特殊丹药，能改变修士的灵根属性。例如将火灵根转为水灵根，或将木灵根转为金灵根。

| 属性 | 详情 |
|------|------|
| **品阶** | 五阶（单属性转换）、七阶（多属性转换） |
| **转换成功率** | 五阶：60%；七阶：40% |
| **转换范围** | 可将一种灵根属性完全转换为另一种；七阶可同时转换两种 |
| **限制** | 每人一生只能服用一次；转换后原属性亲和力永久消失 |
| **副作用** | 转换后三年内，新灵根属性的修炼速度降低30%，需重新适应 |
| **失败后果** | 灵根受损，品质下降一到两级，但属性不变 |
| **五阶材料** | 目标属性本源精华×1、五行调和草×5、五阶灵脉结晶×3 |
| **市场价** | 500万-1500万上品灵石（极度稀缺） |

## 六、驻颜丹

驻颜丹可使修士的容貌定格在服用时的状态，此后岁月不再在面容上留下痕迹。

| 属性 | 详情 |
|------|------|
| **品阶** | 三阶 |
| **有效时间** | 永久（单次服用） |
| **效果** | 容貌不再衰老，但身体机能仍按实际年龄衰退 |
| **限制** | 每人仅第一枚有效；无法逆转已有的衰老痕迹（需在年轻时服用） |
| **材料成本** | 5万灵石 |
| **附加效果** | 皮肤光泽度提升，气质微增 |
| **常见需求者** | 女性修士（需求量极大，是三阶丹药中最畅销的品类之一） |
| **材料** | 驻颜花×1、千露草×3、灵泉水×10、三阶妖兽血×5 |

驻颜丹虽品阶不高，但因需求量大、材料相对易得，是许多三阶炼丹师的主要收入来源。

## 七、寿元丹

寿元丹是最逆天的特殊丹药之一，能直接增加修士的寿元。天地规则下，每个境界有固定的寿元上限，寿元丹可突破这一限制。

| 品阶 | 名称 | 增加寿元 | 服用上限 | 材料成本 |
|------|------|----------|----------|----------|
| 三阶 | 延寿丹 | +50年 | 3枚 | 3万灵石 |
| 五阶 | 续命丹 | +200年 | 2枚 | 300万灵石 |
| 七阶 | 长生丹 | +500年 | 1枚 | 3000万灵石 |
| 九阶 | 万寿丹 | +1000年 | 1枚 | 无价 |

寿元丹的效果不可叠加突破境界寿元上限的50%。例如筑基期修士寿元上限200年，服用寿元丹最多增加到300年。

**天道反噬**：寿元丹本质上是在"偷取"天地寿元。每次服用，天道会降下因果纠缠，表现为日后渡天劫时威力增加10%。服用越多，因果越重。

**特殊禁忌**：寿元丹不可用于已陨落之人（寿元已尽者不受丹药影响）。若修士寿元将尽时强行服用续命丹，可延长最后一口气的时间，但仅能维持清醒交谈，无法战斗或修炼。

## 八、特殊类丹药的获取难度

特殊类丹药中，辟谷丹和驻颜丹相对常见，市场流通量大。隐息丹和避毒丹需求稳定，各大坊市均有出售。而易灵丹和寿元丹则属于战略资源，极少在公开市场流通，多通过拍卖会、私下交易或宗门内部渠道获得。
