# 突破类丹药

突破类丹药是丹药体系中最珍贵的品类。修士每逢境界瓶颈，自行突破的成功率往往不足三成，而服用对症的突破丹药可将成功率提升至六到九成。正因如此，一枚高阶突破丹足以引发宗门大战。

## 一、各境界突破丹药详解

### 筑基丹（二阶）

筑基丹是修仙界第一道分水岭用药。炼气圆满修士服用后，可将筑基成功率从30%提升至65%。

| 属性 | 详情 |
|------|------|
| **核心材料** | 百年灵芝×1、天麻根×3、聚灵草×5、二阶妖丹×1 |
| **辅助材料** | 清泉水、灵蜂蜜、玉髓粉 |
| **炼制难度** | ★★☆☆☆（需二阶炼丹师） |
| **基础成丹率** | 45%（二阶炼丹师） |
| **市场价** | 3000-5000下品灵石 |
| **副作用** | 丹毒残留约2%，需半年自然代谢 |
| **替代方案** | 灵泉灌体（成功率+20%，需宗门灵泉）；自行苦修（成功率30%，无副作用） |

### 凝丹丹（三阶）

助炼气圆满或筑基后期修士凝结金丹。服用后结丹成功率从15%提升至50%。

| 属性 | 详情 |
|------|------|
| **核心材料** | 三阶妖丹×2、金阳花×1、玄冰草×3、五行精华各一份 |
| **辅助材料** | 千年灵液、兽骨粉、灵石粉末 |
| **炼制难度** | ★★★☆☆（需三阶炼丹师） |
| **基础成丹率** | 35% |
| **市场价** | 2万-5万中品灵石 |
| **副作用** | 丹毒残留约3%，金丹可能出现杂质纹路 |
| **替代方案** | 天地灵脉闭关（成功率+10%，需三年）；双修辅助（成功率+15%，限特定体质） |

### 凝婴丹（四阶）

金丹修士凝结元婴的关键助力。元婴期是修仙路上第一道天堑，无丹药辅助，成功率仅8%。

| 属性 | 详情 |
|------|------|
| **核心材料** | 四阶妖丹×1、婴变果×1、九转灵芝×2、天地灵气结晶×3 |
| **辅助材料** | 龙涎香、凤血石粉、五行精金 |
| **炼制难度** | ★★★★☆（需四阶炼丹师，且需元婴修士护法） |
| **基础成丹率** | 25% |
| **市场价** | 50万-100万上品灵石（有价无市） |
| **副作用** | 丹毒残留约5%，元婴初期不稳，需十年巩固 |
| **替代方案** | 秘境机缘（如灵泉、天材地宝，成功率看运气）；转修散仙之路（放弃肉身，成功率40%） |

### 化神丹（五阶）

元婴修士冲击化神境的必备之物。化神境需元婴与天地法则初步融合，化神丹能助修士感悟法则之力。

| 属性 | 详情 |
|------|------|
| **核心材料** | 五阶妖丹×1、化神草×1（万年生长）、天火精华、五行本源之气 |
| **辅助材料** | 虚空石粉、雷击木、星辰砂 |
| **炼制难度** | ★★★★★（需五阶炼丹大师，耗时七七四十九天） |
| **基础成丹率** | 15% |
| **市场价** | 500万-2000万上品灵石（基本不出售） |
| **副作用** | 丹毒残留约8%，化神初期法则感悟不纯 |
| **替代方案** | 天劫感悟（在渡劫中领悟，成功率5%，极度危险）；大能灌顶（需化神以上修士牺牲修为） |

### 炼虚丹（六阶）、合体丹（七阶）、大乘丹（八阶）、渡劫丹（九阶）

六阶以上突破丹药每一种都是战略资源，材料中必须含有同阶天地灵物或大妖内丹。炼虚丹需耗时一年炼制，合体丹需三年，大乘丹需十年，渡劫丹的炼制周期可达三十年以上。市场上几乎不见流通，均由顶级宗门自行炼制、内部消耗。

## 三、突破丹药的特殊规则

**不可叠加**：同一境界突破只能服用一枚突破丹药，第二枚无效。但可在突破前服用辅助类丹药（如凝神丹提升神识、聚灵丹补充灵力），与突破丹药效果叠加。

**天劫影响**：服用突破丹药突破的修士，渡天劫时雷劫威力会增强10%-30%。天道平衡，借丹药之力越强，天劫反噬越大。

**根基折扣**：完全靠丹药突破的修士，该境界的灵力上限会降低5%-15%。极品丹药影响最小（5%），下品丹药影响最大（15%）。

## 四、炼丹师等级与突破丹药

炼制突破丹药的炼丹师，其等级至少需与丹药品阶齐平。低一阶的炼丹师强行炼制，成丹率不足5%，且极易炸炉。

顶级宗门通常培养多位高阶炼丹师，形成梯队。如大宗门会配备三到五位四阶炼丹师专门炼制凝婴丹，以保证宗门弟子的元婴突破需求。
