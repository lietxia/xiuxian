# 聚灵丹

## 丹药概述

**丹药名称：** 聚灵丹  
**品级：** 三阶上品  
**类型：** 加速修炼类丹药  

## 主治/效果

聚灵丹为筑基期修士梦寐以求的修炼加速丹药，其核心功效在于大幅提升灵力吸纳效率。正常修炼时，修士需吐纳天地灵气纳入丹田，而聚灵丹服用后，可在丹田内形成一层"灵旋"，犹如漩涡一般主动吸附周遭灵气，使吸纳速度提升两到三倍。

此丹特别适合以下场景：**闭关冲刺突破**——在冲击筑基中期、后期等关键节点时，灵气需求量剧增，单纯靠自然吸纳往往力有不逮；**灵气稀薄之地修炼**——如荒漠、深海、秘境深处等灵气匮乏之处；**争分夺秒的比斗间隙**——短暂服药即可快速恢复灵力。

服用聚灵丹后，修士周身会浮现淡青色灵光，持续约一个时辰。在此期间打坐修炼，相当于平时三日的功效。连续服用三日配合闭关，效果尤为显著。

## 炼制材料

**主药：** 灵芝孢子三钱（须取百年灵芝所产，活性充足者）  
**辅药：** 聚灵石粉一钱、五行土（取灵矿脉深处之土）二钱、青木精华一滴  
**药引：** 聚灵草汁液三滴（此草极难培育，为丹药命名之源）  

此丹需以五行相生之理炼制，土生金、金生水、水生木，三种辅药须按特定顺序依次投入。

## 炼制难度

三阶上品丹药，对炼丹师五行理论基础要求较高。材料搭配顺序、火候五行对应皆有严格讲究。成功率约为三成，资深炼丹师（十年以上）可达六成。由于炼制难度较高，市面流通量较少。

## 副作用/注意事项

1. **药效期间切忌中断修炼**——灵旋一旦启动便不可强行中止，否则灵气反噬会导致经脉剧痛。
2. 不可连续服用超过三日，否则体内灵气淤积过重，轻则经脉肿胀，重则走火入魔。
3. 体质虚弱者慎用，气血不足者可能承受不住灵旋高速运转带来的负担。
4. 不可在渡劫前后服用，雷劫余威未消时服用此丹风险极大。
5. 服用后应多饮温水，帮助灵气疏导。
