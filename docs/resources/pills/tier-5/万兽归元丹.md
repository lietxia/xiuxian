# 万兽归元丹

## 丹药概述

**丹药名称：** 万兽归元丹  
**品级：** 五阶上品  
**类型：** 御兽增益类丹药  

## 主治/效果

万兽归元丹乃驭兽师一脉的无上珍宝，其核心功效在于**令修士与灵兽之间的灵魂联系发生质的飞跃，从主从关系进化为"心意相通"**。

服用此丹后，修士眉心会生出一枚"万兽印记"，从此与契约灵兽之间建立了心灵层面的直接连接。无论相隔多远，修士都能感知灵兽的位置、状态、情绪甚至是模糊的想法；而灵兽也能在更深层次上理解主人的意图，无需言语命令即可默契配合。

此丹最令人称道的效果，是它能**大幅提升灵兽的灵智与成长潜力**。许多野生灵兽因灵智未开，修炼到一定程度后便再难寸进。服用万兽归元丹后，灵兽的灵智仿佛被开启了一扇窗户，能理解更复杂的道理、学习更精妙的法术、突破原本的血脉桎梏。

对于拥有多只灵兽的驭兽师而言，此丹更是必备之物——丹中的"万兽归元"之意，能令多只灵兽之间也建立心灵链接，从此协同作战如同一个整体，进退有度、配合天衣无缝。

## 炼制材料

**主药：** 万兽精血一份（须取自十种不同妖兽，各取其精血三滴，融合而成）  
**辅药：** 通灵神木屑二钱、兽魂晶一枚（取自即将突破的妖兽，死后魂魄凝聚而成）、百草甘露半盏  
**药引：** 驭兽师与灵兽各一滴精血（须在服药时同时滴入，以血为桥连接彼此灵魂）  

炼丹时需将十种妖兽精血依次注入，每次间隔一个时辰，以特定顺序引导药性归元。

## 炼制难度

五阶上品，炼制难度极高。辅药中的通灵神木屑极难获取，且主药中十种妖兽精血的融合顺序不得有丝毫差错。成功率约一成半，每炉得丹一至二枚。

## 副作用/注意事项

1. **服用前须与灵兽完成契约**，否则印记无法落地，药力消散。
2. 服用后三日内，修士与灵兽之间会出现短暂的心灵过载期，双方都可能出现头痛、幻觉等症状，属正常现象。
3. 印记一旦生成便不可消除，修士若欲更换契约灵兽，将付出巨大代价。
4. 若灵兽死亡，印记会反噬修士神识，造成严重创伤。
5. 服用后修士身上会残留淡淡的妖兽气息，低阶妖兽可能将其视为同类或敌人。
6. 不可在灵兽怀孕或受伤期间服用。
