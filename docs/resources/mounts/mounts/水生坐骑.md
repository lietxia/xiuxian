# 水生坐骑

水生坐骑是在水中使用的坐骑，适合水域探索。

---

## 基本信息

**名称**：水生坐骑
**类型**：水族坐骑
**等级**：三至六阶
**获得方式**：水域捕捉、购买、龙宫传承
**培养难度**：★★★☆☆

## 特殊环境

水生坐骑需要特殊的水域环境：
- 深海环境：龙龟、巨鲸、海龟
- 浅海环境：鲨鱼、海蛇、巨蟹
- 两栖环境：飞鱼、水马、海马
