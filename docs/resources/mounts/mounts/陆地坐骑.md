# 陆地坐骑

陆地坐骑是修士在陆地上最常用的代步工具。

---

## 基本信息

**名称**：陆地坐骑
**类型**：灵兽坐骑
**等级**：一至五阶
**获得方式**：野外捕捉、购买、任务奖励
**培养难度**：★★☆☆☆
