# 炼器装备体系

炼器是修仙界四大技艺（炼丹、炼器、布阵、画符）之一，通过将各种灵材进行炼制，化作武器、防具、法宝、饰品等装备。炼器师以烈火锻造，以灵气淬炼，将天地精华凝聚于方寸之间，赋予凡物以通天彻地之能。

## 品阶总览

| 品阶 | 对应修为 | 装备品质定位 |
|------|----------|-------------|
| 一品 | 练气期 | 凡品：凡铁炼化，初具灵性 |
| 二品 | 筑基期 | 下品：灵材锻造，锋芒初现 |
| 三品 | 金丹期 | 中品：灵韵内敛，威能渐显 |
| 四品 | 元婴期 | 上品：灵通自成，灵性十足 |
| 五品 | 化神期 | 极品：灵宝初成，可改规则 |
| 六品 | 炼虚期 | 珍品：法则入器，威力无匹 |
| 七品 | 合体期 | 圣品：道器之胚，几近通灵 |
| 八品 | 大乘期 | 神品：通灵彻变，神威莫测 |
| 九品 | 渡劫期 | 至品：造化神器，毁天灭地 |

> **起名原则**：品阶越高，名字越长、越有气势。一品名称简短质朴（如"流云剑"），九品名称气势恢宏（如"永恒万劫混沌鸿蒙太古开天大道归一先天至宝神剑"）。
