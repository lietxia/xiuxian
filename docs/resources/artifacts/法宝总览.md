# 法宝总览

> 法宝是修仙者最重要的战斗装备，从低阶法器到先天灵宝，每一件法宝都承载着修士的心血与期望。

---

## 法宝概述

法宝是修仙者以特殊材料炼制、铭刻禁制、蕴养灵力而成的器具。品级越高的法宝，威能越强，对使用者的要求也越高。

### 法宝核心要素

| 要素 | 说明 | 重要性 |
|------|------|--------|
| 材质 | 法宝的物理基础，决定上限 | 基础 |
| 禁制 | 铭刻在法宝上的法阵，赋予特殊能力 | 核心 |
| 灵力 | 法宝运转的能量来源 | 关键 |
| 器灵 | 高阶法宝孕育的灵智，大幅提升威力 | 灵魂 |

### 法宝分类

| 分类 | 说明 | 代表 |
|------|------|------|
| 攻击型 | 以攻击为主 | 飞剑、宝刀、神枪 |
| 防御型 | 以防御为主 | 护盾、宝甲、灵罩 |
| 辅助型 | 特殊功能 | 储物袋、传音符、飞行法宝 |
| 特殊型 | 蕴含法则之力 | 先天灵宝、道器 |

---

## 法宝获取方式

| 方式 | 说明 | 难度 |
|------|------|------|
| 购买 | 在坊市、拍卖会购买 | 取决于财力 |
| 炼制 | 自行或委托炼器师炼制 | 需要材料和技术 |
| 遗迹探索 | 在上古遗迹中发现 | 危险但回报大 |
| 击杀夺取 | 从敌人手中夺取 | 有因果风险 |
| 天地孕育 | 天然生成的灵宝 | 极为稀有 |
| 传承获得 | 师门传承或前辈馈赠 | 需要机缘 |

---

## 法宝与使用者

### 认主方式

| 认主方式 | 适用法宝 | 程度 | 说明 |
|----------|----------|------|------|
| 灵力灌注 | 普通法器 | 低 | 注入灵力即可使用 |
| 滴血认主 | 灵器以上 | 中 | 以精血建立联系 |
| 神魂烙印 | 法宝以上 | 高 | 以神识刻入灵魂印记 |
| 契约绑定 | 灵宝以上 | 极高 | 需双方法宝与修士同意 |
| 本命融合 | 本命法宝 | 最高 | 法宝与性命相连 |

### 使用限制

| 限制类型 | 说明 |
|----------|------|
| 修为限制 | 法宝品阶越高，需要越高的修为才能催动 |
| 灵力消耗 | 催动法宝需要消耗灵力，品阶越高消耗越大 |
| 越级反噬 | 强行催动超出自身境界的法宝，会遭到反噬 |
| 契合度 | 与法宝契合度越高，发挥的威力越大 |

---

## 法宝成长

| 成长方式 | 说明 | 适用法宝 |
|----------|------|----------|
| 温养 | 长期以灵力蕴养，缓慢提升 | 所有法宝 |
| 祭炼 | 以特殊方法祭炼，主动提升 | 法宝以上 |
| 天劫淬炼 | 经历天劫洗礼，品质飞跃 | 灵宝以上 |
| 吞噬融合 | 吞噬其他法宝或材料进阶 | 特殊法宝 |
| 法则注入 | 注入天地法则，本质提升 | 顶级法宝 |
