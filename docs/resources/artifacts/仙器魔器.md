# 仙器魔器

> 超越凡俗的至高法宝——仙器为仙人所铸，魔器为魔道所炼，两者截然不同却各有千秋。

---

## 仙器体系

### 仙器概述

仙器是以仙界材料、仙人手法炼制的法宝，核心为仙家禁制。禁制层数越多、品级越高、材质越好，仙器品阶越高。

### 仙器等级

| 等级 | 品质划分 | 灵智程度 | 禁制层数 |
|------|----------|----------|----------|
| 下品仙器 | 三等 | 初具灵智 | 13-24层 |
| 中品仙器 | 二等 | 灵智如幼童 | 25-36层 |
| 上品仙器 | 一等 | 灵智如少年 | 37-48层 |
| 绝品仙器 | 特等 | 灵智高超 | 49-60层 |
| 王品仙器 | 极品 | 可化形显化 | 61-72层 |
| 圣品仙器 | 至尊 | 灵智完满 | 73层以上 |

### 仙器特性

| 特性 | 说明 |
|------|------|
| 仙力驱动 | 需要仙力催动，凡间修士无法使用 |
| 自主修炼 | 仙器可自行修炼提升 |
| 化形显化 | 高阶仙器可化为童男童女或灵兽形态 |
| 法则之力 | 蕴含一缕天地法则之力 |
| 不灭之身 | 除非被更高阶法宝击碎，否则不会损毁 |

### 著名仙器

| 仙器名称 | 品阶 | 类型 | 特点 |
|----------|------|------|------|
| 诛仙四剑 | 先天至宝 | 攻击 | 杀伐第一 |
| 七宝妙树 | 先天灵宝 | 攻击 | 刷尽万物 |
| 天地玄黄尺 | 后天功德至宝 | 攻击 | 杀人不沾因果 |
| 乾坤鼎 | 先天至宝 | 辅助 | 炼化万物 |
| 太极图 | 先天至宝 | 防御 | 镇压地水火风 |

## 魔器体系

### 魔器概述

魔器是以魔气、煞气、怨气等负面能量炼制的法宝，威力霸道但副作用明显。

### 魔器等级

| 等级 | 品质划分 | 煞气程度 | 驱动方式 |
|------|----------|----------|----------|
| 下品魔器 | 三等 | 初级煞气 | 魔气催动 |
| 中品魔器 | 二等 | 浓郁煞气 | 精血催动 |
| 上品魔器 | 一等 | 恐怖煞气 | 灵魂催动 |
| 绝品魔器 | 特等 | 化形煞气 | 魔道本源 |
| 魔尊器 | 极品 | 天魔之气 | 魔尊级法力 |
| 魔神器 | 至尊 | 混沌魔气 | 魔神之力 |

### 魔器特性

| 特性 | 说明 |
|------|------|
| 嗜血吞噬 | 可吞噬血液增强威力 |
| 反噬主人 | 使用不当会遭到魔器反噬 |
| 腐蚀灵力 | 魔气可腐蚀正道修士灵力 |
| 污染心神 | 长期使用容易入魔 |
| 威力递增 | 杀戮越多威力越强 |

### 著名魔器

| 魔器名称 | 品阶 | 类型 | 特点 |
|----------|------|------|------|
| 六魂幡 | 顶级魔器 | 攻击 | 伤人神魂 |
| 混元锤 | 上品魔器 | 攻击 | 蛮力无双 |
| 摄魂铃 | 中品魔器 | 控制 | 迷乱心智 |
| 血河图 | 绝品魔器 | 阵法 | 困杀利器 |
| 万魔幡 | 魔尊器 | 召唤 | 召唤万魔 |

## 仙魔对比

| 对比项 | 仙器 | 魔器 |
|--------|------|------|
| 能量来源 | 仙力/灵力 | 魔气/煞气 |
| 炼制材料 | 仙界材料 | 阴煞材料 |
| 副作用 | 几乎无 | 反噬主人 |
| 成长方式 | 自行修炼 | 吞噬杀戮 |
| 正面对决 | 稳定持久 | 爆发凶猛 |
| 获取难度 | 极难 | 需入魔道 |
