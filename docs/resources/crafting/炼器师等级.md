# 炼器师等级

> 炼器师是修仙界与炼丹师齐名的尊贵职业，以锻造法宝为业，等级森严。

---

## 炼器师等级划分

| 等级 | 称号 | 可炼法宝 | 真火要求 | 社会地位 |
|------|------|----------|----------|----------|
| 一阶 | 炼器学徒 | 凡器 | 无需真火 | 入门 |
| 二阶 | 初阶炼器师 | 低阶法器 | 初步掌握火控 | 普通工匠 |
| 三阶 | 中阶炼器师 | 中阶法器/灵器 | 凝练心火 | 受人尊重 |
| 四阶 | 高阶炼器师 | 宝器 | 金丹真火 | 宗门核心 |
| 五阶 | 宗师级炼器师 | 法宝 | 元婴真火 | 一方势力座上宾 |
| 六阶 | 大宗师级炼器师 | 灵宝 | 化神真火 | 修仙界传奇 |
| 七阶 | 器道圣者 | 通天灵宝 | 法则之力 | 万年难遇 |

---

## 晋升条件

### 一阶→二阶 学徒→初阶炼器师

| 条件 | 要求 |
|------|------|
| 修为 | 筑基期 |
| 理论 | 掌握基础灵材知识 |
| 实践 | 成功炼制20件以上凡器 |
| 考核 | 现场炼制指定低阶法器 |

### 二阶→三阶 初阶→中阶炼器师

| 条件 | 要求 |
|------|------|
| 修为 | 筑基后期或金丹期 |
| 火候 | 凝练一缕心火 |
| 理论 | 掌握灵纹铭刻技术 |
| 实践 | 成功炼制50件以上法器 |
| 考核 | 炼制指定灵器，品质中品以上 |

### 三阶→四阶 中阶→高阶炼器师

| 条件 | 要求 |
|------|------|
| 修为 | 金丹期 |
| 火候 | 金丹真火 |
| 理论 | 自创或改良至少一条灵纹 |
| 实践 | 成功炼制宝器级法宝 |
| 考核 | 炼制指定宝器，品质上品以上 |

### 四阶→五阶 高阶→宗师级

| 条件 | 要求 |
|------|------|
| 修为 | 元婴期 |
| 火候 | 元婴真火 |
| 理论 | 自创至少三条灵纹或一套禁制 |
| 实践 | 成功炼制法宝级法宝 |
| 特殊 | 法宝需产生灵性 |

---

## 炼器师天赋要求

| 天赋 | 说明 | 重要程度 |
|------|------|----------|
| 火属性灵根 | 火候控制的基础 | 必须 |
| 精神力 | 铭刻禁制需要强大的精神力 | 极高 |
| 灵敏感知 | 感知材料特性和火候变化 | 高 |
| 耐心 | 炼器耗时漫长，需极大耐心 | 高 |
| 创造力 | 自创灵纹和禁制 | 决定上限 |

---

| 二阶 | 10-100灵石 | 1000-5000灵石 | 略低于同阶 |

| 六阶 | 无法估量 | 无价 | 传奇级 |
