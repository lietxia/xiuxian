# 材料分类

> 炼器材料是法宝的基础，材料品质直接决定法宝的上限。了解材料分类是炼器师的基本功。

---

## 材料总览

### 按来源分类

| 来源 | 说明 | 品质范围 | 稀有度 |
|------|------|----------|--------|
| 矿物 | 天然矿石提炼 | 低至顶级 | 取决于矿脉 |
| 植物 | 灵木、灵藤等 | 低至高级 | 取决于年份 |
| 妖兽材料 | 妖兽骨骼、皮甲、内丹 | 中至顶级 | 取决于妖兽等级 |
| 天地灵物 | 天地自然孕育 | 高至传说 | 极稀有 |
| 虚空材料 | 虚空中的特殊物质 | 传说级 | 几乎不可得 |

### 按品阶分类

| 品阶 | 对应法宝 | 说明 | 代表材料 |
|------|----------|------|----------|
| 低阶灵材 | 法器 | 常见灵矿灵木 | 玄铁、灵木 |
| 中阶灵材 | 灵器 | 品质较好 | 百年寒铁、万年灵木 |
| 高阶灵材 | 宝器 | 稀少 | 千年玄铁、神木 |
| 顶阶灵材 | 法宝 | 极稀有 | 万年玄铁、龙骨 |
| 灵物级 | 灵宝 | 天地灵物 | 天地精华凝结 |
| 仙材级 | 仙器 | 仙界材料 | 仙金、仙玉 |

## 材料处理

| 处理方式 | 说明 | 效果 |
|----------|------|------|
| 初炼 | 去除杂质 | 纯度提升 |
| 精炼 | 反复淬炼 | 品质提升 |
| 融合 | 多种材料融合 | 新的属性 |
| 淬火 | 以真火淬炼 | 激活材料灵性 |
| 养灵 | 长期蕴养 | 灵性增强 |
