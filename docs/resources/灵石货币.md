# 灵石货币

> 灵石是修仙世界的基础货币，蕴含灵气，可用于修炼和交易。

---

## 灵石概述

### 什么是灵石？

灵石是修仙世界的基础货币，是一种蕴含灵气的矿石。灵石不仅可以作为货币使用，还可以用于修炼，吸收其中的灵气。

### 灵石的特点

| 特点 | 说明 |
|------|------|
| 蕴含灵气 | 灵石蕴含纯净的灵气 |
| 可用于修炼 | 可以吸收灵气修炼 |
| 可作为货币 | 可以作为交易货币 |
| 价值稳定 | 价值相对稳定 |
| 易于携带 | 便于携带和储存 |
