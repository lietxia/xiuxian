# 妖兽栖息地

> 妖兽栖息地是各类妖兽生活和修炼的场所。本文详细介绍各界妖兽分布、危险区域、资源产地及领地规则。

## 妖兽栖息地概述

妖兽栖息地遍布修仙界各个界面，从凡间界到仙界都有妖兽存在。不同界面、不同区域的妖兽等级和类型各有不同。

### 栖息地分类

| 分类 | 特点 | 妖兽等级 | 说明 |
|------|------|---------|------|
| 常规栖息地 | 灵气中等 | 1-6级 | 大多数修仙界区域 |
| 灵脉区域 | 灵气浓郁 | 4-8级 | 灵脉附近 |
| 秘境深处 | 灵气极浓 | 7-10级 | 各界秘境 |
| 绝地禁区 | 极危险 | 10-13级 | 各界禁地 |
| 界面核心 | 最强妖兽 | 13级~真灵级 | 各界核心区域 |

## 领地规则

妖兽领地是妖兽势力范围的划分，进入他人领地可能引发冲突。

### 领地等级

| 领地类型 | 妖兽等级 | 领地范围 | 进入风险 |
|---------|---------|---------|---------|
| 微小领地 | 1-3级 | 数十丈 | 低 |
| 小型领地 | 4-5级 | 数百丈 | 中 |
| 中型领地 | 6-7级 | 数里 | 高 |
| 大型领地 | 8-10级 | 十里以上 | 极高 |
| 巨型领地 | 11-13级 | 百里以上 | 天灾级 |
| 圣地领地 | 真灵级 | 万里以上 | 不可进入 |

### 领地规则

1. **标记领地**：妖兽以气味、声音、爪痕标记领地
2. **入侵必战**：进入他人领地，领主妖兽必来驱逐或杀戮
3. **领地争夺**：妖兽之间争夺领地，胜者为王
4. **领地等级**：等级越高，领地越大
5. **领地资源**：领地内资源归领主所有
