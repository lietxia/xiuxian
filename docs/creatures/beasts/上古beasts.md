# 上古妖兽

> 上古妖兽是远古时代存留下来的强大异兽，实力远超普通妖兽。本文详细介绍上古妖兽的特点、代表性异兽及遭遇风险。

## 上古妖兽概述

上古妖兽，指远古时代（上古纪元至远古纪元）存在并延续至今的强大异兽。它们经历了天地大变，有些至今仍存活于世，有些沉睡于秘境绝地。上古妖兽实力惊人，10级以上的上古妖兽足以灭杀元婴期修士，13级上古妖兽可抗衡大乘修士。

### 上古妖兽特点

| 特点 | 说明 |
|------|------|
| 寿元极长 | 可存活数万年甚至数十万年 |
| 实力恐怖 | 同级碾压普通妖兽 |
| 血脉纯正 | 上古血脉，压制普通妖兽 |
| 神通强大 | 拥有上古传承神通 |
| 灵智超凡 | 智慧不弱于人类修士 |
