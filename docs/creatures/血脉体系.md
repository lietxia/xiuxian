# 血脉体系

> 血脉是妖兽天赋实力的根本决定因素，血脉越高，上限越高、神通越强。本文详细介绍血脉等级、血脉觉醒、血脉传承与血脉改造。

---

## 血脉概述

血脉，即妖兽体内蕴含的祖先传承之力。血脉等级决定了妖兽的天赋、上限、神通种类。血脉越高，妖兽越强，进化可能性越大。

### 血脉的重要性

| 重要性 | 说明 |
|------|------|
| 决定上限 | 血脉等级决定妖兽等级上限 |
| 决定天赋 | 血脉越高，天赋神通越多越强 |
| 血脉压制 | 高血脉压制低血脉妖兽 |
| 化形能力 | 血脉越高，化形越完美 |
| 寿元关联 | 血脉越高，寿元越长 |

---

## 血脉等级划分

| 血脉等级 | 稀有度 | 等级上限 | 天赋神通数量 | 代表妖兽 | 血脉压制范围 |
|---------|--------|---------|------------|---------|------------|
| 低等血脉 | 常见 | 1-3级 | 0-1个 | 普通虫兽、低等鸟兽 | 无 |
| 普通血脉 | 较常见 | 4-6级 | 1-2个 | 普通虎狼、蛇兽 | 低等血脉 |
| 优良血脉 | 较稀有 | 7-8级 | 2-3个 | 虎王、蛇王、狼王 | 普通、低等血脉 |
| 稀有血脉 | 稀有 | 9-10级 | 3-4个 | 蛟龙、灵凤、冰凤 | 优良、普通、低等血脉 |
| 上古血脉 | 极稀有 | 11-13级 | 4-5个 | 真龙血脉者、天凤血脉者 | 稀有以下所有血脉 |
| 真灵血脉 | 万中无一 | 真灵级 | 5个以上 | 真龙、天凤、鲲鹏 | 压制一切血脉 |

## 血脉纯度

血脉纯度是衡量血脉强弱的具体指标，同一血脉等级内仍有纯度差异。

| 纯度等级 | 描述 | 影响 | 说明 |
|---------|------|------|------|
| 混杂 | 血脉混杂，不纯 | 天赋神通减弱、上限降低 | 大多数野生妖兽 |
| 普通 | 血脉普通纯净 | 正常发挥血脉能力 | 多数宗门培养妖兽 |
| 纯净 | 血脉纯净 | 天赋神通增强、上限提升 | 精心培养的妖兽 |
| 极纯 | 血脉极纯 | 大幅提升血脉能力 | 稀有培养成果 |
| 完美 | 血脉完美纯净 | 血脉能力最大化 | 天地孕育或特殊培养 |
| 至纯 | 血脉至纯无瑕 | 可觉醒更高血脉 | 极难达成 |

## 血脉觉醒

血脉觉醒是妖兽激活体内潜藏血脉的过程，可大幅提升实力。

### 血脉觉醒条件

| 条件 | 重要性 | 说明 |
|------|--------|------|
| 血脉传承 | 关键 | 父母血脉越高，觉醒概率越高 |
| 精血淬炼 | 重要 | 吞噬同族高血脉精血 |
| 天地灵地 | 重要 | 在对应血脉灵地修炼 |
| 天劫淬炼 | 重要 | 渡劫时血脉觉醒概率高 |
| 天材地宝 | 加速 | 服用血脉觉醒丹药 |
| 岁月积累 | 必要 | 修炼足够长时间 |

### 血脉觉醒丹药

| 丹药 | 品级 | 效果 | 适用血脉 |
|------|------|------|---------|
| 血脉初醒丹 | 三品 | 觉醒低等血脉 | 低等血脉 |
| 血脉觉醒丹 | 四品 | 觉醒普通血脉 | 普通~优良血脉 |
| 血脉洗炼丹 | 五品 | 提升血脉纯度 | 所有血脉 |
| 血脉升华丹 | 六品 | 觉醒稀有血脉 | 优良~稀有血脉 |
| 血脉涅槃丹 | 八品 | 觉醒上古血脉 | 稀有~上古血脉 |
| 血脉至尊丹 | 九品 | 觉醒真灵血脉 | 传说级 |

## 血脉传承

血脉传承是妖兽将血脉能力传递给后代的方式。

| 传承方式 | 效果 | 条件 |
|---------|------|------|
| 自然传承 | 血脉略降一代 | 父母血脉越高，后代血脉越高 |
| 精血传承 | 血脉继承率高 | 父母分出部分精血传承 |
| 传承秘法 | 血脉继承率极高 | 族内血脉传承秘法 |
| 天地赐予 | 血脉可能超越父母 | 极少见，天地认可 |

### 血脉传承规则

1. **血脉衰减**：普通情况下，后代血脉略低于父母
2. **血脉继承**：父母血脉越高，后代血脉上限越高
3. **血脉突变**：极少数后代血脉突变，超越父母
4. **血脉融合**：不同血脉父母的后代可能血脉融合

## 血脉改造

血脉改造是人为改变妖兽血脉等级的手段。

| 改造方式 | 效果 | 风险 | 条件 |
|---------|------|------|------|
| 血脉融合术 | 融合多种血脉 | 高 | 需要高人施法 |
| 血脉移植术 | 移植高血脉 | 极高 | 极危险，易失败 |
| 血脉洗炼术 | 提升血脉纯度 | 中 | 需要洗灵池 |
| 血脉涅槃术 | 血脉涅槃升华 | 极高 | 需要涅槃丹和秘法 |
| 血脉剥夺术 | 剥夺他人血脉 | 极高 | 需剥夺他人血脉，天道所不容 |
