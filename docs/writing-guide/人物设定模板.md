# 修仙小说人物设定模板

> 本模板旨在帮助作者系统化地构建修仙小说中各类角色的完整设定，确保人物形象立体、逻辑自洽、具有成长空间。

---

## 一、主角模板

主角是整部作品的核心，其设定需要足够丰富以支撑长篇叙事，同时要有清晰的成长轨迹和内在驱动力。

### 基础信息

| 项目 | 填写说明 | 示例 |
|------|----------|------|
| **姓名** | 需符合世界观设定，可包含姓氏、表字、道号等 | 林青云（字：子渊，道号：青玄子） |
| **年龄** | 出场年龄及故事关键节点年龄 | 出场16岁，故事结束时约300岁 |
| **外貌** | 五官特征、身材、气质、标志性特征 | 剑眉星目，身姿挺拔，左眼角有一颗泪痣，常年一袭青衫 |

### 出身背景

**填写说明：** 详细描述主角的家庭出身、成长环境、早期经历。这是人物性格形成的基础，也是后续情节发展的重要伏笔。

**示例：**
林青云出身于东域一个小型修仙家族——林家。林家最强者仅为筑基后期，在当地三流势力中勉强立足。林青云是家族旁系子弟，父亲早逝，母亲为凡人，自幼饱受嫡系子弟欺凌。十岁那年，家族测试灵根，他被测出五灵根资质，被视为"废柴"，从此更加边缘化。然而正是这段经历，造就了他坚韧不拔的性格和对力量的极度渴望。

### 灵根属性与纯度

**填写说明：** 灵根是决定修仙资质的核心要素。需明确属性（金木水火土及变异灵根）、纯度（百分比或等级）、特殊之处。

**示例：**
- **主灵根：** 木灵根（纯度65%，中等偏上）
- **副灵根：** 水灵根（纯度30%）、火灵根（纯度5%）
- **隐藏灵根：** 因血脉觉醒后显现的雷灵根（纯度90%，极品）
- **特殊说明：** 初期被视为五灵根废柴，实则是罕见的"五行相生"体质，五灵根平衡反而成为后期修炼特殊功法的优势

### 悟性等级

**填写说明：** 悟性影响功法领悟速度、法术学习效率和突破瓶颈的能力。可分为：凡品、良品、上品、极品、绝世、传说等。

**示例：**
悟性等级为"极品"，尤其在剑道领悟上堪称绝世。普通剑法看一遍即可入门，三遍即可小成。对剑意的感知敏锐，能在战斗中快速领悟对手的剑道精髓。但阵法、炼丹方面悟性仅为良品，需要付出更多努力。

### 体质特点

**填写说明：** 特殊体质是修仙小说中的重要设定，可以是先天遗传，也可以是后天奇遇所得。需说明体质名称、效果、限制、觉醒条件。

**示例：**
- **体质名称：** 青木圣体（先天）+ 不灭剑骨（后天获得）
- **青木圣体效果：** 木系功法修炼速度+300%，疗伤恢复能力极强，对毒物有天然抗性，可催生灵药加速生长
- **不灭剑骨效果：** 骨骼坚如法宝，剑气亲和度极高，可承受常人无法承受的剑气灌注
- **限制：** 青木圣体需定期吸收木系天材地宝维持，否则会逐渐衰退；不灭剑骨在获得时会经历七七四十九天的骨髓重塑之痛

### 血脉传承

**填写说明：** 血脉可以是远古祖先的馈赠，也可以是特殊种族的遗留。需说明血脉来源、纯度、觉醒程度、带来的能力和隐患。

**示例：**
林青云身怀上古剑仙"青莲剑主"的稀薄血脉（纯度约1%），这是他在获得剑骨传承时意外觉醒的。觉醒后获得能力：
1. 剑道亲和力大幅提升
2. 可施展血脉神通"青莲剑歌"（短时间内剑气威力翻倍）
3. 对剑类法宝有天然的感应能力

隐患：血脉觉醒会引起某些远古势力的注意，可能成为被研究或猎取的对象。

### 主修功法

**填写说明：** 主角的核心修炼功法，需详细描述功法名称、品阶、特性、修炼要求、成长潜力。

**示例：**
**《青莲剑诀》**（天阶上品，可进化）
- **来源：** 上古剑修洞府传承
- **特性：** 以剑入道，剑气与灵气同修，攻防一体
- **核心招式：** 
  - 第一重·青莲初绽（筑基期可修）：剑气如莲花绽放，范围攻击
  - 第二重·莲开并蒂（金丹期可修）：双剑合璧，威力倍增
  - 第三重·万剑归宗（元婴期可修）：万道剑气合一，毁天灭地
  - 第四重·青莲剑域（化神期可修）：以剑成域，领域内剑道无敌
- **特殊之处：** 可与木系功法配合，催生"剑莲"这一特殊攻击手段

### 辅修功法

**填写说明：** 辅助性功法，可以是身法、炼体术、神识修炼法等。

**示例：**
1. **《乙木长春功》**（地阶中品）：主修功法，提供浑厚灵力基础，配合青木圣体效果极佳
2. **《御风九变》**（地阶上品）：身法，九种变化，飘忽不定，难以捉摸
3. **《不灭金身诀》**（天阶下品）：炼体功法，配合不灭剑骨，肉身强度堪比法宝
4. **《神识凝剑术》**（玄阶极品）：神识攻击法门，可将神识凝聚成剑，直接攻击对手识海

### 常用武器/法宝

**填写说明：** 列出主角的主要装备，包括本命法宝、常用武器、防御法宝、辅助法宝等。

**示例：**
| 名称 | 类型 | 品阶 | 特性 |
|------|------|------|------|
| 青莲剑 | 本命飞剑 | 可成长（当前天阶中品） | 以青莲剑气温养，与主人心意相通，可施展青莲剑诀全部威力 |
| 玄龟盾 | 防御法宝 | 地阶上品 | 可抵挡元婴期全力一击三次，自动护主 |
| 储物戒指 | 辅助 | 天阶下品 | 内部空间百丈见方，时间流速为外界十分之一，可保鲜灵药 |
| 遁空符 | 消耗品 | 一次性 | 危急时刻可瞬间遁出千里，每月可制作三张 |

### 性格特点

**填写说明：** 性格是人物的灵魂，需要多维度描述，包括优点、缺点、核心价值观、行为模式等。

**示例：**
**核心性格：** 坚韧、执着、重情重义、有仇必报

**详细分析：**
- **坚韧：** 早年经历造就了他百折不挠的意志，无论遭遇多大挫折都能重新站起
- **执着：** 对剑道的追求近乎偏执，一旦认定目标绝不放弃
- **重情重义：** 对真心待他的人加倍回报，对敌人则毫不留情
- **有仇必报：** 恩怨分明，睚眦必报，但从不牵连无辜
- **冷静：** 战斗中能保持绝对冷静，善于分析局势
- **缺陷：** 有时过于固执，不愿接受他人帮助；对亲近之人保护欲过强，容易冲动

### 成长目标

**填写说明：** 主角在不同阶段的追求和目标，这是推动情节发展的核心动力。

**示例：**
- **短期目标（练气-筑基）：** 摆脱"废柴"之名，获得家族认可，保护母亲不受欺凌
- **中期目标（金丹-元婴）：** 建立属于自己的势力，寻找父亲死亡的真相，为师门报仇
- **长期目标（化神-大乘）：** 飞升仙界，追寻剑道极致，复活已故的挚爱
- **终极目标：** 成为万古第一剑仙，打破天地桎梏，让天下修士不再受资质所限

### 人物弧线

**填写说明：** 描述主角从故事开始到结束的完整成长轨迹。

**示例：**

**起点：**
十六岁的林青云是家族公认的废柴，五灵根资质被判定终生无法突破练气三层。母亲病重无钱医治，他被嫡系子弟欺辱却无力反抗。在一次被陷害坠入悬崖后，意外获得上古剑修传承，命运开始转折。

**转折：**
1. **第一次转折（筑基期）：** 家族大比中一鸣惊人，击败曾经欺辱他的嫡系天才，获得进入大宗门的机会
2. **第二次转折（金丹期）：** 发现父亲死亡真相与正道大派有关，被迫在复仇与正道之间做出选择，最终选择以自己的方式追求正义
3. **第三次转折（元婴期）：** 挚爱为救他而死，他一度堕入魔道，后在师友帮助下重新找回本心，领悟"有情剑道"
4. **第四次转折（化神期）：** 揭露上古秘辛，发现飞升之路被人为阻断，决定打破桎梏，为后世修士开辟新路

**终点：**
三百年后，林青云以"青莲剑主"之名飞升仙界，留下无数传说。他在飞升前将毕生所学整理成《青莲剑典》，供后世有缘人修习。仙界之中，他继续追寻剑道极致，最终成为仙界公认的剑道至尊。

配角是主角成长路上的重要陪伴者，他们的存在让故事更加丰满，也衬托出主角的不同侧面。

## 三、反派模板

反派是推动剧情的重要力量，好的反派不是单纯的"恶"，而是有自己的动机和逻辑。

### 悲剧背景型

**核心设定：** 曾经也是好人，因遭遇重大变故而黑化。

| 项目 | 填写说明 | 示例 |
|------|----------|------|
| **姓名** | | 莫问天 |
| **原身份** | 黑化前是什么人 | 正道大派长老，德高望重，弟子众多 |
| **黑化原因** | 什么事件导致转变 | 挚爱被宗门高层害死，为报仇修炼禁术，逐渐迷失自我 |
| **行为逻辑** | 为什么做坏事 | 认为正道虚伪，要毁灭现有秩序建立新世界 |
| **与主角关系** | | 主角曾视其为榜样，后发现真相，成为必须击败的对手 |
| **悲剧性** | 令人同情之处 | 临终前恢复清明，将毕生所学传给主角，请求主角完成他未竟的心愿 |
| **经典台词** | | "我不是天生为恶，是这个世界逼我成魔！" |

### 野心勃勃型

**核心设定：** 天生枭雄，为达目的不择手段。

| 项目 | 填写说明 | 示例 |
|------|----------|------|
| **姓名** | | 帝释天 |
| **身份** | | 魔道至尊，一统魔道六宗的霸主 |
| **目标** | 想要什么 | 打破天地桎梏，成为万古第一人，甚至取代天道 |
| **手段** | 做事风格 | 冷酷无情，顺我者昌逆我者亡，但赏罚分明，手下忠心耿耿 |
| **魅力点** | 为什么有人追随 | 雄才大略，言出必行，对追随者慷慨大方 |
| **与主角关系** | | 欣赏主角的潜力，多次招揽，被拒绝后视为必须铲除的威胁 |
| **结局** | | 最终决战被主角击败，临死前大笑："痛快！这天下，交给你了！" |

### 身不由己型

**核心设定：** 并非本性为恶，而是被形势所迫或被人控制。

| 项目 | 填写说明 | 示例 |
|------|----------|------|
| **姓名** | | 白素心 |
| **原身份** | | 医仙谷圣女，心地善良，救死扶伤 |
| **被迫原因** | | 被种下噬心蛊，不得不听从幕后黑手命令 |
| **内心挣扎** | | 每次害人后都痛苦万分，暗中救助受害者，收集幕后黑手证据 |
| **与主角关系** | | 主角发现其苦衷后决定帮助她，两人联手对抗真正的敌人 |
| **救赎** | | 蛊毒解除后，用余生弥补过错，成为主角最重要的盟友之一 |

## 四、NPC模板

NPC是构建修仙世界真实感的基础，虽然戏份不多，但能让世界更加立体。

### 店铺老板模板

| 项目 | 填写说明 | 示例（丹药铺） | 示例（法宝铺） |
|------|----------|----------------|----------------|
| **姓名** | | 钱多多 | 欧冶魂 |
| **店铺名** | | 百草堂 | 神兵阁 |

## 五、使用建议

1. **不要一次性填满所有内容**：先填写核心信息，随着故事发展逐步补充
2. **保持设定一致性**：人物设定一旦确定，除非有合理的情节推动，否则不要随意更改
3. **留白给成长空间**：不要把人物设定得太满，要给角色在故事中成长变化的空间
4. **人物之间要有联系**：配角和主角之间、配角彼此之间要有合理的关系网
5. **反派也要有血有肉**：避免脸谱化的反派，给他们合理的动机和背景
6. **NPC也要有记忆点**：即使是路人，也可以有一两个鲜明的特征，让读者记住
