# 技能分类

> 技能是修仙者战斗的核心能力，包括主动技能、被动技能、终极技能、组合技能等。

---

## 技能概述

### 什么是技能？

技能是修仙者运用灵气施展的特殊能力，是战斗的核心。技能分为主动技能、被动技能、终极技能、组合技能等。

### 技能的特点

| 特点 | 说明 |
|------|------|
| 多样性 | 技能种类繁多 |
| 等级划分 | 技能有等级划分 |
| 属性对应 | 技能有属性对应 |
| 组合性 | 技能可以组合使用 |
| 成长性 | 技能可以成长 |

---

## 主动技能

### 主动技能的定义

主动技能是需要修仙者主动施展的技能，消耗灵气，有冷却时间。

### 主动技能分类

| 分类 | 说明 | 代表技能 |
|------|------|----------|
| 攻击技能 | 用于攻击敌人 | 火球术、水箭术 |
| 防御技能 | 用于防御保护 | 护盾术、防御结界 |
| 辅助技能 | 用于辅助增益 | 加速术、治疗术 |
| 控制技能 | 用于控制敌人 | 束缚术、定身术 |
| 位移技能 | 用于移动位置 | 瞬移、闪现 |

### 主动技能等级

| 等级 | 说明 | 威力 |
|------|------|------|
| 初级 | 基础技能 | 较弱 |
| 中级 | 进阶技能 | 中等 |
| 高级 | 高级技能 | 较强 |
| 顶级 | 顶级技能 | 很强 |
| 禁忌 | 禁忌技能 | 极强 |

---

## 被动技能

### 被动技能的定义

被动技能是持续生效的技能，不需要主动施展，自动发挥作用。

### 被动技能分类

| 分类 | 说明 | 代表技能 |
|------|------|----------|
| 属性增强 | 增强基础属性 | 力量增强、速度增强 |
| 防御增强 | 增强防御能力 | 护体罡气、防御强化 |
| 恢复增强 | 增强恢复能力 | 自动回血、自动回蓝 |
| 抗性增强 | 增强抗性能力 | 元素抗性、控制抗性 |
| 特殊效果 | 特殊被动效果 | 反伤、吸血 |

### 被动技能等级

| 等级 | 说明 | 效果 |
|------|------|------|
| 普通 | 基础被动 | 效果一般 |
| 优秀 | 进阶被动 | 效果较好 |
| 精良 | 高级被动 | 效果很好 |
| 史诗 | 顶级被动 | 效果极好 |
| 传说 | 传说被动 | 效果逆天 |

---

## 终极技能

### 终极技能的定义

终极技能是威力最强的技能，通常有很长的冷却时间，但威力巨大。

### 终极技能分类

| 分类 | 说明 | 代表技能 |
|------|------|----------|
| 毁灭技能 | 毁灭性攻击 | 灭世一击、天地同寿 |
| 领域技能 | 释放领域 | 时间静止、空间封锁 |
| 召唤技能 | 召唤强大存在 | 召唤神兽、召唤分身 |
| 变身技能 | 变身形态 | 三头六臂、法天象地 |
| 超越技能 | 超越极限 | 超越生死、逆转时空 |

### 终极技能等级

| 等级 | 说明 | 威力 |
|------|------|------|
| 禁术 | 禁忌技能 | 极强 |
| 仙术 | 仙人技能 | 超强 |
| 神术 | 神灵技能 | 逆天 |
| 大道术 | 大道技能 | 超脱 |
| 混沌术 | 混沌技能 | 无极 |

---

## 组合技能

### 组合技能的定义

组合技能是多个技能组合使用的技能，可以产生更强的效果。

### 组合技能分类

| 分类 | 说明 | 代表组合 |
|------|------|----------|
| 攻击组合 | 多个攻击技能组合 | 连续攻击、组合攻击 |
| 防御组合 | 多个防御技能组合 | 多重防御、综合防御 |
| 控制组合 | 多个控制技能组合 | 连续控制、综合控制 |
| 辅助组合 | 多个辅助技能组合 | 综合增益、连续治疗 |
| 混合组合 | 多种技能组合 | 攻防一体、控制攻击 |

### 组合技能的要点

1. **技能搭配**：技能之间要搭配合理
2. **时机把握**：把握技能释放时机
3. **顺序安排**：安排好技能释放顺序
4. **消耗控制**：控制技能消耗
5. **冷却管理**：管理技能冷却
