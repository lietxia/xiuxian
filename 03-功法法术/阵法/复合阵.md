## 一、复合阵概述

### 1.1 什么是复合阵

复合阵，是相对于单一功能阵法（攻击阵、防御阵、困阵等）而言的综合性阵法。与单一阵法相比，复合阵具有以下三大特点：

- **攻防困一体**：同时具备攻击、防御和困锁三大功能，可攻可守可困
- **功能联动**：阵内各功能相互联动，形成一加一大于二的效果
- **变化无穷**：阵法运转之中，千变万化，令敌人防不胜防

### 1.2 复合阵的三大构成

| 构成要素 | 说明 | 重要性 |
|:---:|:---|:---:|
| 攻之核心 | 阵法中的攻击力量来源 | ★★★ |
| 防之根基 | 阵法的防护屏障与结构 | ★★★ |
| 困之锁链 | 限制敌人行动的禁制 | ★★★ |

### 1.3 复合阵与单一阵的区别

| 对比项 | 单一阵法 | 复合阵 |
|:---:|:---:|:---:|
| 功能数量 | 仅一种 | 三种以上 |
| 布阵难度 | 较低 | 极高 |
| 灵气消耗 | 较少 | 巨大 |
| 适用场景 | 专精某一目的 | 综合战场 |
| 破解难度 | 针对破解即可 | 需同时应对多重威胁 |

## 三、一阶至三阶复合阵

### 3.1 一阶复合阵

一阶复合阵是入门级复合阵，功能简单但已具备攻防困雏形，适合初学者练习。

#### 三才阵

- **阶位**：一阶
- **布阵人数**：3人（可单人操控三枚阵基）
- **布阵材料**：普通灵石三枚、符纸三张、灵木三根
- **布阵条件**：需三名修士协同，或单人操控三处阵基

**阵法结构：**

```
        【天位】— 攻击
           ╱    ╲
         ╱        ╲
       ╱            ╲
     ╱                ╲
【人位】——————【核心】——————【人位】— 防御
     ╲                ╱
       ╲            ╱
         ╲        ╱
           ╲    ╱
        【地位】— 困锁
```

**功能详解：**

| 功能 | 名称 | 效果描述 |
|:---:|:---:|:---|
| 攻 | 天火降临 | 从天位降下凡火攻击敌人，威力相当于一阶火球符 |
| 防 | 金刚护体 | 人位形成灵气护盾，抵挡低阶攻击 |
| 困 | 大地束缚 | 地位突起土刺，限制敌人行动（数息） |

**威力评价**：★☆☆☆☆（入门级）
**破解方法**：以蛮力破阵基，或打断布阵者与阵基的联系

#### 四象基础阵

- **阶位**：一阶
- **布阵人数**：4人（可单人操控四枚阵基）
- **布阵材料**：普通灵石四枚、符纸四张、朱砂墨少许
- **布阵条件**：需四象方位（青龙、白虎、朱雀、玄武）各置一阵基

**阵法结构：**

```
    【朱雀位】— 火攻
         ╲    ╱
           ╲╱
    【青龙位】 ╳ 【白虎位】
     风助    核心    金杀
           ╱╲
         ╱    ╲
    【玄武位】— 水防/困锁
```

**功能详解：**

| 功能 | 方位 | 效果描述 |
|:---:|:---:|:---|
| 攻（朱雀） | 南方火位 | 火焰攻击，附带有灼烧效果 |
| 攻（白虎） | 西方金位 | 金属性攻击，穿透力强 |
| 防（玄武） | 北方水位 | 水盾防御，可抵挡物理攻击 |
| 困（青龙） | 东方木位 | 藤蔓缠绕，限制移动 |

**威力评价**：★☆☆☆☆（入门级）
**特点**：四象相生相克，功能均衡

## 七、复合阵布阵要诀

### 7.1 布阵基础

1. **选定阵眼**：阵眼是阵法的核心，通常选择阵法威力最强之处
2. **布置阵基**：阵基是阵法的支撑点，需按方位布置
3. **连通阵纹**：以灵气在阵基之间绘制阵纹，连接各阵基
4. **激活阵法**：最后一步，以灵气激活阵法核心

### 7.2 复合阵特殊要求

- **神识要求**：复合阵需同时操控多个阵基，对神识要求极高
- **灵气储备**：复合阵消耗灵气巨大，需有充足的灵气储备
- **协同配合**：多人布阵时，各布阵者需心意相通，协同配合
- **阵图精通**：需精通各类阵图，能将多种阵法融合为一

### 7.3 复合阵的进阶之路

| 阶段 | 能力 | 可布阵阶 |
|:---:|:---|:---:|
| 入门 | 掌握三才、四象 | 一阶~二阶 |
| 小成 | 精通五行、八卦 | 三阶~四阶 |
| 大成 | 领悟天地之道 | 五阶~六阶 |
| 圆满 | 掌握大道法则 | 七阶~八阶 |
| 极致 | 万道归一 | 九阶 |
