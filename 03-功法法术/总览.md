## 功法体系概述

### 功法的定义

功法是修仙者修炼的核心技能和方法，包括攻击、防御、辅助、幻术、遁术等多种类型，是修仙者提升实力的重要途径。

### 功法的特点

| 特点 | 说明 |
|------|------|
| 多样性 | 种类繁多，包括攻击、防御、辅助、幻术、遁术等 |
| 等级划分 | 分为9个等级，对应修仙者的不同境界 |
| 修炼难度 | 等级越高，修炼难度越大 |
| 威力递增 | 等级越高，威力越大 |
| 属性对应 | 不同属性功法对应不同灵根属性 |

### 功法等级划分

| 等级 | 对应境界 | 威力 | 修炼难度 |
|------|----------|------|----------|
| 一阶 | 练气期 | 较小 | 低 |
| 二阶 | 筑基期 | 中等 | 中等 |
| 三阶 | 金丹期 | 较大 | 较高 |
| 四阶 | 元婴期 | 强大 | 高 |
| 五阶 | 化神期 | 极强 | 极高 |
| 六阶 | 炼虚期 | 逆天 | 逆天 |
| 七阶 | 合体期 | 无边 | 无边 |
| 八阶 | 大乘期 | 通天 | 通天 |
| 九阶 | 渡劫期 | 混沌 | 混沌 |

## 法术体系概述

### 法术的定义

法术是修仙者运用灵气施展的特殊能力，包括攻击、防御、辅助、控制等多种类型。

### 法术的分类

| 分类 | 说明 | 代表法术 |
|------|------|----------|
| 元素法术 | 运用五行元素 | 火球术、水箭术、金刃术 |
| 召唤法术 | 召唤灵兽或分身 | 召唤灵兽、召唤分身 |
| 封印法术 | 封印敌人或宝物 | 封印术、禁锢术 |
| 禁术 | 禁忌法术 | 禁忌功法、禁忌法术 |

---

## 阵法体系概述

### 阵法的定义

阵法是修仙者通过特殊布置形成的法术结构，具有攻击、防御、困敌、传送等多种功能。

### 阵法的分类

| 分类 | 说明 | 代表阵法 |
|------|------|----------|
| 攻击阵法 | 用于攻击敌人 | 杀阵、困阵 |
| 防御阵法 | 用于防御保护 | 护山大阵、防御结界 |
| 困阵幻阵 | 用于困敌迷惑 | 迷阵、幻阵 |
| 传送阵法 | 用于空间传送 | 传送阵、空间通道 |

---

## 符箓体系概述

### 符箓的定义

符箓是修仙者绘制的特殊法术载体，可以储存法术，随时释放使用。

### 符箓的分类

| 分类 | 说明 | 代表符箓 |
|------|------|----------|
| 攻击符箓 | 用于攻击敌人 | 火符、雷符、剑符 |
| 防御符箓 | 用于防御保护 | 护身符、防御符 |
| 辅助符箓 | 用于辅助增益 | 隐身符、加速符、治疗符 |

## 功法传承方式

### 1. 门派传承
通过门派内部传承，只有门派弟子才能学习。

### 2. 家族传承
通过家族内部传承，只有家族成员才能学习。

### 3. 奇遇传承
通过奇遇获得，如机缘巧合下获得上古功法秘籍。

### 4. 自创功法
修仙者根据自己的理解和感悟，创造出属于自己的功法。
