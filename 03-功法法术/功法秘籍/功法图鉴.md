# 功法图鉴

## 概述

功法，是修仙之道的根基与核心，是每一位修仙者踏上修行之路的根本依托。所谓功法，即"功"与"法"之合称——"功"者，蓄积之力也，指修炼者积累灵气的法门；"法"者，运用之道也，指运用灵气的方法。功法图鉴，便是系统记载修真界所有已知功法的完整典籍，涵盖了功法的名称、属性、功效、修炼方法、注意事项等全部信息。

功法图鉴是修仙者了解功法世界的重要窗口，不同属性的功法适合不同灵根、不同体质的修士修炼。功法选择得当，可以事半功倍；功法选择不当，轻则修炼缓慢，重则走火入魔。因此，每一位有志于长生的修仙者，都必须对功法图鉴有深入的了解，才能选择最适合自己的功法。

---

## 功法图鉴定义

功法图鉴按属性分类，详细记载了修真界所有已知功法的完整信息。每一种功法都有其独特的属性定位和功效特点，修士在选择功法时，必须综合考虑自身灵根、体质、修为阶段等多方面因素，才能做出最佳选择。

功法图鉴的分类原则是：以功法的核心属性为依据，将所有功法分为金、木、水、火、土、风、雷、冰、暗、光十大属性体系。每一属性体系下，又按功法的功效分为攻击、防御、辅助三大类别。这种分类方式既便于修士按图索骥，快速找到适合自己的功法，又便于理解功法之间的相生相克关系。

## 各属性功法的相生相克表

在修真界中，五行功法之间存在着相生相克的关系。了解这些关系，对于修士选择功法、制定战斗策略至关重要。

### 五行相生表

| 属性 | 相生 | 说明 |
|------|------|------|
| 木 | 生火 | 木生火，木属性功法可增强火属性功法威力 |
| 火 | 生土 | 火生土，火属性功法可增强土属性功法威力 |
| 土 | 生金 | 土生金，土属性功法可增强金属性功法威力 |
| 金 | 生水 | 金生水，金属性功法可增强水属性功法威力 |
| 水 | 生木 | 水生木，水属性功法可增强木属性功法威力 |

### 五行相克表

| 属性 | 相克 | 被克 | 说明 |
|------|------|------|------|
| 木 | 克土 | 被金克 | 木属性功法克制土属性，但被金属性克制 |
| 土 | 克水 | 被木克 | 土属性功法克制水属性，但被木属性克制 |
| 水 | 克火 | 被土克 | 水属性功法克制火属性，但被土属性克制 |
| 火 | 克金 | 被水克 | 火属性功法克制金属性，但被水属性克制 |
| 金 | 克木 | 被火克 | 金属性功法克制木属性，但被火属性克制 |

### 特殊属性关系

| 属性组合 | 关系 | 说明 |
|----------|------|------|
| 风 + 雷 | 相辅 | 风助雷势，雷借风威，二者叠加威力倍增 |
| 冰 + 水 | 同源 | 冰为水之极，冰系功法本质上是强化版水系 |
| 光 + 暗 | 相克 | 光暗不两立，二者相遇必有一伤 |
| 暗 + 冰 | 相近 | 暗与冰皆属阴寒，二者可相互增强 |
| 火 + 雷 | 相激 | 火与雷相遇会产生剧烈反应，威力大增但失控风险也大 |
| 光 + 火 | 相生 | 光明之火为至阳之火，威力远超单一属性 |

### 相生相克在战斗中的应用

**利用相生增强威力：**
- 火属性修士可兼修木属性功法，增强火系功法威力
- 水属性修士可兼修金属性功法，增强水系功法威力

**利用相克克制敌人：**
- 面对土属性敌人，优先使用木属性功法
- 面对金属性敌人，优先使用火属性功法
- 面对水属性敌人，优先使用土属性功法
- 面对火属性敌人，优先使用水属性功法
- 面对木属性敌人，优先使用金属性功法

**避免被克制：**
- 面对克制自己属性的敌人时，可使用辅助手段改变自身属性
- 或使用无属性功法，避免被克制
