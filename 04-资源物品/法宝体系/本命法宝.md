# 本命法宝

> 本命法宝——与修士性命相连的独特法宝，随主成长，生死与共。是修仙者最重要的伙伴。

---

## 本命法宝概述

本命法宝是修士在金丹期以上，以自身金丹/元婴为核心，以精血神魂为引，炼制出的与自身性命相连的法宝。

### 本命法宝特点

| 特点 | 说明 |
|------|------|
| 生死相连 | 法宝损毁则修士重伤，修士身亡则法宝受损 |
| 随主成长 | 随修士修为提升而成长 |
| 心意相通 | 可如臂使指，无需催动即可响应 |
| 独一无二 | 每位修士只能拥有一件本命法宝 |
| 不可剥夺 | 除非修士身死，否则无法被夺取 |

## 炼制条件

### 金丹期炼制

| 条件 | 要求 |
|------|------|
| 修为 | 金丹期以上 |
| 材料 | 顶级灵材或天地灵物 |
| 方法 | 以金丹之力温养材料 |
| 时间 | 至少三年 |
| 成功率 | 约30% |

### 元婴期炼制

| 条件 | 要求 |
|------|------|
| 修为 | 元婴期以上 |
| 材料 | 法宝级材料 |
| 方法 | 以元婴神识铭刻法则 |
| 时间 | 至少十年 |
| 成功率 | 约50% |

| 阶段 | 对应修为 | 法宝等级 | 能力 |
|------|----------|----------|------|
| 胚胎期 | 金丹期 | 顶阶法器→宝器 | 基础能力 |

| 方式 | 说明 | 效果 |
|------|------|------|

| 天劫淬炼 | 随主人渡劫时一起淬炼 | 大幅提升 |
| 吞噬材料 | 吞噬同属性高级材料 | 阶段性提升 |

## 本命法宝的损毁与修复

### 损毁后果

| 损毁程度 | 对修士影响 | 恢复难度 |
|----------|-----------|----------|
| 轻微受损 | 修为轻微震荡 | 容易修复 |
| 中度受损 | 修为倒退一个小境界 | 需要时间蕴养 |
| 重度受损 | 修为倒退一个大境界 | 需要天材地宝修复 |
| 彻底损毁 | 元气大伤，修为大幅倒退 | 几乎无法修复 |

### 修复方法

| 方法 | 适用损毁 | 所需材料 | 成功率 |
|------|----------|----------|--------|
| 灵力蕴养 | 轻微受损 | 无需特殊材料 | 100% |
| 祭炼修复 | 中度受损 | 同属性灵材 | 80% |
| 天材修复 | 重度受损 | 同属性天材地宝 | 30%-50% |
| 重铸 | 彻底损毁 | 等同重新炼制 | 10%以下 |
