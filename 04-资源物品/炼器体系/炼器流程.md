# 炼器流程

> 炼器流程是炼器师必须掌握的核心技术，每一步都关乎法宝的最终品质。

---

## 炼器总流程

### 基本步骤

| 步骤 | 顺序 | 说明 | 难度 |
|------|------|------|------|
| 选材 | 1 | 选择合适的灵材 | 中 |
| 处理 | 2 | 清洗、切割、提炼材料 | 中 |
| 熔炼 | 3 | 以真火熔炼材料 | 高 |
| 铸形 | 4 | 铸造法宝基本形态 | 高 |
| 铭纹 | 5 | 铭刻灵纹和禁制 | 极高 |
| 凝丹 | 6 | 凝聚核心能量节点 | 极高 |
| 开光 | 7 | 注入灵力激活法宝 | 高 |
| 蕴养 | 8 | 长期蕴养稳定法宝 | 需时间 |

---

## 各步骤详解

### 第一步：选材

| 要点 | 说明 |
|------|------|
| 属性匹配 | 材料属性需与目标法宝属性匹配 |
| 品质达标 | 材料品质需达到法宝等级要求 |
| 数量充足 | 准备充足材料，留有余量 |
| 相容性 | 多种材料之间需具备相容性 |
| 年份要求 | 灵草灵木需达到一定年份 |

### 第二步：处理

| 处理方式 | 说明 | 目的 |
|----------|------|------|
| 清洗 | 以灵泉水清洗 | 去除表面杂质 |
| 切割 | 按丹方要求切割 | 便于熔炼 |
| 提纯 | 反复提炼 | 提高纯度 |
| 淬火 | 以真火初步淬炼 | 激活灵性 |
| 鉴定 | 鉴定材料品质 | 确保达标 |

### 第三步：熔炼

| 阶段 | 火候 | 时间 | 关键 |
|------|------|------|------|
| 初始熔炼 | 文火 | 视材料而定 | 均匀受热 |
| 深度熔炼 | 武火 | 视材料而定 | 完全熔化 |
| 融合熔炼 | 丹火 | 精确控制 | 材料完美融合 |
| 淬炼提纯 | 武火+丹火交替 | 反复多次 | 去除所有杂质 |

### 第四步：铸形

| 方式 | 说明 | 适用法宝 |
|------|------|----------|
| 模具铸造 | 以模具定型 | 批量法器 |
| 手工锻造 | 以锤击定型 | 高品质法宝 |
| 灵力铸形 | 以灵力直接塑形 | 高阶法宝 |
| 虚空造物 | 以心神为炉，虚空铸形 | 顶级法宝（元婴期以上） |

### 第五步：铭纹

这是炼器最核心也最困难的步骤。

| 铭纹类型 | 难度 | 效果 | 铭刻要求 |
|----------|------|------|----------|
| 基础灵纹 | 中等 | 赋予基础能力 | 精神力足够 |
| 高阶灵纹 | 高 | 赋予特殊能力 | 金丹期以上神识 |
| 宝禁 | 极高 | 法宝核心能力 | 元婴期以上 |
| 道纹 | 传说 | 蕴含法则 | 需领悟对应法则 |

| 铭纹数量对照 | 法宝等级 |
|-------------|----------|
| 1-12道灵纹 | 法器 |
| 13-24道灵纹 | 灵器/宝器 |
| 25-100道灵纹 | 法宝 |
| 100-1000道灵纹 | 灵宝 |
| 1000-12000道灵纹 | 通天灵宝 |
| 12000道以上 | 玄天级 |

### 第六步：凝聚核心

| 核心 | 说明 | 作用 |
|------|------|------|
| 灵力核心 | 注入灵力形成能量节点 | 法宝的动力源 |
| 法则核心 | 注入法则之力 | 高阶法宝的根基 |
| 灵智种子 | 埋下灵智萌芽 | 未来孕育器灵 |

### 第七步：开光

| 开光方式 | 适用法宝 | 效果 |
|----------|----------|------|
| 灵力灌注 | 普通法器 | 激活基础功能 |
| 真火开光 | 灵器以上 | 全面激活 |
| 法则开光 | 灵宝以上 | 激活法则之力 |
| 天劫开光 | 顶级法宝 | 天道认可 |

### 第八步：蕴养

| 蕴养方式 | 时间 | 效果 |
|----------|------|------|
| 灵力蕴养 | 数月 | 稳定法宝 |
| 真火蕴养 | 数年 | 提升品质 |
| 滴血蕴养 | 长期 | 建立联系 |
| 天地蕴养 | 数十年 | 极品品质 |

## 炼器失败原因

| 失败原因 | 发生概率 | 后果 |
|----------|----------|------|
| 材料品质不足 | 较高 | 法宝品质下降或失败 |
| 火候控制失误 | 中等 | 材料损毁 |
| 灵纹铭刻错误 | 较低 | 需重新铭刻或材料报废 |
| 融合失败 | 中等 | 材料相互排斥 |
| 天地干扰 | 极低 | 全部报废 |
| 炼器师状态不佳 | 中等 | 各环节均可能出错 |
