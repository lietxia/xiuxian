# 冥界详解

> 本文档详细介绍冥界（阴间）的地理、势力、轮回体系与规则。

---

## 一、冥界概述

冥界是修仙世界中灵魂轮回的中转站，由地藏王菩萨统领，十殿阎罗分管。它既是亡魂的归宿，也是鬼修的修炼之地，掌控着生死的边界。

**核心特征**：
- 灵气类型：阴气、死气、冥气
- 修行理念：魂力修炼、轮回法则
- 界域意志：轮回法则
- 与各界关系：灵魂通道连接所有界域

---

## 二、冥界地理

### 鬼门关

冥界的入口，所有亡魂必经之地：

- 位置：凡间界与冥界交界处
- 守护者：鬼差、黑白无常
- 功能：登记亡魂身份
- 特点：生人禁入，违者必罚

### 黄泉路

从鬼门关到奈何桥的必经之路：

- 长度：九百里
- 特点：阴气弥漫，无回头路
- 亡魂状态：无法停留，只能前行
- 传说：路边的彼岸花永不凋谢

### 忘川河与奈何桥

**忘川河**：
- 宽度：无定，越走越宽
- 河水：腐蚀记忆
- 落水后果：永堕河中，不得超生

**奈何桥**：
- 分为三桥：上、中、下
- 上桥：有功德者通行
- 中桥：普通亡魂通行
- 下桥：大恶者被迫通行

### 孟婆庄

奈何桥边，亡魂喝孟婆汤之地：

- 庄主：孟婆
- 功能：制作并发放孟婆汤
- 孟婆汤效果：抹除前世记忆
- 特殊情况：部分记忆可保留（需特殊安排）

| 天道 | 享乐为主 | 大功德 |
| 人道 | 苦乐参半 | 善恶参半 |

| 饿鬼道 | 贪婪受苦 | 贪婪恶业 |
| 地狱道 | 极苦受刑 | 大恶 |

### 轮回的记忆抹除机制

**正常流程**：
1. 喝孟婆汤
2. 完全抹除前世记忆
3. 进入轮回池
4. 按业力分配道位

**特殊情况**：
- 大功德者：可保留部分记忆
- 大愿未了者：可带愿转世
- 轮回印记持有者：可觉醒记忆

## 六、冥界规则

### 阳寿未尽不得入冥

**基本规则**：
- 只有寿终或意外死亡者才能入冥
- 阳寿未尽被强行拉入冥界者，会被送回

**例外情况**：
- 游魂：可在边界游荡
- 执念：因执念留在阳间
- 特殊安排：阎罗特许

### 因果记录与审判

**记录机制**：
- 生死簿自动记录
- 善恶判官核实
- 业力判官计算

**审判流程**：
1. 秦广王初审
2. 按罪分配地狱
3. 受刑完毕
4. 转轮王分配轮回

### 轮回的决定因素

| 因素 | 影响 |
|------|------|
| 功德 | 提升道位 |
| 业力 | 降低道位 |
| 愿力 | 可能带愿转世 |
| 因果 | 决定具体境遇 |

### 违反冥界法则的后果

| 违规行为 | 后果 |
|----------|------|
| 阳人闯入 | 被驱逐、寿元折损 |
| 鬼魂逃逸 | 被追捕、打入地狱 |
| 篡改生死簿 | 永堕无间地狱 |
| 破坏轮回 | 天道惩罚 |
