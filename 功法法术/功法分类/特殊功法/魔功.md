# 魔功

## 概述

魔功，乃修仙之道中以魔入道之法，走的是与正道截然相反之路。此道以破坏、吞噬、掠夺为要义，不修天道法则，唯修己身魔念。修炼魔功者，往往短时间内实力暴涨，但代价是心性渐失，最终沦为魔物。

**魔功之核心特点：**

- **速成之道**：修炼进度极快，短时间可达惊人境界
- **吞噬万物**：可吞噬他人修为、血肉、灵魂化为己用
- **以战养战**：战斗越激烈，实力提升越快
- **心魔缠身**：修炼至深处，心魔反噬，难以自拔
- **天理不容**：修炼魔功者，为天道所不容，渡劫时雷劫更甚

**修炼要点：** 魔功一途，虽能速成，但后患无穷。修炼者需有极强的心志，方能压制心魔，否则终将沦为杀戮机器，为天下所不容。
