# 修仙项目术语表

## 1. 修炼相关术语

### 1.1 修炼境界

| 术语 | 定义 |
|------|------|
| 练气期 | 修仙的起始阶段，修炼者开始吸纳天地灵气，淬炼自身，通常分为1-10层 |
| 筑基期 | 练气期的突破，将灵气固化在体内，形成气海，奠定修仙基础，分为初期、中期、后期、大圆满 |
| 结丹期 | 筑基期的突破，将气海中的灵气凝聚成金丹，标志着进入高阶修仙者行列 |
| 元婴期 | 结丹期的突破，金丹破碎，形成元婴，标志着灵魂和肉体的双重升华 |
| 化神期 | 元婴期的突破，元婴与肉体融合，化身神灵，掌握天地之力 |
| 炼虚期 | 化神期的突破，虚化肉体，与天地融为一体，初步掌握空间之力 |
| 合体期 | 炼虚期的突破，肉体与灵魂完全融合，掌握更高级的空间之力 |
| 大乘期 | 合体期的突破，掌握天地规则，达到渡劫成仙的门槛 |
| 渡劫期 | 大乘期大圆满后的特殊阶段，需要渡过天劫才能成仙 |

### 1.2 修炼资质

| 术语 | 定义 |
|------|------|
| 灵根 | 修仙者吸收和使用天地灵气的资质，决定修炼速度和属性亲和力 |
| 悟性 | 修仙者对功法和天地规则的理解能力，影响修炼速度和瓶颈突破 |
| 体质 | 修仙者的身体条件，影响修炼速度和对特定功法的适应性 |
| 血脉 | 修仙者继承的特殊血脉，可能带来特殊能力或属性加成 |

| 符箓类功法 | 用于制作符箓的功法 |

## 7. 术语使用规范

1. 所有术语应按照本术语表中的定义使用
2. 首次使用术语时应给出完整定义，后续可直接使用简称
3. 避免使用同义词或近义词，确保术语使用的一致性
4. 新添加的术语应及时更新到本术语表中
5. 术语翻译应保持一致性，避免同一术语有多种翻译
