# 因果法则

> 因果是世界的基本法则之一，种因得果，循环不息。修仙者需谨言慎行，避免种下恶因。

---

## 因果的本质

### 什么是因果？

因果是世界运行的基本法则之一，指的是任何行为都会产生相应的结果。种下善因，会得到善果；种下恶因，会得到恶果。因果循环，报应不爽。

### 因果的特点

| 特点 | 说明 |
|------|------|
| 必然性 | 有因必有果 |
| 循环性 | 因果循环不息 |
| 延迟性 | 因果报应可能延迟 |
| 公正性 | 因果报应公正无私 |
| 不可逆性 | 因果不可强行逆转 |
